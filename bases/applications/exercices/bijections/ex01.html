f est une application de E dans F.
Montrer que les deux propositions suivantes sont équivalentes:
  1. f est bijective
  2. ∀ X ⊆ E f(E-X)=F-f(X)

aide

Utiliser la propriété 2. avec des parties X particulières.

solution

Prenons X =∅ Si 2. est vérifiée Nous avons donc
f(E- ∅) = F-f( ∅)
soit encore
f(E)=F
d'où f est surjective
Prenons X={x} où x est un élément quelconque de E
Alors f(E-{x})=F-f({x})= F-{f(x)}
Donc on ne peut trouver x' ∈ E-{x}, c'est à dire x' ≠ x tel que f(x')=f(x) et f est injective.

Supposons réciproquement que f soit bijective
Soit X ⊆ E
On a forcément f(E-X) ⊆ F-f(X) sinon f ne pourrait être injective.
Cela dit prenons un y ∈ F-f(X), comme f est surjective cet y possède un antécédent, pour cause d'injectivité cet antécédent ne peut être dans X.
Donc y ∈ f(E-X), et cela achève la démonstration.

f is a map from E to F
Show that the following two propositions are equivalent:
  1. f is bijective
  2. ∀ X ⊆ E f(E-X)=F-f(X)

hint

Use property 2. with some special subsets X..

solution

Let X =∅ If 2.is true we have
f(E- ∅) = F-f( ∅)
furthermore
f(E)=F
so f is surjective
Take X={x} where x is any element of E
Then f(E-{x})=F-f({x})= F-{f(x)}
Hence we cannot find x' ∈ E-{x}, that is x' ≠ x such as f(x')=f(x) and f is injective.

Suppose conversely that f is bijective
Let X ⊆ E
We have by necessity f(E-X) ⊆ F-f(X) otherwise f could not be injective.
This being said take y ∈ F-f(X), since f is surjective this y has a preimage, because of injectivity this preimage cannot be in X.
So y ∈ f(E-X), and this completes the proof.
