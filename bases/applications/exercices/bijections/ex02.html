Montrer que si E et F sont deux ensembles finis ayant un même nombre n d' éléments les 3 propositions suivantes sont équivalentes:

  1. f est injective
  2. f est surjective
  3. f est bijective

En déduire le nombre de bijections de E dans F, le nombre de permutations de E.

aide

On connait déjà le nombre \(A_{n}^{p}\) d'injections d'un ensemble à p éléments dans un ensemble à n éléments.

solution

Si f est injective alors les images des éléments de E sont n éléments distincts de F.
Comme F a exactement n éléments tout élément est donc une image, f est donc surjective, donc bijective.
Si f est surjective, alors tout élément de F possède au moins un antécédent. Si un élément de F possède deux antécédents, alors le nombre d'éléments de E est strictement plus grand que celui de F, contrairement à l’hypothèse. Donc, f surjective implique f injective, et bijective.
Le nombre de bijections de E sur F ou bien le nombre de permutations de E est donc \(A_{n}^{p}=n!\)

Show that if E and F are two finite sets having the same number nd' elements the following 3 propositions are equivalent:

  1. f is injective
  2. f is surjective
  3. f is bijective

Deduce the number of bijections from E into F, the number of permutations of E.

help

We already know the number \(A_{n}^{p}\) of injections from a set with p elements into a set with n elements.

solution

If f is injective then the images of the elements of E are n distinct elements of F.
As F has exactly n elements any element is therefore an image, f is therefore surjective, therefore bijective.
If f is surjective, then every element of F has at least one preimage. If an element of F has two preimages, then the number of elements of E is strictly greater than that of F, contrary to the hypothesis. So f surjective implies f injective, and bijective.
The number of bijections of E onto F or else the number of permutations of E is therefore \(A_{n}^{p}=n!\)
