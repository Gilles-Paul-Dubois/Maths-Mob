Si E possède p éléments et si F possède n éléments, combien y a-t-il d'applications de E dans F ?

Écrire un programme qui génère toutes les applications de E dans F

aide

Examiner le nombre de possibilités pour l'image du premier élément de E, puis pour le couple (e1 ,e2 ) où e1 et e2 sont les deux premiers éléments de E et ainsi de suite.
Pour le programme, on peut écrire une version itérative.

solution

Le nombre total de possibilités est pn .C'est pourquoi l'ensemble est noté FE

If E has p elements and if F has n elements, how many maps are there from E to F?

Write a program that generates all the mappings from E to F

help

Examine the number of possibilities for the image of the first element of E, then for the ordered pair (e 1 , e2 ) where e1 and e2 are the first two elements of E and so on.
For the program, we can write an iterative version.

solution

The total number of possibilities is pn This is why the set is denoted FE.
