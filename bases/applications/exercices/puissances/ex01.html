Soit f une application de E dans E. (E ensemble à n éléments)
  1. Montrer que f idempotente ⇔ l'ensemble des points fixes de f est égal à f(E)
  2. Combien d'applications idempotentes n'ont aucun point fixe ?
  3. Combien d'applications idempotentes ont 1 point fixe exactement ?
  4. Combien d'applications idempotentes ont 2 points fixes exactement ?
  5. Combien d'applications idempotentes ont exactement p points fixes ?
  6. Combien y a-t-il d'applications idempotentes de E dans E ?

aide

Montrer que f(x) = x ⇔ ∃ y |  f(y)=x

solution

  1. Si y=f(x) alors f(y)=f(f(x))=f(x)=y, donc y est fixe pour f. Il en résulte que tout point de f(E) est fixe pour f. Il est évident en outre que tout point fixe appartient à f(E).
  2. Si f est idempotente avec 0 point fixe équivaut à f(E)=∅. Cela n'est possible que si E=∅, l'unique application de E dans E est alors l'unique application idempotente de E dans E. Dans le cas contraire (si E est non vide), il n'existe aucune application idempotente de E dans E n'ayant aucun point fixe.
  3. La seule application ayant y pour unique point fixe est la constante x → y. Il s'en suit que le nombre d'applications idempotentes ayant exactement un point fixe est égale à n=Card(E).
  4. Soit P= {a,b} une partie à 2 éléments de E. On cherche les applications idempotentes admettant exactement les points de P pour points fixes. D'après la première question, pour une telle application on a f(a)=a, f(b)=b et pour tout c de E-P f(c)=a ou f(c)=b. Le nombre des applications idempotentes admettant a et b pour points fixes correspond donc exactement au nombre des applications de E-P dans {a,b}, ce nombre est donc 2 n-2 . Donc pour avoir le nombre total des applications idempotentes admettant exactement 2 points fixes il faut multiplier ce nombre par \(C_{n}^{2}=\frac{n(n-1)}{2}\), nombre de paires dans un ensemble à n éléments. Réponse finale \(C_{n}^{2}\times 2^{n-2}\)
  5. En raisonnant comme dans la question précédente avec p au lieu de 2 on trouve: \(C_{n}^{p}\times p^{n-p}\)
  6. Lorsque n est différent de 0 pour obtenir le nombre des applications idempotentes de E dans E, il suffit d'ajouter le nombre des applications idempotentes ayant 1 point fixe avec celles ayant deux points fixes, puis trois points fixes, etc... On trouve donc pour formule:
    \(C_{n}^{1}+C_{n}^{2}\times 2^{n-2}+C_{n}^{3}\times 3^{n-3}+...+C_{n}^{n-1}\times (n-1)+1\)
This exercise suppose known the notion of .
Let f be a map from E to E. (E set with n elements)
  1. Show that f idempotent ⇔ the set of fixed points of f is equal to f(E)
  2. How many idempotent maps have no fixed point?
  3. How many idempotent maps have exactly 1 fixed point?
  4. How many idempotent maps have exactly 2 fixed points?
  5. How many idempotent maps have exactly p fixed points?
  6. How many idempotent maps are there from E to E?

hint

Show that f(x) = x ⇔ ∃ y | f(y)=x

solution

  1. If y=f(x) then f(y)=f(f(x))=f(x)=y, so y is fixed for f. It follows that every point of f(E) is fixed for f. It is also obvious that every fixed point belongs to f(E).
  2. If f is idempotent with 0 fixed point is equivalent to f(E)=∅. This is only possible if E=∅, then the unique map from E to E is the unique idempotent map from E to E. Otherwise (if E is non-empty), there is no idempotent map from E into E having no fixed point.
  3. The only map having y as unique fixed point is the constant x → y. It follows that the number of idempotent maps having exactly one fixed point is equal to n=Card(E).
  4. Let P= {a,b} a subset with 2 elements of E. We are looking for idempotent mappings having exactly points of P for fixed points. From the first question for such mapping f we have, f(a)=a, f(b)=b and for every c of E-P f(c)=a or f(c)=b. The number of idempotent mappings having a and b for fixed points corresponds exactly to the number of maps from E-P to {a,b}, this number is such 2 n-2 . So to get the total number of maps having exactly two fixed points we must multiply this number by \(C_{n}^{2}=\frac{n(n-1)}{2}\), number of pairs in a set with n elements. Final answer \(C_{n}^{2}\times 2^{n-2}\)
  5. Reasoning like previously with p instead of 2 we find : \(C_{n}^{p}\times p^{n-p}\)
  6. When n is not null to get the number of idempotent functions from E to E, we must add the number of maps having one fixed point with the number of those have 2 fixed points, and so on. So we find the formula :
    \(C_{n}^{1}+C_{n}^{2}\times 2^{n-2}+C_{n}^{3}\times 3^{n-3}+...+C_{n}^{n-1}\times (n-1)+1\)
