Combien y a-t-il d'injections d'un ensemble à p éléments dans un ensemble à n éléments (valeur de \(A_{n}^{p}\) )
Ecrire un programme qui génère toutes les injections de {1,2,..n} dans E où E est un ensemble fini quelconque.

aide

Question 1:
Il faut compter toutes les suites (x1 ,...,xp ) d'éléments distincts de E.
Question 2:
On pourra programmer récursivement.
Soit a un éléments de E, les suites contenant a seront obtenues par adjonction de a à celles ne le contenant pas.

solution

Il y a n possibilités pour x1 , n-1 pour x2 et ainsi de suite.
Donc \(A_{n}^{p}\)= n(n-1)(n-2)...(n-p+1) = p! \(C_{n}^{p}\)
Génération des injections de {1,2,..p} dans E. Programme Python récursif.

How many injections are there from a set with p elements into a set with n elements (value of \(A_{n}^{p}\) )
Write a program which generates all the injections from {1,2, .. n} into E where E is any finite set.

help

Question 1:
We must count all the sequences (x1 ,...,xp ) of distinct elements of E.
Question 2:
We can program recursively.
Let a be an element of E, the sequences containing a will be obtained by adding a to those not containing it.

solution

There are n possibilities fo x1 , n-1 for x2 and so on
Therefore \(A_{n}^{p}\) = n(n-1)(n-2)...(n-p+1) = p! \(C_{n}^{p}\)
