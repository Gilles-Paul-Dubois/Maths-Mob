Points fixes (invariants)

Fixed (invariant) points

Définition

définition
Soit f une application de E dans E. Un élément x de E est appelé un 'point fixe' (on dit aussi un 'invariant' ) de f s'il est égal à sa propre par f.
x point fixe de f ⇔ f(x)=x.

Une application peut n'avoir aucun point fixe (n→n+1 de ℕ dans ℕ).
Pour l'identité tous les éléments sont fixes.
Les points fixes jouent un rôle important dans de nombreux cas:
  • Dans le cas fini, ils permettent de faire des raisonnements par récurrence dans les problèmes de décomposition
  • Pour les transformations géométriques du plan et de l'espace (rotations, symétries, homothéties, affinités, etc..), ils jouent un rôle particulier.
  • Pour les fonctions d'une variable réelle, ou d'autres espaces métriques plus complexes, ils peuvent servir à déterminer les solutions de certaines équations (f(x)=0 ⇔ x est un point fixe de f+Id)

Definition

définition
f being a map from E to E. An element x of E is called a 'fixed point' (alternatively we can use the words 'fixpoint' or 'invariant point' ) of f if it is equal to its own by f.
x fixed point of f ⇔ f(x)=x.

A map can have no fixed point (n→n+1 from ℕ to ℕ).
On the opposite all points are fixed for identity map.
These points play an important role in a number of cases :
  • In the finite case, they allow to build induction reasoning in some decomposition problems.
  • For geometrical mappings from the plane or 3D space (rotations, symmetries, homotheties, stretchings etc.) they play a special role.
  • For functions of a real variable, or other metric spaces they can be used to solve equations (f(x)=0 ⇔ x is an invariant of f+Id)

Notations usuelles

Voici les notations usuelles pour les permutations des entiers entre 1 et n soit pour l'ensemble {1,2,...,n-1,n}.

Sur 2 lignes

La première ligne représente les entiers dans l'ordre naturel et dans la seconde on trouve en dessous de chaque entier son image.

\[ \begin{pmatrix} 1& 2& 3& 4& 5& 6& 7& 8& 9\\ 2& 4& 3& 6& 9& 1& 7& 8& 5\\ \end{pmatrix}\]

Sur une seule ligne

La première ligne est supposée on écrit donc en pratique que la seconde ce qui donnerait pour l'exemple ci-dessus :

\[ \begin{pmatrix} 2& 4& 3& 6& 9& 1& 7& 8& 5\\ \end{pmatrix}\]

Sur une ligne incomplète

Tous les élements manquants sont supposés être des points fixes. Ce qui donnerait toujours pour le même exemple :

\[ \begin{pmatrix} 2& 4& 6& 9& 1& 5\\ \end{pmatrix}\]

Usual notations

Here are the usual notations for integers between 1 and n that is for the set {1,2,...,n-1,n}.

On two rows

The first row represents the integers in natural order and in the second we find below each integer its image.

\[ \begin{pmatrix} 1& 2& 3& 4& 5& 6& 7& 8& 9\\ 2& 4& 3& 6& 9& 1& 7& 8& 5\\ \end{pmatrix}\]

On only one line

The first line is omitted in practice we write only the second one, which gives for the above example :

\[ \begin{pmatrix} 2& 4& 3& 6& 9& 1& 7& 8& 5\\ \end{pmatrix}\]

On one line not full

All missing elements are supposed to be fixpoints. It would give this for the previous example :

\[ \begin{pmatrix} 2& 4& 6& 9& 1& 5\\ \end{pmatrix}\]
