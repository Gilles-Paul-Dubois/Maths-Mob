Introduction

Les applications sont des relations particulières. Ce sont celles pour lesquelles chaque élément de la source possède une image et une seule.
Elles jouent à côté des relations binaires et conjointement avec elles un grand rôle dans les mathématiques modernes.
Les applications s'organisent à leur tour en ensembles très vastes, avec lesquels ont peut faire les opérations courantes (réunion, produit).
De fait l'ensemble des applications de E dans F, peut être vu comme un produit dont tous les facteurs sont égaux à F, ses facteurs étant indexés par E. Quand E est un ensemble fini l'ensemble des applications de E dans F s'identifie à Fn où n est le nombre d'éléments de E, un n-uple (x1, ..., xn) de Fn s'identifiant à l'application f:i → xi de E dans F. Cette façon de voir s'étend au cas ou E est un ensemble infini.
Bien que les applications partagent toutes la même propriété pour tous les éléments de la source (une image et une seule), une discrimination peut être faite sur les éléments du but selon le nombre de leurs antécédents. Ces considérations conduisent à la définition des injections, surjections et bijections.
Les bijections sont parmi les applications des éléments privilégiés puisqu'elles mettent les ensembles en relation 'un à un'. On les utilise souvent pour identifier des ensembles distincts par construction. Il est fait un très large usage des bijections dans les constructions des ensembles de nombres. C'est l'existence de bijections particulières qui permet d'écrire 2=+2=2/1=2.0 bien qu'a priori ces éléments appartiennent à des ensembles distincts.
Lorsque les bijections ont pour source et but le même ensemble on parle de 'permutations' de cet ensemble. Les permutations d'un ensemble sont importantes dans la mesure où elles constituent historiquement les premiers exemples de 'groupes'.
Une attention particulière sera apportée aux relations existant entre les applications entre ensembles et les relations d'équivalence ou les relations d'ordre existant sur ces ensembles.

Introduction

Applications are special relations. These are those for which each element of the departure set has one image and only one.
They play alongside binary relations and together with them a great role in modern mathematics.
The applications are organized in their turn into very large sets, with which the current operations (union, product) can be carried out.
In fact, the set of maps (applications or mappings)from E to F can be seen as a product of which all the factors are equal to F, its factors being indexed by E. When E is a finite set the set of maps from E to F identifies with Fn where n is the number of elements of E, an n-tuple (x1 , ..., xn ) of Fn identifying with the application f: i → xi from E to F. This view extends to the case where E is an infinite set.
Although the applications all share the same property for all the elements of the source (one image and only one), a discrimination can be made on the elements of the destination set according to the number of their preimages. These considerations lead to the definition of injections, surjections and bijections.
The bijections are among the applications privileged elements since they put the sets in relation 'one to one'. They are often used to identify distinct sets by construction. A very large use is made of bijections in the construction of sets of numbers. It is the existence of particular bijections which makes it possible to write 2 = + 2 = 2/1 = 2.0 although at first sight these elements belong to distinct sets.
When the bijections have for departure and destination the same set, we speak of 'permutations' of this set. Permutations of a set are important insofar as they historically constitute the earliest examples of 'groups'.
Particular attention will be paid to the relations existing between the applications between sets and the equivalence relations or the order relations existing on these sets.
