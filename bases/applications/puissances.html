théorème
Si E est lui-même un ensemble fini, l'ensemble de toutes les applications de E dans E étant également fini, il est impossible que toutes les puissances de f forment un ensemble infini. Il y a donc forcément un phénomène cyclique. Il existe une valeur n telle que fn=fp avec p < n , on a alors fn+1=fp+1, et ainsi de suite.
définition
Une applications de E dans E est dite 'idempotente' si toutes ses puissances strictement positives sont égales à elle-même.

Definitions

definition
When f is a from E to E, it is possible to f with itself, the result f of is denoted as well f2.
The same way fn=fn-1 of=f of of o... of (n factors).
definition
To insure validity of the formula fpofq=fp+q, we also agree to define fo as the identity and f 1 =f.
Let us state a remark which we will use later:
theorem
If E is a finite set,the set of all mappings from E to E is also finite, so it is impossible that all the powers of the same function f to be infinite. So there should be some periodicity. It exists a value n for which fn=fp with p < n , in this case we have fn+1=fp+1,and so on.
definition
A mapping from E to E is said to be 'idempotent' If all strictly positive powers of f are equal to f itself.
