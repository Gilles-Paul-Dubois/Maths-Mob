Définitions

définition
Le quantificateur 'universel' (notation ∀) est utilisé en relation avec une variable pour exprimer qu'un prédicat P dépendant de cette variable est vrai pour toutes les valeurs de la variable.
On note donc cela :
∀x P(x)
définition
Le quantificateur 'existentiel' (notation ∃ ) est utilisé en relation avec une variable pour exprimer qu'un prédicat P dépendant de cette variable est vrai pour au moins une valeur de la variable.
On note donc cela :
∃x P(x)
Les deux quantificateurs sont liés, c'est à dire que l'un peut s'exprimer au moyen de l'autre et vice-versa au moyen de la négation.
∃x P(x) peut s'écrire ¬( ∀x ¬P(x))
Notons que dans ces cas encore, la variable utilisée est arbitraire :
∀x P(x) est équivalente à ∀y P(y)
Nous verrons cependant que dans le cas de formules complexes utilisant plusieurs quantificateurs et donc plusieurs variables il importe d'éviter les télescopages, et donc de respecter certaines règles.
Détaillons maintenant ce qu'on entend par 'domaine de quantification'.
La phrase 'tout le monde dort' est souvent jugée vraie, même quand tous les humains de la planète ne sont pas endormis. Il s'agit souvent de tous les occupants de la maison.
Il y a donc toujours un domaine de quantification qui n'est pas déterminé par le contenu de la phrase, mais qui est un élément contextuel.
On peut toujours restreindre explicitement le domaine de quantification à un ensemble:
Si l'on veut exprimer que la propriété P(x) est vraie pour tous les éléments de E seulement on écrira:
∀x (x∈E ⇒ P(x))
que l'on abrégera parfois en :
∀x∈E P(x) (notation non strictement conforme à la syntaxe du calcul des prédicats)

Definitions

definition
The 'universal' quantifier (notation ∀) is used in relation with a variable to express that a predicate P depending on this variable is true for all values ​​of the variable.
So we note that:
∀x P(x)
definition
The 'existential' quantifier (notation ∃) is used in relationship with a variable to express that a predicate P depending on this variable is true for at least one value of the variable.
So we note that:
∃x P(x)
The two quantifiers are linked, i.e. one can to express oneself by means of the other and vice versa using negation.
∃x P(x) can be written ¬( ∀x ¬P(x))
Note that in these cases again, the variable used is arbitrary:
∀x P(x) is equivalent to ∀y P(y)
We will however see that in the case of complex formulas using several quantifiers and therefore several variables it is important to avoid telescoping, and therefore to respect certain rules.
Let us now detail what is meant by 'quantification domain' .
The phrase 'everyone is asleep' is often held to be true, even when everyone the humans of the planet are not asleep. It is often about all occupants of the house.
There is therefore always a domain of quantification which is not determined by the content of the sentence, but which is an element contextual.
We can always restrict explicitly the domain of quantification to a set:
If we want to express that the property P(x) is true for all elements of E only we will write:
∀x (x∈E ⇒ P(x))
sometimes abbreviated as:
∀ x∈E P(x) (notation not strictly conforming to the syntax of predicate calculus)
