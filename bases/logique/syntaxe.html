Définitions

Cette section donne une brève présentation de la grammaire du calcul des prédicats dans un langage formel couramment utilisé en logique classique par la plupart des mathématiciens.

Cette grammaire peut donc être formalisée et utilisée dans tout programme d'ordinateur pour décider si une proposition (assertion) est bien formée.

On considère donc, comme données primitives du langage:
  • V un ensemble de symboles appelés variables , toujours infini.
  • C un ensemble de symboles appelés constantes , éventuellement vide.
  • P un ensemble de symboles de prédicats*
  • F un ensemble de symboles de fonctions* , éventuellement vide.
  • les symboles ∀ et ∃
  • les connecteurs logiques usuels du calcul des propositions
  • les parenthèses "(" et ")", allant toujours par paires.
(*) Les mots 'prédicat' et 'fonction' ont ici les sens définis dans les sections précédentes.
définition
On appellera termes les formules composées ainsi :
  • tout élément de V , est un terme,
  • tout élément de C , est un terme,
  • si f est un symbole de fonction n -aire et t1,t2,...,tn sont des termes, alors f(t1,t2,..,tn) est un terme.
définition
Un terme est 'clos' s'il ne contient pas de nom de variable.

Les termes ont pour but de représenter les objets sur lesquels vont s'appliquer des prédicats.

définition

Les énoncés ou formules du 'calcul des prédicats du premier ordre' sont les suivants, et uniquement les suivants :

  • p(t1,t2,...,tn) si t1, t2,..tn sont des termes et p un prédicat (comme dans le calcul propositionnel un tel enonçé est dite 'atomique').
  • Toute formule formée à partir d'énoncés au moyen des 5 connecteurs usuels du calcul propositionnel.
  • ∀x e est un énoncé si e est un énoncé et x une variable
  • ∃x e est un énoncé si e est un énoncé et x une variable
  • (e) est un énoncé si e en est un
Exemple: On veut exprimer formellement que le sinus d'un nombre réel est toujours compris entre -1 et 1, ce qui donnera:
∀ x ((x∈ℝ) ⇒((EstInf(-1,sin(x))∧( EstInf(sin(x),1))))
avec les conventions d'écriture définissant le prédicat EstInf(x,y) ssi x ≤ y et x∈ℝ pouvant être écrit sous forme prédicative EstElementDe(x,ℝ).
Il va de soi qu'EN PRATIQUE on écrira:
∀x ∈ ℝ -1 ≤ sin(x) ≤ 1
Il s'agit simplement de montrer qu'en théorie tout énoncé peut prendre la forme standard décrite plus haut.

Definitions

This section gives a brief presentation of the grammar of the calculus predicates in a formal language commonly used in logic classic by most mathematicians.

This grammar can therefore be formalized and used in any computer program for deciding whether a proposition (assertion) is well formed.

We therefore consider, as primitive data of the language:
  • V a set of symbols called variables , always infinite.
  • VS a set of symbols called constants , possibly empty.
  • P a set of symbols of predicates*
  • F a set of symbols of functions* , possibly empty.
  • the symbols ∀ and ∃
  • the usual logical connectors of the calculus of propositions
  • the parentheses "(" and ")", always going in pairs.
(*) The words 'predicate' and 'function' have here the meanings defined in the previous sections.
definition
We will call terms compound formulas so :
  • all element of V , is a term,
  • all element of VS , is a term,
  • if f is a n -ary function symbol and t1,t,...,tn are terms, then f(t1,t2,..,tn) is a term.
definition
A term is 'closed' if it does not contain a variable name.

The terms are intended to represent the objects on which predicates will apply.

definition

The statements or formulas from 'first order logic' are the following, and only the following:

  • p(t1,t2,...,tn) if t1, t2,..tnare terms and p a predicate (like in propositional calculus such a statement is said 'atomic' ).
  • Any formula formed from statements using the 5 connectors commonplace in propositional calculus.
  • ∀xe is a statement if e is a statement and x a variable
  • ∃xe is a statement if e is a statement and x a variable
  • (e) is a statement if e is one
Example: We want to express formally that the sine of a real number is always between -1 and 1, which will give:
∀ x ((x∈ℝ) ⇒((EstInf(-1,sin(x))∧(EstInf(sin(x),1))))
with the writing conventions defining the predicate IsInf(x,y) iff x ≤ y and x∈ℝ which can be written in the predicative form IsElementOf(x,ℝ).
It goes without saying that IN PRACTICE we will write:
∀x ∈ ℝ -1 ≤ sin(x) ≤ 1
It is simply a question of showing that in theory all statement can take the standard form described above.
