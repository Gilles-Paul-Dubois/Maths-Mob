Soit R une relation logique où la lettre x ne figure pas.
Montrer que:
∀ x (R ∨ S(x)) ⇔ (R ∨ ∀ x S(x))
∃ x (R ∧ S(x)) ⇔ ( R ∧ ∃ x S(x))

aide

x ne figurant pas dans R, R ne dépend pas de x et on a:
soit ∀ x R
soit ∀ x ¬ R

solution

Supposons qu'on ait R fausse et  ∀ x (R ∨ S(x))
c'est donc puisque R ∨ S(x) est vraie que S(x) est vraie ∀ x
Si maintenant R est vraie indépendemment de x on a toujours   (R ∨ ∀ x S(x)) vraie
Donc dans tous les cas :
∀ x (R ∨ S(x)) ⇒ (R ∨ ∀ x S(x))
La réciproque se vérifie de la même manière.
Si R est vraie ∃ x (R ∧ S(x)) se résume à ∃ x ( S(x)) donc ( R ∧ ∃ x S(x)) et
∃ x (R ∧ S(x)) ⇒ ( R ∧ ∃ x S(x))
La réciproque se vérifie de la même manière.

Let R be a logical relation where the letter x does not appear.
Show that:
∀ x (R ∨ S(x)) ⇔ (R ∨ ∀ x S(x))
∃ x (R ∧ S(x)) ⇔ ( R ∧ ∃ x S(x))

hint

x not appearing in R, R does not depend on x and we have:
either ∀ x R
Or ∀ x ¬ R

solution

Suppose we have R false and  ∀x (R ∨ S(x))
It is because R ∨ S(x) is true that S(x) is true ∀ x
If now R is true independently of x we ​​still have  (R ∨ ∀ x S(x)) true
So in any case:
∀ x (R ∨ S(x)) ⇒ (R ∨ ∀ x S(x))
The converse is verified in the same way.
If R is true ∃ x (R ∧ S(x)) sums to ∃ x ( S(x)) so ( R ∧ ∃ x S(x)) and
∃ x (R ∧ S(x)) ⇒ ( R ∧ ∃ x S(x))
The converse is verified in the same way.
