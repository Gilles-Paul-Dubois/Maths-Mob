Ecrire un programme qui liste toutes les sous-formules d'une formule donnée.

aide

Reprendre tous les cas traités par l'analyseur syntaxique.

solution

Le programme Python suivant affiche:  Set(['A', '((-A)&B)|C', 'C', 'B', '-A', '(-A)&B'])

Le programme Julia 1.6 suivant affiche:  Set(["(A&B)|(!C)", "B", "A", "C", "A&B", "!C"])

Write a program which lists all subformula of any given formula.

hint

Consider again all cases treated by the parser.

solution

Notice to English readers :
Examples programs or exercises solutions, either in Python or in Julia have the SAME original code, only comments are translated. That's why the logical constant with true value is denoted 'V' (for French 'vrai') instead of 'T' for English true. You can easily change that.

The following Python program prints :  Set(['A', '((-A)&B)|C', 'C', 'B', '-A', '(-A)&B'])

The following Julia 1.6 program prints : Set(["(A&B)|(!C)", "B", "A", "C", "A&B", "!C"])
