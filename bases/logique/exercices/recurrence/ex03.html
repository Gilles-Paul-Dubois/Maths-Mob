Soit (un) n∈ℕ la suite définie par:

u 0 =1
∀ n ∈ ℕ \[u_{n+1}=\sum_{k=0}^{n}u_{k}\]
Montrer par récurrence forte  que pour tout n ≥ 1 on a: un=2n-1

aide

Commencer par étudier le cas n=1, et utiliser la formule:

1+k+k2 +...+kn = (1-kn+1)/(1-k)
(Somme d'une progression géométrique)

solution

La propriété est évidemment vraie pour n=1.

Le calcul suivant montre que si elle est vraie pour  1 ≤ k ≤ n, elle est aussi vraie à l'ordre n+1:

\(u_{n+1}=\sum_{k=0}^{n}u_{k}=u_{0}+\sum_{k=1}^{n}u_{k}=1+\sum_{k=1}^{n}2^{k-1}=1+\sum_{k=0}^{n-1}2^{k}=1+\frac{1-2^{n}}{1-2}=2^{n}\)

Let (un) n∈ℕ the sequence defined by :

u 0 =1
∀ n ∈ ℕ \[u_{n+1}=\sum_{k=0}^{n}u_{k}\]
Show using strong induction that for every n ≥ 1 we have: un=2n-1

hint

Begin to study the case n=1, and use the formula :

1+k+k2 +...+kn = (1-kn+1)/(1-k)
(sum of a geometric progression)

solution

The property is obviously true for n=1.

The following computations shows that if it true for  1 ≤ k ≤ n, it is also true at order n+1:

\(u_{n+1}=\sum_{k=0}^{n}u_{k}=u_{0}+\sum_{k=1}^{n}u_{k}=1+\sum_{k=1}^{n}2^{k-1}=1+\sum_{k=0}^{n-1}2^{k}=1+\frac{1-2^{n}}{1-2}=2^{n}\)
