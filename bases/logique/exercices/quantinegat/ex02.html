Enoncer des assertions équivalentes aux négations des assertions suivantes:

  1. Dans toutes les prisons tous les détenus détestent tous les gardiens
  2. Pour tout entier x il existe un entier y tel que pour tout entier z la relation z<y+1 implique la relation z<x
L'assertion 2 est-elle vraie ?

aide

Examiner le cas x=0 dans l'assertion 2.

solution

Il existe une prison où il existe un détenu qui ne déteste pas un gardien.
L'assertion 2 est vraie pour les entiers relatifs prendre y=x-1.
Elle est fausse pour les entiers naturels (cas x=0).

State assertions equivalent to the negations of the following assertions:

  1. In all the prisons all the inmates hate all the guards
  2. For any integer x there exists an integer y such that for any integer z the relation z<y+1 implies the relation z<x
Is statement 2 true?

hint

Examine the case x=0 in assertion 2.

solution

There is a prison where there is an inmate who does not hate a guard.
Assertion 2 is true for integers taking y=x-1.
It is false for natural numbers (case x=0).
