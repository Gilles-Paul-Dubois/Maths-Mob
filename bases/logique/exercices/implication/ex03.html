Enoncer en français, la réciproque et la contraposée de chacune des 5 implications suivantes :

  1. Si je pense, alors je suis.
  2. Si je ne pense pas, alors je n'existe pas.
  3. Si je ne pense pas, alors je suis Zeus.
  4. Si je suis Zeus, alors je pense.
  5. Pour lire dans les cartes, il est nécessaire d'avoir consulté l'oracle.

aide

Aucune difficulté particulière, reportez vous aux définitions:
La réciproque de P ⇒ Q est  Q ⇒ P
La contraposée de P ⇒ Q est ¬ Q ⇒ ¬ P

solution

Réciproques:
  1. Si je suis, alors je pense.
  2. Si je n'existe pas, alors je ne pense pas.
  3. Si je suis Zeus alors je ne pense pas.
  4. Si je pense alors je suis Zeus.
  5. Pour consulter l'oracle il faut lire dans les cartes.
Contraposées:
  1. Si je ne suis pas alors je ne pense pas.
  2. Si j'existe alors je pense.
  3. Si je ne suis pas Zeus, alors je pense.
  4. Si je ne pense pas alors je ne suis pas Zeus.
  5. Sans avoir consulter l'oracle impossible de lire dans les cartes.

State in French, the reciprocal and the contrapositive of each of the following 5 implications:

  1. If I think, then I am.
  2. If I don't think, then I don't exist.
  3. If I don't think, then I am Zeus.
  4. If I am Zeus, then I think.
  5. To read in the cards, it is necessary to have consulted the oracle.

hint

No particular difficulty, refer to the definitions:
The reciprocal of P ⇒ Q is Q ⇒ P
The contrapositive of P ⇒ Q is ¬Q ⇒ ¬P

solution

Reciprocals:
  1. If I am, then I think.
  2. If I don't exist, then I don't think.
  3. If I am Zeus then I don't think.
  4. If I think then I am Zeus.
  5. To consult the oracle you have to read the cards.
Contrapositives :
  1. If I'm not then I don't think.
  2. If I exist then I think.
  3. If I am not Zeus, then I think.
  4. If I don't think then I am not Zeus.
  5. Without having consulted the oracle impossible to read in the maps.
