Définitions

La valeur de vérité d'une formule dépend évidemment de ses composants et des propriétés des connecteurs vérifonctionnels ¬ , ∧ , ∨ .

Ainsi si P est vraie , Q fausse et R vraie alors:
(P∨Q)∧(¬ R) est fausse car ¬R est fausse.

De fait, la formule de notre exemple contient 3 variables atomiques chacune pouvant avoir deux valeurs de vérité, le nombre total des possibilités est donc de 23 =8

Nous pouvons résumer dans un tableau les différentes possibilités et les valeurs de l'expression correspondante.

définition
Un tel tableau récapitulatif s'appelle une 'table de vérité'.
propriété
Si une formule contient n composants atomiques, la table de vérité correspondante aura 2n lignes.

Les tables de vérité constituent un algorithme naïf pour l'évaluation des formules. Nous verrons qu'il en existe d'autres.

Definitions

The truth value of a formula obviously depends on its components. and properties of connectors not, and, or, with symbols ¬, ∧, ∨.

So if P is true, Q is false and R is true then:
(P∨Q)∧(¬R) is false because ¬R is false.

In fact, the formula of our example contains 3 atomic variables each of which can have two truth values, the total number of possibilities is therefore 23 =8

We can summarize in a table the different possibilities and the values ​​of the corresponding expression.

P Q R P∨Q ¬ R (P∨Q)∧(¬ R)
T T T T F F
T T F T T T
T F T T F F
T F F T T T
F T T T F F
F T F T T T
F F T F F F
F F F F T F
definition
Such a summary table is called a 'truth table' .
property
If a formula contains n atomic components, the truth table corresponding will have 2n lines.

Truth tables are a naive algorithm for evaluation formulas. We will see that there are others.
