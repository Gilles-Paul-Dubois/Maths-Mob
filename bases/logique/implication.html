Définitions

définition
On dit qu'une proposition P 'implique' (synonymes: 'force', 'entraîne') une proposition Q si et seulement si Q est vraie chaque fois que P est vraie.
Exemple:
P: 1=2
Q: 2=3
Si P est vraie alors 1+1=2+1 donc Q.
Cela revient à dire qu'il est impossible qu'on ait P vraie sans avoir Q vraie également.
Autrement dit, il est impossible d'avoir P sans avoir Q.
Autrement dit encore, P∧(¬Q) est une , ou bien ¬(P∧(¬Q)) est une .
Mais en utilisant les équivalences usuelles, cela revient à:
(¬P)∨(¬(¬Q)) tautologie
Soit encore (¬P)∨Q tautologie.
ou finalement, ce qui est encore équivalent Q∨(¬ P) tautologie.
Cela nous conduit à poser la définition suivante:
définition
Nous noterons P⇒Q la formule Q∨(¬ P)
Nous définissons ainsi un nouvel opérateur ⇒ sur l'ensemble des formules du calcul propositionnel.
Notons les propriétés suivantes, qui résultent de la définition:
propriétés
  • Si P est fausse alors P⇒Q est toujours vraie quelle que soit la proposition Q.
  • Si Q est vraie alors P⇒Q est toujours vraie quelle que soit la proposition P.

C'est en application de cette première propriété qu'on démontre par exemple que ∅ ⊆ E pour tout ensemble E.

définition
Si P⇒Q est une implication l'implication Q⇒P s'appelle l'implication 'réciproque'.
L'implication directe peut être vraie sans que sa réciproque soit vraie.
Exemple ( x désigne un nombre entier différent de 2 ):
P: x est premier
Q : x est impair
On a bien P⇒Q mais on n'a pas Q⇒P
définition
Notons maintenant P⇔Q la relation (P⇒Q)∧(Q⇒P) (double implication). On dit encore dans ce cas que P et Q sont 'équivalentes'.
La terminologie laisse à penser qu'il peut y avoir risque de confusion entre P⇔Q et P≡Q . De fait il n'en est rien si on remarque que:
P≡Q si et seulement si P⇔Q est une tautologie.
définition
Si P⇒Q est une implication l'implication ¬Q⇒¬P s'appelle l'implication 'contraposée'.
Toute implication est logiquement équivalente à sa contraposée.

En effet ¬Q⇒¬ P signifie (¬P)∨(¬(¬Q)) et est équivalente à (¬P)∨Q soit encore à Q∨(¬ P)
Voici une liste de tautologies d'usage courant, correspondant compte tenu de la remarque précédente à :
idempotence de ¬ :
¬(¬P) ⇔ P
idempotence de ∧ et ∨ :
P ∧ P ⇔ P
P ∨ P ⇔ P
associativité de ∧ et ∨ :
P ∧ (Q ∧ R) ⇔ (P ∧ Q) ∧ R
P ∨ (Q ∨ R) ⇔ (P ∨ Q) ∨ R
commutativité de ∧ et ∨ :
P ∧ Q ⇔ Q ∧ P
P ∨ Q ⇔ Q ∨ P
distributivité :
(P ∨ Q) ∧ P ⇔ (P ∧ P) ∨ (Q ∧ R)
(P ∧ Q) ∨ R ⇔ (P ∨ R) ∧ (Q ∨ R)
De Morgan:
¬(P ∨ Q) ⇔ (¬ P) ∧ (¬ Q)
¬(P ∧ Q) ⇔ (¬ P) ∨ (¬ Q)

Definitions

definition
We say that a proposition P 'implies' (synonym: 'forces', 'causes') a proposition Q if and only if Q is true whenever P is true.
Example:
P: 1=2
Q: 2=3
If P is true then 1+1=2+1 so Q.
This amounts to saying that it is impossible that we have P true without having Q also true.
In other words, it is impossible to have P without having Q.
In other words again, P∧(¬Q) is a , or ¬(P∧(¬Q)) is a .
But using the usual equivalences, this amounts to:
(¬P) ∨ (¬ (¬Q)) is a tautology
or (¬P)∨Q is a tautology.
or finally, which is still equivalent Q∨(¬P) is a tautology.
This leads us to pose the following definition:
definition
We will denote by P⇒Q the formula Q∨(¬ P)
We thus define a new operator ⇒ on the set of propositional formulas.
Note the following properties, which result from the definition:
properties
  • If P is false then P⇒Q is always true whatever the proposition Q.
  • If Q is true then P⇒Q is always true regardless of the proposition P.

It is by applying this first property that we prove by example that ∅ ⊆ E for any set E.

definition
If P⇒Q ​​is an implication the implication Q⇒P is called 'reciprocal' or 'converse' implication.
The direct implication can be true without its converse being true.
Example ( x denotes an integer different from 2 ):
P: x is prime
Q: x is odd
We have P⇒Q but we don't have Q⇒P
definition
Let us now denote by P⇔Q the relation (P⇒Q)∧(Q⇒P) (double implication). We still say in this case that P and Q are 'equivalent' .
The terminology suggests that there may be a risk of confusion between P⇔Q and P≡Q . In fact it is not so if we notice that:
P≡Q if and only if P⇔Q is a tautology.
definition
If P⇒Q ​​is an implication the implication ¬Q⇒¬P named 'contrapositive' implication.
Any implication is logically equivalent to its contrapositive.

Indeed ¬Q⇒¬ P means (¬P)∨(¬(¬Q)) and is equivalent to (¬P)∨Q or still to Q∨(¬P)
The following is a list of tautologies in common use, taking into account :
idempotence of ¬ :
¬(¬P) ⇔ P
idempotence of ∧ and ∨ :
P ∧ P ⇔ P
P ∨ P ⇔ P
associativity of ∧ and ∨ :
P ∧ (Q ∧ R) ⇔ (P ∧ Q) ∧ R
P ∨ (Q ∨ R) ⇔ (P ∨ Q) ∨ R
commutativity of ∧ and ∨ :
P ∧ Q ⇔ Q ∧ P
P ∨ Q ⇔ Q ∨ P
distributivity:
(P ∨ Q) ∧ P ⇔ (P ∧ P) ∨ (Q ∧ R)
(P ∧ Q) ∨ R ⇔ (P ∨ R) ∧ (Q ∨ R)
De Morgan:
¬(P ∨ Q) ⇔ (¬ P) ∧ (¬ Q)
¬(P ∧ Q) ⇔ (¬ P) ∨ (¬ Q)

Terminologie (langage courant)

définition
Pour exprimer que P⇒Q on dit parfois que 'P est une condition suffisante pour Q' ou bien que 'Q est une condition nécessaire pour P' .
définition
P⇔Q peut donc aussi s'exprimer par: 'P est une condition nécessaire et suffisante pour Q' ; on dit aussi dans ce cas 'P est vrai si et seulement si Q est vrai'.

Les phrases suivantes ont le même sens, elles permettent de varier un peu le discours mathématique:
  • Si P, alors Q
  • Puisque P, donc Q
  • Il suffit de P pour Q
  • P est suffisante pour Q
  • Q est nécessaire pour P
  • Il faut Q pour P

Terminology (everyday's language)

definition
To express that P⇒Q we sometimes say that 'P is a sufficient condition for Q' or that 'Q is a necessary condition for P' .
definition
P⇔Q can therefore also be expressed by: 'P is a necessary and sufficient condition for Q' ; we also say in this case 'P is true if and only if Q is true'.

The following sentences have the same meaning, they allow you to vary a little mathematical discourse:
  • If P, then Q
  • Since P, therefore Q
  • P insures Q
  • P is sufficient for Q
  • Q is needed for P
  • We need Q for P
