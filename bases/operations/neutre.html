Définition

définition
Soit E un ensemble muni d'une loi *.
Un élément ε de E est dit 'neutre' pour * s'il possède la propriété suivante:
∀ x ∈ E ε*x=x*ε=x
C'est à dire que la composition d'un élément quelconque avec le neutre laisse cet élément inchangé.

Quelques exemples

  • Le nombre 0 pour l'addition des nombres
  • Le nombre 1 pour la multiplication des nombres
  • L'application identique pour la composition sur l'ensemble des permutations d'un ensemble
  • L'ensemble E pour la loi ∩ sur ℘(E)
  • L'ensemble vide ∅ pour la loi ∪ sur ℘(E)

Definition

definition
E denotes a set with an operation *.
An element ε of E is called a 'neutral element' or 'identity element' for * if it possesses the following property :
∀ x ∈ E ε*x=x*ε=x
that is to say that composing any element to the right or to the left with this 'neutral' leaves this element unchanged (has no effect).

A few examples

  • the number 0 for addition of numbers
  • The number 1 for the multiplication of numbers
  • The identity map for composition on the set of permutations of a set E.
  • The set E for operation ∩ (intersection) on ℘(E)
  • The empty set ∅ for operation ∪ (union) on ℘(E)

Propriété

théorème
S'il existe un élément neutre pour une opération on voit tout de suite qu'il est unique.

démonstration

Si ε1 et ε2 sont deux neutres pour la loi * on a:
ε 1 *ε 2 =ε 2 car ε 1 est neutre
mais aussi:
ε 1 *ε 2 = ε 1 car ε 2 est neutre

proof

ε 1 *ε 2 =ε 2 because ε 1 is neutral
