Montrer que les anneaux ℤ/nℤ (voir ) sont des corps si et seulement si n est un nombre premier.

aide

Soit r un entier r ∈ {1,2,...,n-1}, c(r) sa classe pour la relation de congruence modulo n.
Traduire c(r) inversible, sachant que c(1) est l'unité.
Utiliser le théorème de Bézout sur les couples de nombres premiers entre eux.

solution

c(r) inversible ⇔ ∃ s |   s ∈ {1,2,...,n-1} c(r)c(s)=c(1)
Or c(r)c(s)=c(rs)
c(rs)=c(1)  ⇔  rs=kn+1  ⇔  1=rs-kn
Or si n est premier, n est premier avec tout nombre strictement inférieur à lui, donc le théorème de Bezout affirme:
∃ p, q  ∈ ℤ |  1= pr+qn.
Soit s le reste de la division de p par n
Alors p=hn+s
Donc
1=(hn+s)r+qn = rs+n(h+q)
On a donc l'égalité voulue avec k=-h-q
Supposons inversement que ℤ/nℤ soit un corps
Alors ∀ r ∈ {1,2,...,n-1} r est inversible
On a donc 1=rp+nq d'après le théorème de Bezout.
Donc n est premier avec tous les nombres qui lui sont strictement inférieurs, donc n'admet aucun diviseur autre que 1 et lui-même, donc n est premier.

Show that the rings ℤ/nℤ (see ) are fields if and only if n is a prime number.

hint

r is an integer r ∈ {1,2,...,n-1}, c(r) its class modulo n.
Translate c(r) has an inverse , given that c(1) is the unit element.
Use Bézout's identity about the pairs of coprime integers.

solution

c(r) is a unit ⇔ ∃ s |   s ∈ {1,2,...,n-1} c(r)c(s)=c(1)
But c(r)c(s)=c(rs)
c(rs)=c(1)  ⇔  rs=kn+1  ⇔  1=rs-kn
But if n is prime, n is coprime with any number strictly inferior to itself and Bézout's lemma says :
∃ p, q  ∈ ℤ |  1= pr+qn.
Let s be the remainder of the division of p by n
So p=hn+s
hence
1=(hn+s)r+qn = rs+n(h+q)
So we have the equality wanted with k=-h-q
Suppose conversely that ℤ/nℤ is a field
Then ∀ r ∈ {1,2,...,n-1} r is a unit
So we have 1=rp+nq from Bézout's identity again.
So n is coprime with all numbers strictly before itself, so has no divisor except 1 and itself , so n is prime.
