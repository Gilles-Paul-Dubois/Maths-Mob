Quels sont les éléments symétrisables pour la loi de composition interne définie sur ℝ par : x*y=x+y-xy ?

aide

1 peut il être symétrisable ?

solution

1*x=1+x-x×1 = 1
donc 1*x ne peut être égal à 0 (neutre de *) voir
Donc 1 ne peut être symétrisable.
Si x est  ≠ 1 alors x*y = 0 s'écrit x+y-xy=0 et équivaut donc à :
y=-x/(1-x)
Donc tous les éléments sont symétrisables sauf 1.

Which elements have inverses for the operation defined on ℝ by : x*y=x+y-xy ?

hint

Is it possible to find an inverse for 1 ?

solution

1*x=1+x-x×1 = 1
hence 1*x cannot be 0 (neutral for *) see
So 1 cannot have any inverse.
If x is  ≠ 1 then x*y = 0 can be written x+y-xy=0 and is equivalent to :
y=-x/(1-x)
As a result every element has an inverse, except 1.
