Soit (A,+, × ) un anneau.
Soit a ∈ A.
Montrer que l'application x → ax  est un endomorphisme de (A,+).
Montrer que c'est un isomorphisme si a est inversible.
Donner un exemple où l'application x → ax n'est pas surjective.
Donner un exemple où l'application x → ax n'est pas injective.

aide

Si a est inversible  considérer l'application \(x\rightarrow a^{-1}x\)

solution

L'application x → ax  est un endomorphisme de (A,+) se traduit par:
a(x+y) = ax+ay qui n'est rien d'autre que la distrivutivité de × sur +.
Si a est inversible l'application \(x\rightarrow a^{-1}x\)  est la bijection réciproque de  l'application \(x\rightarrow ax\), laquelle est donc un automorphisme de (A,+).
Si A=ℤ et a=2 l'image de ℤ est l'ensemble des nombres pairs, l'application est donc non surjective.
Si A  désigne l'ensemble des applications de ℝ dans ℝ et si a est l'application définie par:
a(x)= 0 si x  ≥ 0 et a(x)=1 si x <0
Si f est ainsi définie :
f(x) = 1 si x  ≥ 0 et f(x)=0 sinon
et g est ainsi définie:
g(x) = 2 si x  ≥ 0 et g(x)=0 sinon
alors f ≠ g et cependant af=ag=0  donc \(f\rightarrow af\) n'est pas injective.

(A,+, × ) is a ring and a is an element of A.
Show that the map x → ax  is an endomorphism of (A,+).
Show that it is an isomorphism if a is a unit.
Give an example where the map x → ax is not surjective.
Give an example where the map x → ax is not injective.

help

If a is a unit consider the map \(x\rightarrow a^{-1}x\)

solution

To say that x → ax  is an endomorphism of (A,+) can be translated by:
a(x+y) = ax+ay which is nothing less than the distrivutivity of × over +.
If a is a unit the map \(x\rightarrow a^{-1}x\) is the inverse bijection of \(x\rightarrow ax\), which for this reason is an automorphism of (A,+).
If A=ℤ and a=2 The image of ℤ is the set of even numbers, so the map is not surjective.
If A denotes the set of all functions from ℝ into ℝ and if a is the map defined by :
a(x)= 0 if x  ≥ 0 and a(x)=1 if x <0
If f is defined this way :
f(x) = 1 if x  ≥ 0 and f(x)=0 else
and g thus defined :
g(x) = 2 if x  ≥ 0 and g(x)=0 else
then f ≠ g nevertheless af=ag=0 so \(f\rightarrow af\) is not injective.
