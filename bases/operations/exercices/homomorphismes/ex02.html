Soit G un groupe noté additivement.
Soit a un élément de G.
On désigne par ℤa l'ensemble des multiples entiers de a :
ℤa = { na ∈ G | n ∈ ℤ }

  • Montrer que ℤa est une partie stable pour l'addition.
  • Montrer que ℤa est la plus petite partie stable de G pour + contenant a et -a.
  • Montrer que ℤa avec l'addition induite par l'addition de G est un groupe additif
  • Montrer que l'aplication  h: n → na  de ( ℤ ,+) dans  ( ℤa, +) est un homomorphisme.

aide

Aucune difficulté particulière. Il suffit d'appliquer les définitions.

solution

ma+na =(m+n)a d'où la stabilité de ℤa pour +.
Soit S une partie stable contenant a et -a, alors S doit contenir a+(-a)=0 donc 0a.
Si S contient a alors S contient 1a qui est égal à a par définition.
Si S contient a alors S contient a+a=2a a+2a=3a et ainsi de suite donc, par récuurence tous les  na pour n ∈ ℕ.
Pour la même raison S contient les éléments de la forme (-n)a= n(-a) pour n ∈ ℕ ..
Donc S doit contenir ℤa.
L'opposé de na est (-n)a donc l'opposé de tout élément de ℤa est un élément de ℤa.
La dernière affirmation résulte de (m+n)a=ma+na.

G is a group with additive notation. For any a in G, we denote ℤa the set of all multiples of a :
ℤa = { na ∈ G | n ∈ ℤ }

  • Show that ℤa is a subset stable for addition.
  • Show that ℤa is the smallest subset of G stable for + containing a and -a.
  • Show that ℤa with addition induced by addition of G is an additive group.
  • Show that the map  h: n → na  from ( ℤ ,+) into  ( ℤa, +) is a homomorphism.

help

No special difficulty, just apply definitions.

solution

ma+na =(m+n)a from this the stability of ℤa for +.
Let S be a subset stable for + and containing a and -a, then S must contain a+(-a)=0 that is 0a.
If S contains a then S contains 1a which is equal to a by definition.
If S contains a then S contains a+a=2a a+2a=3a and so on, so, by induction all the na for n ∈ ℕ.
For the same reason S contains all the elements of the form (-n)a= n(-a) for n ∈ ℕ ..
So S must contain ℤa.
the opposite of na is (-n)a so the opposite of every element of ℤa is again an element of ℤa.
The last assertion results from (m+n)a=ma+na.
