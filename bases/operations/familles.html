Définitions

Somme d'une famille indexée:

On suppose que + désigne une sur un ensemble E.

Soit I un ensemble fini d'entiers et une application f:I → E (Une telle application s'appelle couramment une suite).

Nous utiliserons une notation un peu différente de la notation usuelle :

l'image de i par f dans E sera noté xi au lieu de f(i) (notation indicée).

définition
Nous poserons:
\(\sum_{i\in I}^{}x_{i}\) = 0 si I= ∅
xi si I={i}
x1+x2+...+xn si I={1,2,..n}
théorème
Avec ces notations, si I et J sont deux ensembles disjoints:
\(\sum_{i\in I\cup J}^{}x_{i}=\sum_{i\in I}^{}x_{i}+\sum_{i\in J}^{}x_{i}\)

Cette formule s'étend naturellement à une famille finie d'ensembles I1 , I2 ,...,In

NB1: Cette définition peut être étendue de façon naturelle à des ensembles I non nécessairement finis mais pour lesquels les xi sont tous nuls sauf pour un nombre fini de valeurs de i.

NB2: Cette définition peut être étendue en analyse à des ensembles quelconques pourvu qu'on ait étudié des notions comme les limites et les familles sommables (cela sort du cadre de cet exposé).

Produit d'une famille indexée:

On suppose maintenant que × désigne une loi et avec sur un ensemble E.
définition
Nous poserons:
\(\prod_{i\in I}^{}x_{i}\) = 1 si I= ∅
xi si I={i}
x1x2...xn si I={1,2,..n}
théorème
Avec ces notations, si I et J sont deux ensembles :
\(\prod_{i\in I\cup J}^{}x_{i}=\prod_{i\in I}^{}x_{i}\times \prod_{i\in J}^{}x_{i}\)

Cette formule s'étend naturellement à une famille finie d'ensembles disjoints I1 , I2 ,...,In

NB1: Cette définition peut être étendue de façon naturelle à des ensembles I non nécessairement finis mais pour lesquels les xi sont tous égaux à 1 sauf pour un nombre fini de valeurs de i.

NB2: Cette définition peut être étendue en analyse à des ensembles quelconques pourvu qu'on ait étudié des notions comme les limites et les produits infinis (cela sort du cadre de cet exposé).

Definitions

Sum of an indexed family :

We suppose that + denotes a on a set E.

I is a finite set of natural integers and f:I → E a mapping (Such a map is usually called a sequence).

We will use a notation a little different from the usual one :

The image of i by f in E will be denoted xi instead of f(i) (usual notation).

definition
Let us define
\(\sum_{i\in I}^{}x_{i}\) = 0 if I= ∅
xi if I={i}
x1+x2+...+xn if I={1,2,..n}
theorem
With the above conventions, if I and J are two disjoint sets :
\(\sum_{i\in I\cup J}^{}x_{i}=\sum_{i\in I}^{}x_{i}+\sum_{i\in J}^{}x_{i}\)

This formula can be naturally extended to a finite family of sets. I1 , I2 ,...,In

NB1: This definition can be extended naturally to sets I that are not necessarily finite but for which the xi are all zero except for a finite number of values ​​of i.

NB2: This definition can be extended in calculus to any set supposing that such subjects as 'limits' and 'infinite series' have been studied. This is out of the frame of this introductory lesson.

Product of an indexed family

We suppose now that × denotes a l and operation, with a on a set E.
definition
We will define :
\(\prod_{i\in I}^{}x_{i}\) = 1 if I= ∅
xi if I={i}
x1x2...xn if I={1,2,..n}
theorem
With these conventions, if I and J are two sets :
\(\prod_{i\in I\cup J}^{}x_{i}=\prod_{i\in I}^{}x_{i}\times \prod_{i\in J}^{}x_{i}\)

This formula can be naturally extended to a finite family I1 , I2 ,...,In of sets

NB1: This definition can be extended to infinite sets if we have the additional hypothesis that all the xi are 1, except a finite number of them, or to be more accurate for all values of i except a finite number of them.

NB2: The above definition can be extended to sets which are not finite using notions of calculus (convergence, infinite products).It is not our purpose to introduce these notions now.
