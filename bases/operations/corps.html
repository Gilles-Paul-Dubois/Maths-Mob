Définition

définition
Un 'corps' est un dont tous les éléments non nuls sont
Si l'anneau est commutatif, le corps lui-même est dit commutatif.

Quelques exemples:

  • ℚ, ℝ, ℂ sont des corps commutatifs.
  • ℤ est un anneau commutatif qui n'est pas un corps (seuls -1 et +1 sont inversibles).
  • L'ensemble H des quaternions (voir définition) est un corps infini qui n'est pas commutatif.

definition
A 'field' is a the non zero elements of which all have
If the ring is commutative, the field itself is said commutative.

A few examples :

  • ℚ, ℝ, ℂ are commutative fields.
  • ℤ is a commutative ring which is not a field (only -1 and +1 have inverses).
  • The set H of Hamilton's quaternions (see definition) is an infinite field which is not commutative.

Propriété importante

théorème
Il n'existe pas de corps fini non commutatif (théorème de Wedderburn 1905)

Ce résultat, dû à Wedderburn, est ici donné à titre indicatif sans démonstration.

Joseph Wedderburn - (1882/1948-UK)

Important property

theorem
Non commutative finite fields do not exist (theorem of Wedderburn 1905)

This result is given here for memory without any proof.

Joseph Wedderburn - (1882/1948-UK)
