définition
Soit X une partie de E, un élément y est appelé un 'majorant' de X si et seulement si tout élément de X est relié à lui.

Désignons par M l'ensemble des majorants de X.

y ∈ M ⇔ ∀ x ∈ X x ≤ y

Definitions

E is a set with a , which because of lack of symbols we will denote with ≤

E does not need to consist of numbers.

definition
X being a subset of E, an element y of E is called a 'majorant' of X if and only if every element of X is connected to it.

Let's denote by M the subset of all majorants of X.

y ∈ M ⇔ ∀ x ∈ X x ≤ y
definition
The same way a 'minorant' of X is an element y satisfying y ≤ x ∀ x de X.

Examples

If E = ℕ and if m <= n stands for n is a multiple of m

Then if X is any finite subset of numbers, M is the set of all common multiples of all elements of X that is the multiples of the LCM of all elements of X.
