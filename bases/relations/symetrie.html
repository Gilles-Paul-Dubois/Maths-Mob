Définition

définition
Nous dirons qu'une sur un ensemble E est 'symétrique' si dès qu'un élément est relié à un autre, cet autre est relié à lui.
R symétrique ⇔ ∀ x,y ∈E xRy ⇒ yRx

Cela veut dire que le graphe est symétrique par rapport à la diagonale.

Si la symétrie se voit facilement sur un graphe cartésien elle est beaucoup moins apparente sur un schéma saggital.

Definition

definition
We will say that a defined on a set E is 'symmetric' if as soon as an element x is connected to an element y this y is automatically connected to x
R symmetric ⇔ ∀ x,y ∈E xRy ⇒ yRx

It means that the graph of such a relation is symmetric with respect to the diagonal.

If symmetry is easily seen on a Cartesian diagram it's more difficult to notice on an arrow diagram.
