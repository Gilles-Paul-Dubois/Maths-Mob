Soit E un ensemble et A une partie de E.

On désigne par R A la relation sur ℘(E) définie par:
BR A C ⇔ B Δ C ⊆ A

Montrer que c'est une relation d'équivalence sur ℘(E).

Préciser la classe de B pour cette relation

aide

Montrer d'abord que:
(B-C ⊆ A) ∧ (C-D ⊆ A) ⇒ (B-D ⊆ A)

solution


La réflexivité et la symétrie sont évidentes sur la définition.
La transitivité s'exprime ainsi:
(B-C) ∪ (C-B) ⊆ A ∧ (C-D) ∪ (D-C) ⊆ A ⇒ (B-D) ∪ (D-B) ⊆ A
Il suffit donc de montrer que:
(B-C ⊆ A) ∧ (C-D ⊆ A) ⇒ (B-D ⊆ A)
On aura alors par permutation de B,C,D
(C-B ⊆ A) ∧ (D-C ⊆ A) ⇒ (D-B ⊆ A)
Et donc l'implication voulue.
Montrons donc :
(B-C ⊆ A) ∧ (C-D ⊆ A) ⇒ (B-D ⊆ A)
Avec l'hypothèse de gauche il faut montrer que si x ∈ B et x ∉ D alors x ∈ A
Considérons deux cas soit x ∈ C soit x ∉ C
Dans le premier cas x ∈ C-D donc x ∈ A
Dans le second cas x ∈ B-C donc x ∈ A
La relation R A est donc une équivalence.

C appartient à la classe de B équivaut à B ∪ C - B ∩ C ⊆ A
Ou encore à (B-C) ∪ (C-B) ⊆ A
Il est donc nécessaire que B-C ⊆ A et C-B ⊆ A
Ce qui revient C=(B-A) ∪ X avec X ⊆ A

E is a set and A a subset of E.

Let's denote by R A the relation on ℘(E) defined by:
BR A C ⇔ B Δ C ⊆ A

Show that it is an equivalence relation on ℘(E).

Find the class of B for this relation

hint

First show that :
(B-C ⊆ A) ∧ (C-D ⊆ A) ⇒ (B-D ⊆ A)

solution


Reflexivity and symmetry are quite obvious.
Transitivity can be translated by :
(B-C) ∪ (C-B) ⊆ A ∧ (C-D) ∪ (D-C) ⊆ A ⇒ (B-D) ∪ (D-B) ⊆ A
So it's enough to show that :
(B-C ⊆ A) ∧ (C-D ⊆ A) ⇒ (B-D ⊆ A)
We will get by permuting B,C,D
(C-B ⊆ A) ∧ (D-C ⊆ A) ⇒ (D-B ⊆ A)
so our conclusion
So, let us show that :
(B-C ⊆ A) ∧ (C-D ⊆ A) ⇒ (B-D ⊆ A)
With the left assumption we just need to show that if x ∈ B and x ∉ D then x ∈ A
Let's consider two cases x ∈ C either x ∉ C
In this first case x ∈ C-D hence x ∈ A
In the second case x ∈ B-C hence x ∈ A
So the relation R A is an equivalence

C belongs to the class of B means B ∪ C - B ∩ C ⊆ A
Or (B-C) ∪ (C-B) ⊆ A
So it is necessary that B-C ⊆ A and C-B ⊆ A
which comes to C=(B-A) ∪ X with X ⊆ A
