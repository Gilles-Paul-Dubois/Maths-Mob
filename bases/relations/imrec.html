Définition

définition
Soit B un ensemble et Y une partie de B.
On considère une R d'un ensemble A vers B.
' L' image réciproque de Y' est la partie de A constituée par tous les éléments dont les sont dans Y.

Cette partie se note R-1(Y).

x∈R-1(Y)⇔ ∃y∈Y | xRy

Quelques remarques:
L'image réciproque d'une partie peut être vide, même si cette partie n'est pas vide. Par contre l'image réciproque de la partie vide est toujours vide.
Si Y1 ⊆ Y2 alors l'image réciproque de Y1 est incluse dans l'image réciproque de Y2
L'image réciproque de {y} est l'ensemble des de y.

Exemples

Pour la relation x2 +y2 =1 de ℝ vers ℝ

  • L'image réciproque de ℝ est [-1,+1]
  • L'image réciproque de [-1,+1] est aussi [-1,+1]
  • L'image réciproque de toute partie contenant [-1,+1] est aussi [-1,+1]

Definition

definition
B is a set and Y a subset of B.
Let's consider some R from a set A to B.
' The preimage of Y' is the subset of A the elements of which have in Y.

This subset is denoted R-1(Y).

x∈R-1(Y)⇔ ∃y∈Y | xRy

Some remarks:
The preimage of a subset can be empty, even if this subset is not empty itself . But the preimage of the empty set is always empty.
If Y1 ⊆ Y2 then the preimage of Y1 is contained in the preimage of Y2
The preimage of {y} is the set of all of y.

Examples

For the relation x2 +y2 =1 from ℝ to ℝ

  • the preimage of ℝ is [-1,+1]
  • The preimage of [-1,+1] is also [-1,+1]
  • The preimage of any subset containing [-1,+1] is [-1,+1] as well.
