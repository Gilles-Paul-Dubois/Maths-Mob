Reflexivité

Reflexivity

Définition

définition
Nous dirons qu'une sur un ensemble E est 'réflexive' si tout élément est relié à lui-même.
R reflexive ⇔ ∀ x ∈E xRx

Cela veut dire que la diagonale est incluse dans le graphe.

Exemple de relation réflexive

Definition

definition
We will say that a on a set E is 'reflexive' if every element is connected to itself (and maybe to others...).
R réflexive ⇔ ∀ x ∈E xRx

It means that the diagonal is contained in the graph.
