Classes d'équivalence.

Soit une sur un ensemble E, que nous noterons, conformément aux usages par ≡.

définition
Si x est un élément de E nous appellerons 'classe d'équivalence' de x pour la relation ≡, l'ensemble des éléments qui lui sont reliés, c'est à dire encore, l'ensemble des images de x par ≡, ou encore l'image directe de {x} par ≡. Nous noterons cette classe C(x).

Voici une propriété importante:

x ≡ y ⇔ C(x)=C(y)

démonstration

Cela se démontre facilement. Si x ≡ y alors tout élément relié à x est relié à y par transitivité donc C(x) ⊆ C(y). L'inclusion en sens opposé se démontre de la même manière.

Une conséquence immédiate est la suivante:

Deux classes C(x), C(y) sont ou disjointes ou confondues.

démonstration

En effet dès que deux classes possèdent un élément commun tout élément de la première est relié à tout élément de la seconde par transitivité et vice-versa.
En outre les classes d'équivalence sont non vides.

En effet ∀ x ∈ E x ∈ C(x) par réflexivité.

Conclusion:

Les classes d'équivalence pour ≡ forment une de E.
définition
L'ensemble des classes constitue un ensemble appelé 'ensemble quotient de E par la relation ≡'.
définition
Si X est la classe d'équivalence de x pour ≡ ont dit aussi que x est un 'représentant' de X.

Quelques exemples

  • L'ensemble quotient pour la relation ".... à même couleur que ....." est l'ensemble des couleurs (rouge, bleu, vert, jaune, marron, violet, orange).
  • L'ensemble quotient pour la relation ".... à même forme que ....." est l'ensemble des formes (cercle, rectangle, triangle).
  • Prenons sur l'ensemble ℕ la relation:
    x ≡ y (mod r) si et seulement si x et y ont même reste dans la division par r (r désigne ici un entier strictement positif).
    Alors il est facile de voir qu'il y a exactement r classes d'équivalence qui sont:
    C(0), C(1), ......, C(r-1)
    Car le reste de la division de n'importe quel entier par r est forcément un des r nombres: 0,1,2,....,r-1.

Equivalence classes

Consider an on a set E, which, according to usual habits we will denote by ≡.

definition
If x is an element of E we will call 'equivalence class' of x for the relation ≡, the subset of E formed by all elements of E connected to x , that is to say the set of all images of x by ≡, or even the direct image of {x} by ≡.We will denote this class C(x).

Here is an important property:

x ≡ y ⇔ C(x)=C(y)

proof

This can be easily proven. If x ≡ y then every element connected to x is connected to y by transitivity y. Therefore C(x) ⊆ C(y). Reverse inclusion can be shown the same way.

It follows immediately that:

Two classes C(x), C(y) are disjoint or are the same.

proof

As soon as two classes have a common element every element of the first is connected to the second and conversely by transitivity.
Furthermore every equivalence class is non-empty.

In fact ∀ x ∈ E x ∈ C(x) by reflexivity.

Conclusion:

Equivalence classes for ≡ make a of set E.
definition
The set of all classes is itself a set called 'quotient set of E by the relation relation ≡'.
definition
If X is the equivalence class of x for ≡ we also say that x is a 'representative' of X.

A few examples

  • The quotient set of the relation".... has the same color as ....." is the set of colors (red, blue, green, yellow, brown, violet, orange).
  • The quotient set for the relation relation ".... has same shape as ....." is the set of all shapes (circle, rectangle, triangle).
  • Take on set ℕ the relation:
    x ≡ y (mod r) if and only if x and y have the same remainder in the division by (r denotes here a natural positive integer).
    It is easy to se that tehre are exactly r equivalence classes which are :
    C(0), C(1), ......, C(r-1)
    because the remainder of the division of any integer by r is one of the r numbers : 0,1,2,....,r-1.
