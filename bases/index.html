Pré-requis culturel

La lecture de ce module n'exige aucune connaissance mathématique préalable.

Pré-requis technique au 1° janvier 2022

Ce site peut être visité au moyen de tout ordinateur de bureau ou bien de tout mobile (tablette, téléphone) disposant d'un explorateur de web moderne (firefox, chrome, safari, opera, etc..).

Tous les programmes Python donnés en exemple sont copiables et exécutables sur votre machine si vous possédez un interprète Python (téléchargeable gratuitement).
Les sources ont été écrites pour la version  Python 3.4 https://www.python.org/downloads/

Nous pensons que le langage Julia est promis à un grand avenir. C'est pourquoi, pour ce module, tous les exemples donnés en Python ont été doublés en Julia avec la version actuelle 1.6 aussi bien pour le cours que pour les exercices de programmation.

A cette occasion nous avons vérifié la totale compatibilité ascendante de la version 3.4 de Python avec la version actuelle 3.8.

De nombreux programmes de démonstration ont été simplifiés et corrigés si nécessaire.

Il est possible de tester les exemples et de les modifier en ligne, par simple copier/coller sur le site de Jupyter. Il n'est donc pas nécessaire d'installer un EDI quelconque en mode local bien que cela puisse se faire justement à partir de leur site.

A ce jour il existe plusieurs éditeurs multi-langages (python/Julia) et multi plateformes (Windows, Linux, Mac) gratuits intéressants parmi lesquels Pycharm de Jetbrains et atom. Il suffit juste d'installer les plugins correspondants (juno pour atom, julia plugin pour jetbrains).

Anaconda fournit également un environnement Python récent et complet et donne accès à Jupyter si le package IJulia est installé, ainsi qu'à Pycharm.

N'hésitez pas à nous communiquer tout fonctionnement erratique des programmes donnés en exemple. D'avance merci !

Technical prerequisites on January 1, 2022

This site can be visited using any desktop computer or any mobile (tablet, phone) with a modern web browser (firefox, chrome, safari, opera, etc.).

All the Python programs given as examples can be copied and run on your machine if you have a Python interpreter (downloadable free). The sources were written for python 3.4

We believe that the Julia language has a great future. This is why, for this module, all the examples given in Python have been duplicated in Julia with the current version 1.6, both for the course and for the programming exercises.

On this occasion we checked the full backward compatibility of Python version 3.4 with the current version 3.8.

It is possible to test the examples and edit them online, by simply copying / pasting on the Jupyter's site . It is therefore not necessary to install any IDE in local mode although it can be done from their site.

Today there are several interesting free multi-language (python / Julia) and multi-platform (Windows, Linux, Mac) among which editors Pycharm de Jetbrains and atom. You just have to install the corresponding plugins (juno for atom, julia plugin for jetbrains).

Anaconda also provides a recent and complete Python environment and gives access to Jupyter if the IJulia package is installed, as well as Pycharm.

Do not hesitate to communicate to us any erratic functioning of the programs given as examples. Thanks in advance !
