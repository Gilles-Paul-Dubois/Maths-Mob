Pairs

Définitions

définition
Une 'paire' est un à deux éléments.
Conformément à ce qui a été dit précédemment, l'ordre dans lequel ces éléments sont écrits n'importe pas.
Les éléments sont écrits entre accolades comme pour n'importe quel ensemble.
On ne peut dans une paire, comme dans n'importe quel ensemble, répéter un élément.
Ne pas confondre la paire {a,b} avec le couple (a,b) .
Nous verrons ultérieurement la définition des couples (voir ). Dans un couple l'ordre est important (a,b)≠ (b,a).
En outre on peut, dans un couple, répéter un élément.
(a,a) est un couple parfaitement licite mais la paire {a,a} n'existe pas !

Definitions

definition
A 'pair' is a including exactly two elements.
In accordance with what has been said previously, the order in which these items are written does not matter .
The elements are written in braces as for any set.
We cannot repeat an element in a pair, as in any set .
Do not confuse the pair {a,b} with the ordered pair (a,b) .
We will see the definition of ordered pairs later (see ). In such a pair the order is important (a, b) ≠ (b, a).
In addition, one can repeat an element (a,a) is an ordered pair or a 2-uple.
(a, a) is a perfectly legal ordered pair but the pair {a, a} does not exist!
