Définitions

définition
On dit que " A est inclus dans l'ensemble B" (notation A ⊆ B ) si tout élément de A est un élément de B. On dit également dans ce cas que A est un 'sous-ensemble' de B ou encore une 'partie' de B.
Ainsi:
(A ⊆ B) ⇔ (∀ x x ∈ A ⇒ x ∈ B)
définition
Le symbole ⇒ représente l'implication logique et se lit 'implique', 'entraîne' ou 'force'.

Definitions

definition
We say that " A is included in set B " (notation A ⊆ B ) if any element of A is an element of B. We also say in this case that A is a 'subset' of B or a 'part' by B.
Thereby:
(A ⊆ B) ⇔ (∀ x x ∈ A ⇒ x ∈ B)
definition
The symbol ⇒ represented logical implication and reads 'implies', 'drives' or 'forces'.
D'une façon générale, la négation d'une proposition :
∀ x P(x) où P est une propriété quelconque
s'énonce:
∃ x |¬ P(x) où ¬ P est la négation de P

Dans notre cas la négation A non inclus dans B (A ⊄ B) s'exprime:

∃ x | x ∈ A ∧ x ∉ B (∧ désigne ici le symbole logique ET)

définition
A ⊂ B, inclusion stricte, signifie A ⊆ B et A ≠ B
In general, the negation of a proposition:
∀ x P (x) where P is any property
reads:
∃ x | ¬ P (x) where ¬ P is the negation of P

In our case the negation A not included in B (A ⊄ B) is expressed:

∃ x | x ∈ A ∧ x ∉ B (∧ here denotes the logical symbol AND)

definition
A ⊂ B, strict inclusion, means A ⊆ B and A ≠ B

Propriétés

Pour tout ensemble A, on a: ∅ ⊆ A.

En effet, la négation impliquerait l'existence d'un élément x dans ∅.

Réflexivité:

Pour tout ensemble A, on a A ⊆ A.

Transitivité

(A ⊆ B) ∧( B ⊆ C) ⇒ (A ⊆ C).

Antisymétrie

(A ⊆ B) ∧( B ⊆ A) ⇒ (A=B).

Ces 3 propriétés sont à peu près évidentes compte tenu des définitions.

La troisième est utilisée en pratique pour démontrer l'égalité de deux ensembles.

Pour démontrer que A=B, on montre que x ∈ A ⇒ x ∈ B puis l'implication en sens inverse.

Properties

For any set A, we have: ∅ ⊆ A.

Indeed, the negation would imply the existence of an element x in ∅.

Reflexivity:

For any set A, we have A ⊆ A.

Transitivity

(A ⊆ B) ∧( B ⊆ C) ⇒ (A ⊆ C).

Antisymmetry

(A ⊆ B) ∧( B ⊆ A) ⇒ (A=B).

These 3 properties are pretty much obvious given the definitions.

The third is used in practice to demonstrate the equality of two sets.

To prove that A = B, we show that x ∈ A ⇒ x ∈ B then the reverse implication .
