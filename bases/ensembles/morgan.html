theorem

If A, B, and C are 3 , the so-called laws of 'De Morgan' can be stated as follows:

  • A-(B ∩ C) =(A-B) ∪ (A-C)
  • A-(B ∪ C) = (A-B) ∩ (A-C)

Ce qui revient à dire que la passage au complémentaire 'inverse' les opérations de réunion et d'intersection.
La démonstration repose sur une propriété analogue pour les opérateurs logiques:
  • NON: ¬
  • OU: ∨
  • ET: ∧

Montrons par exemple la première:

démonstration

x ∈ A-(B ∩ C) ⇔ x ∈ A ∧ x ∉ (B ∩ C)
x ∉ (B ∩ C) ⇔ ¬ (x ∈ B ∧ x ∈ C)
¬ (x ∈ B ∧ x ∈ C) ⇔ (¬( x ∈ B)) ∨ (¬ (x ∈ C)) ⇔ (x ∉ B ∨ x ∉ C)
donc
x ∈ A-(B ∩ C) ⇔ x ∈ A ∧ (x ∉ B ∨ x ∉ C)
et pour finir par distributivité de ∧ sur ∨
x ∈ A ∧ (x ∉ B ∨ x ∉ C) ⇔ ( x ∈ A ∧ x ∉ B ) ∨ ( x ∈ A ∧ x ∉ C ) ⇔ x ∈ (A-B) ∪ (A-C)

Which amounts to saying that the transition to the complementary 'reverse' the join and intersect operations.
The proof is based on an analogous property for logical operators :
  • NO: ¬
  • OR: ∨
  • AND: ∧

Let us show for example the first:

proof

x ∈ A-(B ∩ C) ⇔ x ∈ A ∧ x ∉ (B ∩ C)
x ∉ (B ∩ C) ⇔ ¬ (x ∈ B ∧ x ∈ C)
¬ (x ∈ B ∧ x ∈ C) ⇔ (¬( x ∈ B)) ∨ (¬ (x ∈ C)) ⇔ (x ∉ B ∨ x ∉ C)
so
x ∈ A-(B ∩ C) ⇔ x ∈ A ∧ (x ∉ B ∨ x ∉ C)
and finally by distributivity of ∧ on ∨
x ∈ A ∧ (x ∉ B ∨ x ∉ C) ⇔ ( x ∈ A ∧ x ∉ B ) ∨ ( x ∈ A ∧ x ∉ C ) ⇔ x ∈ (A-B) ∪ (A-C)

The second formula is demonstrated in a similar way.
