définition
Un 'singleton' est un comportant un seul élément.
Donc :
y ∈ {x} ⇔ y=x
Mais aussi :
{x}={y} ⇔ x=y

Par ailleurs, ne pas confondre l'élément x et le singleton {x} ! Nous verrons plus tard ( ) que les notions sont bien distinctes.

definition
A 'singleton' is a comprising a single element .
Therefore :
y ∈ {x} ⇔ y = x
But also :
{x} = {y} ⇔ x = y

And please don't mix the element x with the set {x} !

We'll see later ( ) that the concepts are quite distinct.
