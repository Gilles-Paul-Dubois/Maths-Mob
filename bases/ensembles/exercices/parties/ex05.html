Montrez que (X ⊆ Y) ⇔ ( ℘(X) ⊆ ℘(Y))

aide

Montrez que (X ⊆ Y) ⇒ ( ℘(X) ⊆ ℘(Y)) puis que (X ⊆ Y) ⇐ ( ℘(X) ⊆ ℘(Y))

solution

Supposons que (X ⊆ Y) Alors toute partie A de X est une partie de Y par transitivité. Donc A ∈ ℘(X) ⇒ A ∈ ℘(Y), soit ℘(X) ⊆ ℘(Y).

Supposons réciproquement que ℘(X) ⊆ ℘(Y). Comme X ∈ ℘(X) on a : X ∈ ℘(Y), donc X ⊆ Y, CQFD.

Show that (X ⊆ Y) ⇔ ( ℘(X) ⊆ ℘(Y))

hint

Show first that (X ⊆ Y) ⇒ ( ℘(X) ⊆ ℘(Y)) and then that (X ⊆ Y) ⇐ ( ℘(X) ⊆ ℘(Y))

solution

Suppose first that (X ⊆ Y) Then every subset A of X is a subset of Y by transitivity. So A ∈ ℘(X) ⇒ A ∈ ℘(Y), or ℘(X) ⊆ ℘(Y).

Suppose reciprocally that ℘(X) ⊆ ℘(Y). Since X ∈ ℘(X) we have : X ∈ ℘(Y), hence X ⊆ Y, QED.
