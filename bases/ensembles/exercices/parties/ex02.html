Démontrez que dans tout ensemble E non vide il y a autant de parties comportant un nombre pair d'éléments que de parties comportant un nombre impair d'éléments.

aide

Commencez par étudier le cas d'un ensemble comportant un nombre impair d'éléments.

solution

Si E comporte un nombre impair d'éléments, il est clair que le complémentaire d'une partie paire est une partie impaire et réciproquement. Il y en a donc autant de chaque sorte.
Si E comporte un nombre pair d'éléments, soit x un élément de E et F= E-{x}. F comporte donc un nombre impair d'éléments.
Les parties de E sont constituées:
  • des parties contenant l'élément x, ensemble P1
  • des parties ne le contenant pas, ensemble P2
Les parties du premier type se divisent elles mêmes en parties impaires (ensemble P1,1) et parties paires (ensemble P1,2).
Même chose pour les parties du second type. Ensemble P2,2 : parties paires de P2. Ensemble P2,1 parties impaires de P2.
Désignons par n le nombre d'éléments de E.
Donc n-1 est le nombre d'éléments de F.
F comporte un nombre impair d'éléments.
Donc P2,2 comporte 2 n-2 éléments, tout comme P2,1
Quand aux parties paires de P1, elles sont obtenues par adjonction de l'élément x aux parties impaires de F, de la même façon que les parties impaires sont obtenues par adjonction de x aux parties paires de F.
En définitive:
Le nombre d'éléments de P2 est 2 n-2+2 n-2= 2 n-1
et il en est de même du nombre d'éléments de P1

Show that in any not empty set E there are as many subsets with a even number of elements than of subsets comprising a odd number of elements.

help

Start by studying the case of a set with an odd number of elements.

solution

If E has an odd number of elements, it is clear that the complementary to an even subset is an odd subset and reciprocally. There are therefore as many of each kind.
If E has an even number of elements, let x be an element of E and F = E-{x}. F therefore has an odd number of elements.
The subsets of E are made up of:
  • subsets containing the element x, set P1
  • subsets not containing it, set P2
The subsets of the first type divide themselves into odd subsets (set P1,1 ) and even subsets (set P1,2 ).
The same goes for the subsets of the second type. Set P2.2 : even subsets of P2 . Set P2.1 subsets odd of P2 .
Denote by n the number of elements of E.
So n-1 is the number of elements of F.
F has an odd number of elements.
So P2.2 has 2 n-2 elements, just like P2.1
As for the even subsets of P1 , they are obtained by addition of the element x to the odd subsets of F, in the same way as the odd subsets are obtained by adding x to the even subsets by F.
Ultimately:
The number of elements of P2 is 2 n-2 +2 n-2 = 2 n-1
and it is the same for the number of elements of P1
