En utilisant l'exercice précédent calculer le nombre B(n) (nombres de Bell) de partitions d'un ensemble à n éléments en fonction des nombres de Stirling S(n,p).

Écrire un programme calculant B(n).

aide

Les partitions de E peuvent être regroupées suivant le nombre de sous-ensembles de la partition.
P(1) : Partitions à une seule partie (une seule)
P(2): Partitions à 2 parties
.....................
P(p): Partitions à p parties
....................
P(n-1): Partitions à n-1 parties (nombre: C 2 n)
P(n) Partitions à n parties (une seule)

solution

Le nombre d'éléments de P(p) est par définition S(n,p). (nombres de Stirling de seconde espèce)
Donc B(n)= S(n,1)+S(n,2)+ ... + S(n-1,n)+S(n,n)

Using the previous exercise calculate the number B(n) (Bell numbers) of partitions of a set with n elements as a function of Stirling numbers S(n,p).

Write a program that calculates B(n).

hint

The partitions of E can be grouped according to the number of subsets of the partition.
P(1): Single part (one)
P(2): 2-part
.....................
P(p): Partitions in p parts
....................
P(n-1): Partitions with n-1 parts (number: C 2 n )
P(n) Partitions with n parts (only one)

solution

The number of elements of P(p) is by definition S(n,p). (numbers of Stirling of the second kind)
So B(n) = S(n,1) + S(n,2) + ... + S(n-1,n) + S(n,n)
