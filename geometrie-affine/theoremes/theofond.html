proof

Let us show that σ is additive, using the additivity of u:
Let   \( \displaystyle \overrightarrow{x} \)   a nonzero vector, so that   \( \displaystyle u\left ( \overrightarrow{x} \right ) \)   is also non-zero.

\( \displaystyle u\left ( (\lambda +\mu )\overrightarrow{x} \right )=\sigma (\lambda +\mu )u\left ( \overrightarrow{x} \right )=u\left ( \lambda \overrightarrow{x} \right )+u\left ( \mu \overrightarrow{x} \right )=\left ( \sigma (\lambda ) +\sigma (\mu )\right )\overrightarrow{x} \)

from where σ(λ+μ)=σ(λ)+σ(μ) since   \( \displaystyle u\left ( \overrightarrow{x} \right ) \)   is not zero.
On the other hand :

\( \displaystyle u\left ( (\lambda \mu ) \overrightarrow{x}\right )=\sigma (\lambda \mu)u\left ( \overrightarrow{x} \right )=\sigma (\lambda )u\left ( \mu \overrightarrow{x} \right )=\sigma (\lambda )\sigma (\mu )u\left ( \overrightarrow{x} \right ) \)

by applying the definition of σ and axioms for the definition of vector spaces.
This completes the proof of the theorem.
