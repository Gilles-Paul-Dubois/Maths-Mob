Pour la conique suivante, déterminer la nature, les éléments caractéristiques et une équation réduite :

\( \displaystyle \frac{x^{2}}{4} - \frac{\sqrt{3}}{2}{xy} + \frac{3}{4}y^{2} - \left( {1 + 3\sqrt{3}} \right)x - \left( {3 - \sqrt{3}} \right)y + 13 = 0 \)

aide

Effectuer une rotation d'angle π/6.

solution

Les formules de changement de repère dans une rotation d'angle π/6 sont :

\( \displaystyle \left\{ \begin{matrix} {x = \frac{\sqrt{3}}{2}X - \frac{1}{2}Y} \\ {y = \frac{1}{2}X + \frac{\sqrt{3}}{2}Y} \\ \end{matrix} \right. \)

En remplaçant on obtient l'équation :
Y²-6X+2Y+13=0
Soit encore

\( \displaystyle \left( {Y + 1} \right)^{2} = 6\left( {X - 2} \right) \)

C'est donc une parabole de sommet S(2,-1) dans le nouveau repère, le paramètre est p=3.
