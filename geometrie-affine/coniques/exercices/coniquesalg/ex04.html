Démontrer la validité du tableau décisionnel de l'équation réduite sans terme en xy, au moyen d'une discussion fondée sur une étude de cas exhaustive.

aide

Effectuer une translation :
x=X+c
y=Y+d
Ecrire l'équation transformée.
Ajuster c et d pour assurer la disparition des termes linéaires en X et Y.

solution

Si x=X+c
y=Y+d
alors l'équation dans le repère de centre Ω(c,d) est :
A(X+c)²+C(Y+d)²+D(X+c)+E(Y+d)+F=0
soit encore :
AX²+CY²+(2Ac+D)X+(2Cd+E)Y+(F+Ac²+Cd²+Dc+Ed)=0

Discussion :

Si AC≠0

Prenons c=-D/2A et d=-E/2C
L'équation est donc
AX²+CY²=N où
N=D2/4A+E2/4C-F
Il y a alors 3 cas possibles pour N

N>0

Si A et C sont tous deux positifs
La courbe est une ellipse dont l'équation rapportée à ses axes de symétrie est :

\( \displaystyle \frac{X^{2}}{\frac{N}{A}} + \frac{Y^{2}}{\frac{N}{C}} = 1 \)

Les demi-axes sont :

\( \displaystyle a = \sqrt{\frac{N}{A}} \)

et   \( \displaystyle b = \sqrt{\frac{N}{C}} \)  
Si A et C sont tous deux négatifs
L'ensemble est vide
Si A et C sont de signes opposés
La courbe est une hyperbole

N=0

L'équation est AX²+CY²=0
Si A et C sont de même signe
L'équation est satisfaite seulement pour X=Y=0 (le seul point Ω).
Si A et C sont de signes contraires
Le membre de gauche de l'équation se factorise et l'ensemble est un couple de droites sécantes en Ω.

N<0

Si maintenant AC=0

Il existe alors 3 possibilités

Cas A=0,C≠0

On distingue à nouveau 2 cas possibles
D≠0
On peut alors choisir c et d de sorte que :
Cd+E=0 et Cd²+Dc+Ed+F=0
L'équation devient Y²=-2(D/C)X et la courbe est une parabole.
D=0
L'équation est du second degré en Y, elle représente donc un couple de droites parallèles. Elles coïncident si E²/4-FC=0.

Cas A≠0,C=0

Symétrique du cas précédent E jouant le rôle de D.

Cas A=0,C=0

l'équation est alors DX+EY+DC+Ed=0
C'est une droite si (D,E)≠(0,0).
Sinon l'équation est F=0.
On obtient alors le plan entier ou bien ∅ selon que F est nul ou non.

Demonstrate the validity of the xy-term reduced equation decision chart, through discussion based on an extensive case study.

hint

Perform a translation:
x=X+c
y=Y+d
Write the transformed equation.
Adjust c and d to ensure the disappearance of linear terms in X and Y.

solution

Whether x=X+c
y=Y+d
then the equation in the center frame Ω(c,d) is:
A(X+c)²+C(Y+d)²+D(X+c)+E(Y+d)+F=0
or again:
AX²+CY²+(2Ac+D)X+(2Cd+E)Y+(F+Ac²+Cd²+Dc+Ed)=0

Discussion:

If AC≠0

Take c=-D/2A and d=-E/2C
So the equation is
AX²+CY²=N where
N=D2/4A+E2/4C-F
There are then 3 possible cases for N

N>0

If A and C are both positive
The curve is an ellipse whose equation related to its axes of symmetry is:

\( \displaystyle \frac{X^{2}}{\frac{N}{A}} + \frac{Y^{2}}{\frac{N}{C}} = 1 \)

The semi-axes are:

\( \displaystyle a = \sqrt{\frac{N}{A}} \)

And   \( \displaystyle b = \sqrt{\frac{N}{C}} \)  
If A and C are both negative
The set is empty
If A and C have opposite signs
The curve is a hyperbola

N=0

The equation is AX²+CY²=0
If A and C have the same sign
The equation is satisfied only for X=Y=0 (the only point Ω).
If A and C have opposite signs
The left side of the equation is factorized and the set is a pair of secant lines at Ω.

N<0

If now AC=0

There are then 3 possibilities

Case A=0,C≠0

We again distinguish 2 possible cases
D≠0
We can then choose c and d so that:
Cd+E=0 and Cd²+Dc+Ed+F=0
The equation becomes Y²=-2(D/C)X and the curve is a parabola.
D=0
The equation is quadratic in Y, so it represents a pair of parallel lines. They coincide if E²/4-FC=0.

Case A≠0,C=0

Symmetric of the previous case E playing the role of D.

Case A=0,C=0

the equation is then DX+EY+DC+Ed=0
It is a line if (D,E)≠(0,0).
Otherwise the equation is F=0.
We then obtain the entire plan or ∅ whether F is zero or not.
