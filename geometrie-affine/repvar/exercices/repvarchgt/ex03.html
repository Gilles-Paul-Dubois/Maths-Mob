Soit A l'espace vectoriel ℂ2 muni de sa structure d'espace affine naturelle.
Soit B1 la base formée des 2 vecteurs :
\( \displaystyle \overrightarrow{u_{1}}=(i,0) \)
\( \displaystyle \overrightarrow{v_{1}}=(0,1-i) \)
Soit O1 le point (1+i,2i).
Et soit enfin R1 le repère (O1,B1)
Soit B2 la base formée des 2 vecteurs :
\( \displaystyle \overrightarrow{u_{2}}=(0,i) \)
\( \displaystyle \overrightarrow{v_{2}}=(1,0) \)
Soit O2 le point (1-i,i).
Et soit enfin R2 le repère (O2,B2)
Soit M le point ayant pour coordonnées (2,3i) dans R1.
Quelles sont ses coordonnées dans R2 ?
(On pourra faire le calcul avec un programme).

aide

Voici un programme Python donnant la solution : 

# -*- coding: utf-8 -*-
# Changement de repère
from reperes import *

def ex03():
    u1=VecteurKn([1j,0])
    v1= VecteurKn([0,1-1j])
    B1=([u1,v1])
    O1=PointKn([1+1j,2j])
    R1=Repere(O1,B1)

    u2=VecteurKn([0,1j])
    v2= VecteurKn([1,0])
    B2=([u2,v2])
    O2=PointKn([1-1j,1j])
    R2=Repere(O2,B2)


    M=R1.Point([2,3j])
    X= R2.Coord(M)
    for x in X:
        print x

def main():
    ex03()

if __name__ == "__main__":
    main()

solution

La réponse est (4-3i,4i)
