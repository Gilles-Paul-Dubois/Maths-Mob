Cet exercice est identique à l'exercice 01 sauf que l'on a remplacé le corps ℝ par le corps ℤ/5ℤ.
Soit A l'espace vectoriel (ℤ/5ℤ)3 muni de sa structure d'espace affine naturelle.
Soit B1 la base formée des 3 vecteurs :
\( \displaystyle \overrightarrow{u_{1}}=(1,0,0) \)
\( \displaystyle \overrightarrow{v_{1}}=(1,1,0) \)
\( \displaystyle \overrightarrow{w_{1}}=(1,1,1) \)
Soit O1 le point (1,2,1).
Et soit enfin R1 le repère (O1,B1)
Soit B2 la base formée des 3 vecteurs :
\( \displaystyle \overrightarrow{u_{2}}=(0,1,1) \)
\( \displaystyle \overrightarrow{v_{2}}=(1,0,1) \)
\( \displaystyle \overrightarrow{w_{2}}=(1,1,1) \)
Soit O2 le point (2,1,2).
Et soit enfin R2 le repère (O2,B2)
Soit M le point ayant pour coordonnées (2,3,4) dans R1.
Quelles sont ses coordonnées dans R2 ?
(On pourra faire le calcul avec un programme).

aide

Voici un programme Python qui résout le problème : 

# -*- coding: utf-8 -*-
# Changement de repère
from reperes import *

def ex02():
    C5=GenZp(5)
    u1=VecteurKn([C5(1),C5(0),C5(0)])
    v1= VecteurKn([C5(1),C5(1),C5(0)])
    w1= VecteurKn([C5(1),C5(1),C5(1)])
    B1=([u1,v1,w1])
    O1=PointKn([C5(1),C5(2),C5(1)])
    R1=Repere(O1,B1)
    u2=VecteurKn([C5(0),C5(1),C5(1)])
    v2= VecteurKn([C5(1),C5(0),C5(1)])
    w2= VecteurKn([C5(1),C5(1),C5(0)])
    B2=([u2,v2,w2])
    O2=PointKn([C5(2),C5(1),C5(2)])
    R2=Repere(O2,B2)
    M=R1.Point([C5(2),C5(3),C5(4)])
    X= R2.Coord(M)
    for x in X:
        print x

def main():
    ex02()

if __name__ == "__main__":
    main()

solution

Les coordonnées de M dans R2 sont (4,4,4).
