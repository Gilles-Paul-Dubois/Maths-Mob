Trouver les invariants de l'application du plan affine réel dans lui-même qui à tout point M d'affixe z associe le point M' d'affixe z' où z'=az+b (a et b nombres complexes quelconques).
Même question pour l'application z'=a z

aide

Résoudre l'équation z'=z dans les deux cas.

solution

Cas de l'application z'=az+b

Examinons d'abord le cas particulier a=1

Si b=0 tout point est fixe (application identique)
Si b≠0 aucun point n'est fixe (translation de vecteur non nul)

Si maintenant a ≠ 1

l'équation z=az+b possède une unique solution z=b/(1-a)

Cas de l'application z'=a z

L'équation à résoudre est z=a z et possède toujours la solution z=0.
Pour avoir d'autres solutions il est nécessaire que |a| soit égal à 1.
Dans ce cas a=eiθ
Soit alors α l'argument de z.
On a z=e-2iαz
L'équation z=a z s'écrit donc :
z=eiθe-2iαz
Soit encore ei(θ-2α) = 1
Ce qui équivaut donc à α=θ/2 .
On trouve donc une droite passant par l'origine.

Find the invariants of the map of the real affine plane in itself which to any point M with affix z associates the point M' with affix z' where z'=az+b (a and b complex numbers).
Same question for map z'=a z

hint

Solve the equation z'=z in both cases.

solution

Case of the application z'=az+b

Let's first examine the special case a=1

If b=0 any point is fixed (identity map)
If b≠0 no point is fixed (nonzero vector translation)

If now a ≠ 1

the equation z=az+b has a unique solution z=b/(1-a)

Case of application z'=a z

The equation to solve is z=a z and always has the solution z=0.
To have other solutions it is necessary that |a| is equal to 1.
In this case a=eiθ
Let then α z's argument.
We have z=e-2iαz
The equation z=a z is therefore written:
z=eiθe-2iαz
Let ei(θ-2α) = 1
Which is therefore equivalent to α=θ/2 .
We therefore find a straight line passing through the origin.
