On considère une épreuve aléatoire n'ayant que deux issues possibles:

  • Un évènement de probabilité p (succès) (o<p<1).
  • L'évènement contraire de probabilité q=1-p (échec).
On répète n fois cette expérience (schéma de Bernoulli).
On considère la variable aléatoire X dont la valeur est le nombre de succès.
  • Quelle est l'espérance de X ?
  • Quelle est la variance de X ?
  • Montrer que pour tout nombre positif ε on a   \( \displaystyle P\left ( \left | \frac{X}{n}-p \right |< \varepsilon \right ) \leqslant \frac{pq}{\varepsilon ^{2}} \)  

aide

X peut prendre chacune des valeurs 0,1, .... ,n.
La loi de X est donnée par :

\( \displaystyle P(X = k) = C_{n}^{k}p^{k}q^{n - k} \)

solution

Un calcul rapide donne :
E(X)=np
v(X)=npq
L'espérance de X/n est p et sa variance pq/n .
L'inégalité de Tchebycheff donne:

\( \displaystyle P\left( {{\left |{\frac{X}{n} - p}\right |} < \epsilon} \right) \leq \frac{pq}{\epsilon^{2}} \)

We consider a random test with only two possible outcomes:

  • An event with probability p (success) (o<p<1).
  • The contrary event with probability q=1-p (failure).
We repeat this experiment n times (Bernoulli diagram).
We consider the random variable X whose value is the number of successes.
  • What is the expectation of X?
  • What is the variance of X?
  • Show that for any positive number ε we have   \( \displaystyle P\left ( \left | \frac{X}{n}-p \right |< \varepsilon \right ) \leqslant \frac{pq}{\varepsilon ^{2}} \)  

X can take each of the values 0,1, ...., n.
The law of X is given by:

\( \displaystyle P(X = k) = C_{n}^{k}p^{k}q^{n - k} \)

hint

solution

A quick calculation gives:
E(X)=np
v(X)=npq
The expectation of X/n is p and its variance pq/n.
Chebyshev's inequality gives:

\( \displaystyle P\left( {{\left |{\frac{X}{n} - p}\right |} < \epsilon} \right) \leq \frac{pq}{\epsilon^{2}} \)
