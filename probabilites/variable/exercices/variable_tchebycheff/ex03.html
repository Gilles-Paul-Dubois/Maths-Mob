Soit X une variable aléatoire réelle et

\( \displaystyle \left. g:{\mathbb{R}}^{+}\rightarrow{\mathbb{R}}^{+} \right. \)

une fonction strictement croissante.
Montrer que ∀ a ≥ 0 on a :

\( \displaystyle P\left( {{|X|} \geq a} \right) \leq \frac{E\left( {g\left( {|X|} \right)} \right)}{g(a)} \)

aide

Utiliser l'inégalité de Markov.

solution

g étant strictement croissante on a :

\( \displaystyle \left. {|X|} \geq a\ \Leftrightarrow\ g\left( {|X|} \right) \geq g(a) \right. \)

L'inégalité de Markov donne alors :

\( \displaystyle E\left ( g\left ( \left | X \right | \right ) \right )\geqslant g(a)P\left ( g\left ( \left | X \right |\right ) \geqslant g(a)\right ) \)

D'où nous tirons l'inégalité cherchée.
