Ceci est une vérification expérimentale, d'un résultat démontré dans le cours. Ecrire un programme python et/ou consacré au jeu de bridge, qui calcule de façon expérimentale, au moyen de fréquences, les probabilités pour que:

  • La valeur d'une main soit entre v1 et v2 (v1 et v2 deux nombres donnés à l'avance entre 0 et 40).
  • Le nombre de couleurs d'une main soit entre c1 et c2 (c1 et c2 deux nombres donnés à l'avance entre 1 et 4).
Constater que les deux variables :
  • X : valeur en points d'honneurs d'une main
  • Y : Nombre de couleurs distinctes d'une main
ne sont pas indépendantes.
Montrer cependant que pour des valeurs courantes valeurs entre 10 et 15, couleurs entre 3 et 4, les deux variables sont très faiblement corrélées.

Écrire pour cette dernière classe une méthode calculant le nombre de couleurs distinctes de la main, et une méthode calculant la valeur de la main en points d'honneurs.

This is an experimental verification of a result demonstrated in the course. Write a Python program and/or dedicated to the game of bridge, which experimentally calculates, using frequencies, the probabilities so that:

  • The value of a hand is between v1 and v2 (v1 and v2 two numbers given in advance between 0 and 40).
  • The number of suits in a hand is between c1 and c2 (c1 and c2 two numbers given in advance between 1 and 4).
Note that the two variables:
  • X: value in honor points of a hand
  • Y: Number of distinct colors on a hand
are not independent.
Show however that for current values values between 10 and 15, colors between 3 and 4, the two variables are very weakly correlated.

hint

For Python:

Write a 'card' class with 'suit' and 'figure' data members, a 'value' method.
Write a 'mainB' class (bridge hand) with a random constructor defining 13 distinct cards among 52.
Write for this last class a method calculating the number of distinct suits of the hand, and a method calculating the value of the hand in honor points.
