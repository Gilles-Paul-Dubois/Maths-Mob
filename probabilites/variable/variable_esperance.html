Cas des probabilités à densité

Nous pouvons nous inspirer du calcul de la moyenne d'un caractère donné par des classes. La contribution à l'espérance de l'évènement x < X ≤ X+dx sera x(F(x+dx)-F(x)) où F est la fonction de répartition de X; si et seulement si F a pour densité f cela donne xf(x)dx. En faisant tendre dx vers 0 et par sommation nous obtiendrons :

\( \displaystyle E(X) = {\int_{- \infty}^{+ \infty}{xf}(x){dx}} \)

qui est une que nous prendrons comme définition de l'espérance d'une v.a. absolument continue.
Unlike the finite case, the hope of a such random variable may not be defined. This will be the case if we choose, for example, a real k > p in the previous example.

Case of probabilities with density

We can take inspiration from calculating the average of a given character by classes. The contribution to the expectation of the event x < X ≤ X+dx will be x(F(x+dx)-F(x)) where F is the distribution function of X; if and only if F has density f this gives xf(x)dx. By making dx tend towards 0 and by summation we will obtain:

\( \displaystyle E(X) = {\int_{- \infty}^{+ \infty}{xf}(x){dx}} \)

Quelques propriétés de l'espérance mathématique

Les propriétés suivantes se vérifient immédiatement.

Some properties of mathematical expectation

The following properties are immediately verified.
propriété 2
Si X est la fonction caractéristique d'un évènement A alors l'espérance de X est P(A).
property 2
If X is the characteristic function of an event A then the expectation of X is P(A).
propriété 3
Si X est une v.a. constante k, son espérance existe et est égale à k.
property 3
If X is a constant r.v. k, its expectation exists and is equal to k.
propriété 4
Si X est d'espérance E(X), alors pour toute constante λ , la v.a. λX a pour espérance λE(X).
property 4
If X has expectation E(X), then for any constant λ , the r.v. λX has the expectation λE(X).
propriété 5
Si X et Y sont deux variables aléatoires sur le même espace probabilisé alors :
E(X+Y)=E(X)+E(Y)
property 5
If X and Y are two random variables on the same probability space then:
E(X+Y)=E(X)+E(Y)

Centered variables

définition 4
Une v.a. est dite "centrée" si elle possède une espérance mathématique et si cette espérance est nulle.
definition 4
A r.v. is said to be “centered” if it has a mathematical expectation and if this expectation is zero.
propriété 7
Si X est une variable possédant l'espérance E(X) alors la variable X-E(X) est centrée.
property 7
If X is a variable having the expectation E(X) then the variable X-E(X) is centered.
