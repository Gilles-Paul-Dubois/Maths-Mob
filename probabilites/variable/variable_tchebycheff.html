Le théorème

Commençons par un lemme technique :

The theorem

Let's start with a technical lemma:
lemme 1
Soit X une variable aléatoire sur un espace probabilisé (Ω, \( \mathfrak{T}\),P). On suppose que l'on est dans l'un des deux cas suivants :
  • Ω est un ensemble fini ou dénombrable
  • X est absolument continue et a une densité continue
et que X possède une espérance E(X). Dans ces conditions si X ≥ 0 alors E(X) ≥ 0.
lemma 1
Let X be a random variable on a probability space (Ω, \( \mathfrak{T}\),P). We assume that we are in one of the following two cases:
  • Ω is a finite or countable set
  • X is absolutely continuous and has continuous density
and that X has an expectation E(X). Under these conditions if X ≥ 0 then E(X) ≥ 0.
Le résultat est évident dans le premier cas.
Dans le second cas il est clair que la fonction f est nulle sur ]-∞,0[ car si f(x0) > 0 pour un certain x0 <0 alors par continuité f sera positive sur un intervalle [a,b] contenant x0 et contenu dans ]-∞,0[ on aura donc

\( \displaystyle P\left( {X \in \lbrack a,b\rbrack} \right) = {\int_{a}^{b}f(x)}{dx} > 0 \)

ce qui contredit X ≥ 0.
Nous supposons désormais que toutes les variables aléatoires considéres vérifient les conditions de notre lemme. De là nous tirons l'inégalité de Markov :
The result is obvious in the first case.
In the second case it is clear that the function f is zero on ]-∞,0[ because if f(x0) > 0 for a certain x0 <0 then by continuity f will be positive on an interval [a,b] containing x0 and contained in ]-∞,0[ we will therefore have

\( \displaystyle P\left( {X \in \lbrack a,b\rbrack} \right) = {\int_{a}^{b}f(x)}{dx} > 0 \)

which contradicts X ≥ 0.
We now assume that all the random variables considered satisfy the conditions of our lemma. From here we derive the Markov inequality:
théorème 1
∀ a > 0

\( \displaystyle P\left( {X \geq a} \right) \leq \frac{E(X)}{a} \)

theorem 1
∀ a > 0

\( \displaystyle P\left( {X \geq a} \right) \leq \frac{E(X)}{a} \)

Soit E l'évènement X≥a et soit Y la fonction caractéristique de E, alors Y est une variable aléatoire (nous l'avons vu comme exemple dans ) et E(Y)=P(E).
On a donc X ≥ aY.
C'est à dire encore X-aY ≥ 0.
D'où nous tirons E(X-aY) ≥ 0 d'après notre lemme.
Compte tenu des propriétés de l'espérance mathématique on a E(X-aY) = E(X)-aE(Y)
Donc E(X)-aE(Y) ≥ 0
Soit encore E(X)-aP(E) ≥ 0
ce qui nous donne l'inégalité cherchée.
L'inégalité de Markov possède un corollaire important :
Let E be the event X≥a and let Y be the characteristic function of E, then Y is a random variable (we saw it as an example in ) and E(Y)=P(E).
We therefore have X ≥ aY.
That is to say again X-aY ≥ 0.
From where we draw E(X-aY) ≥ 0 according to our lemma.
Taking into account the properties of mathematical expectation we have E(X-aY) = E(X)-aE(Y)
So E(X)-aE(Y) ≥ 0
Let again E(X)-aP(E) ≥ 0
which gives us the desired inequality.
Markov's inequality has an important corollary:
théorème 2
Soit Φ une fonction croissante, continue, positive ou nulle sur l'intervalle I.
Soit Y une v.a. réelle définie sur un espace probabilisé (Ω, \( \mathfrak{T}\),P) et telle que l'image de Y soit contenue dans I.
Dans ces conditions

\( \displaystyle P\left( {Y \geq b} \right) \leq \frac{E\left( \Phi(Y) \right)}{\Phi(b)} \)

theorem 2
Let Φ an increasing, continuous, positive or zero function on the interval I.
Let Y be a real r.v. defined on a probability space (Ω, \( \mathfrak{T}\),P) and such that the image of Y is contained in I.
In these conditions

\( \displaystyle P\left( {Y \geq b} \right) \leq \frac{E\left( \Phi(Y) \right)}{\Phi(b)} \)

On applique l'inégalité de Markov à X=Φ(Y) et a = Φ(b).
Nous obtenons ainsi que ∀ b > 0

\( \displaystyle P\left( \Phi(Y) \geq \Phi(b) \right) \leq \frac{E\left( \Phi(Y) \right)}{\Phi(b)} \)

La croissance de Φ entraîne que :
{Y≥b} ⊆ {Φ(Y) ≥ Φ(b)}
Donc que
P(Y ≥ b) ≤ P(Φ(Y)≥Φ(b)).
De là nous tirons l'inégalité de Bienaymé-Tchebycheff, dont l' énoncé suit :
We apply Markov's inequality to X=Φ(Y) and a = Φ(b).
We thus obtain that ∀ b > 0

\( \displaystyle P\left( \Phi(Y) \geq \Phi(b) \right) \leq \frac{E\left( \Phi(Y) \right)}{\Phi(b)} \)

The growth of Φ results in:
{Y≥b} ⊆ {Φ(Y) ≥ Φ(b)}
So that
P(Y ≥ b) ≤ P(Φ(Y)≥Φ(b)).
From there we derive the Chebychev inequality, the statement of which follows:
théorème 3
On suppose que X possède une espérance et une variance.
Dans ces conditions :
∀ α > 0

\( \displaystyle P\left( {\mid X - E(X) \mid \geq \alpha} \right) \leq \frac{v(X)}{\alpha^{2}} \)

theorem 3
We assume that X has an expectation and a variance.
Under these conditions:
∀ α > 0

\( \displaystyle P\left( {\mid X - E(X) \mid \geq \alpha} \right) \leq \frac{v(X)}{\alpha^{2}} \)

La démonstration s'obtient en utilisant le corollaire avec Y=|X-E(X)|, I=[0,+∞[ et Φ(x)=x².
The proof is obtained using the corollary with Y=|X-E(X)|, I=[0,+∞[ and Φ(x)=x².
