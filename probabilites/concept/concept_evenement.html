Cas des évènements élémentaires

Selon nos définitions antérieures un événement élémentaire ω est un élément de Ω. Mais du point de vue de la réalisation, la survenue de l'élément ω correspond avec notre nouvelle définition à l'évènement {ω}.
Bien que ces objets soient formellement différents, il est courant d'identifier ω avec {ω}, ce que nous ferons de façon systématique.

Case of elementary (atomic) events

According to our previous definitions an elementary event ω is an element of Ω. But from the point of view of the realization, the occurrence of the element ω corresponds with our new definition to the event {ω}.
Although these objects are formally different, it is common to identify ω with {ω}, what we will do systematically.

Implication

Examples

  • The event "the card is a spade" is incompatible with the event "the card is a heart", but is not incompatible with the event "the card is a king".
  • An event and its opposite are always incompatible.

Material conditional

définition 8
On dit qu'un évènement E1 'implique' ( 'entraîne' ou 'force' ) un évènement E2 si E1 ⊆ E2.
definition 8
An event E1 is said to 'force' ( or 'imply' ) an event E2 if E1 ⊆ E2.

Exemples

  • L'évènement "La carte est un coeur" implique "la carte est rouge".
  • L'évènement "La carte est un roi" implique "la carte est un honneur".

Systèmes complets

Examples

  • The event "The card is a heart" implies "the card is red".
  • The event "The card is a king" implies "the card is an honor".
