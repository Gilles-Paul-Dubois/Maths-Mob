Applications bilinéaires

Tous les espaces vectoriels sont supposés sur le même corps commutatif de base K. Nous supposons en outre que ce corps n'est pas de caractéristique 2, c'est à dire que 1+1 ≠ 0 (comme dans le cas du corps ℤ/2ℤ.

Soit f une application défine sur un produit d'espaces vectoriels E1×E2 et à valeur dans un espace vectoriel F.

définition
On dit que f est 'bilinéaire' si:
  • f(u1+u'1,u2)=f(u1,u2)+f(u'1,u2) ∀ (u1,u'1,u2) ∈ E1× E1×E2
  • f(λu1,u2)=λf(u1,u2) ∀ (λ,u1,u2) ∈ K×E1×E2
  • f(u1,u2+u'2)=f(u1,u2)+f(u1,u'2) ∀ (u1,u2,u'2) ∈ E1×E2×E2
  • f(u1,λu2)=λf(u1,u2) ∀ (λ,u1,u2) ∈ K×E1×E2
Il résulte immédiatement de la définition que:
  • La somme de deux applications bilinéaires est encore bilinéaire.
  • Le produit d'une application bilinéaire par un scalaire est encore bilinéaire.
  • L'ensemble des applications bilinéaires de E1×E2 dans F est muni d'une structure d'espace vectoriel.

Exemples d'applications bilinéaires

  1. Chacune des n2 applications de Kn×Kn → K ((x1,...,xn),(y1,...,yn)) → xkyh est bilinéaire.
  2. Toute des précédentes est bilinéaire, en particulier ((x1,...,xn),(y1,...,yn)) → x1y1+...+xnyn, le produit scalaire usuel dans le cas de ℝn
  3. Si E est l'espace de toutes les suites de nombres réels. L'application ((xn)n∈ℕ,(yn)n∈ℕ) → (xnyn)n∈ℕ est bilinéaire.
  4. Si f et g désignent des applications indéfiniment dérivables d'un intervalle I ⊆ ℝ, l'application (f,g) → (fg)' est bilinéaire.

Formes multilinéaires

définition
Tout comme dans le cas des applications linéaires, si le but est le corps K lui-même on dit 'formes multilinéaires' au lieu 'd'applications multilinéaires'. Il s'agit donc d'une précision supplémentaire.

Exemples

  1. Les exemples 1 et 2 donnés précédemment correspondent à des formes multilinéaires.
  2. Si E est l'espace des applications d'un intervalle I dans ℝ et si t est un point de I l'application (f,g) → f(t)g(t) est une forme bilinéaire sur E.
  3. Le produit de deux formes linéaires est une forme bilinéaire. Plus généralement, le produit de p formes linéaires est une forme p-linéaire. Cet exemple est important.

Applications alternées

Soient E1,...,En n ensembles quelconques (pas forcément des espaces vectoriels) et F un groupe additif.
définition
Une application f: E1×...×En → F est dite 'alternée' ssi:
f(x1,...,xn) = 0 chaque fois que deux au moins des xi sont égaux.
définition
Cela s'applique naturellement aux applications multilinéaires. On définit donc tout naturellement les 'applications multilinéaires alternées' et les 'formes multilinéaires alternées' .

Exemples:

  1. Le produit vectoriel classique de l'espace des vecteurs libres de ℝ3 est bilinéaire alterné.
  2. L'application de ℝ2×ℝ2 dans ℝ donnée par:
    ((a,b),(c,d)) → ad-bc est une forme bilinéaire alternée.
  3. L'application (u,v,w) → (u|v|w) (produit mixte= u.(v ∧ w)) est une forme trilinéaire alternée sur ℝ3

Bilinear maps

All vector spaces are assumed over the same commutative field K. We further assume that this field is not of characteristic 2, i.e. 1+1 ≠ 0 (as in the case of the field ℤ/2ℤ.

Let f be a map with domain a product of vector spaces E1×E2 and codomain a vector space F.

definition
We say that f is 'bilinear' if:
  • f(u1+u'1,u2)=f(u1,u2)+f(u'1,u2) ∀ (u1,u'1,u2) ∈ E1× E1× And2
  • f(λu1,u2)=λf(u1,u2) ∀ (λ,u1,u2) ∈ K×E1×E2
  • f(u1,u2+u'2)=f(u1,u2)+f(u1,u'2) ∀ (u1,u2,u'2) ∈ E1×E2× And2
  • f(u1,λu2)=λf(u1,u2) ∀ (λ,u1,u2) ∈ K×E1×E2
It follows immediately from the definition that:
  • The sum of two bilinear maps is still bilinear.
  • The product of a bilinear map by a scalar is still bilinear.
  • The set of bilinear maps of E1×E2 in F is endowed with a vector space structure.

Examples of bilinear maps

  1. Each of the n2 applications of Kn×Kn → K ((x1,...,xn),(y1,...,yn)) → xkyh is bilinear.
  2. All of the previous ones is bilinear, in particular ((x1,...,xn),(y1,...,yn)) → x1y1+...+xnyn, the usual scalar product in the case of ℝn
  3. If E is the space of all sequences of real numbers. The map ((xn)n∈ℕ,(yn)n∈ℕ) → (xnyn)n∈ℕis bilinear.
  4. If f and g denote indefinitely differentiable maps of an interval I ⊆ ℝ, the map (f,g) → (fg)' is bilinear.

Multilinear forms

definition
Just as in the case of linear maps, if the codomain is the field K itself we say 'multilinear forms' instead of 'multilinear maps'. This is therefore an additional clarification.

Examples

  1. Examples 1 and 2 given previously correspond to multilinear forms.
  2. If E is the space of maps of an interval I in ℝ and if t is a point of I the map (f,g) → f(t)g(t) is a bilinear form on E.
  3. the product of two linear forms is a bilinear form. More usually the product of p linear forms is a p-linear form. This example is important.

Alternating applications

Let E1,...,En n arbitrary sets (not necessarily vector spaces) and F an additive group.
definition
A map f: E1×...×En → F is said 'alternating' iff:
f(x1,...,xn) = 0 whenever at least two of the xi's are equal.
definition
This naturally applies to multilinear applications. We therefore naturally define the 'alternating multilinear maps' and the 'alternating multilinear forms' .

Examples:

  1. The classical vector product of the free vector space of ℝ3 is alternating bilinear.
  2. The application from ℝ2×ℝ2 to ℝ given by:
    ((a,b),(c,d)) → ad-bc is an alternating bilinear form.
  3. The map (u,v,w) → (u|v|w) (mixed product= u.(v ∧ w)) is an alternating trilinear form on ℝ3
