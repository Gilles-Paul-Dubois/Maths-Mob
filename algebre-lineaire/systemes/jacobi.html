Matrices à diagonale dominante

The idea is to use the remark given in the previous page devoted to , by asking:
M=E+F with:
  • E is a strict lower triangular matrix extracted from A (zero diagonal elements).
  • F is a strict upper triangular matrix extracted from A.
D being the diagonal of A. We will now give sufficient conditions for the Picard Banach's theorem applies. We must therefore ensure that ||D-1M|| < 1

Diagonally Dominant Matrices

définition 1
Une matrice A = (ai,j)1≤i,j≤n est dite à 'diagonale dominante' si elle vérifie :

\( \left| \alpha_{i,i} \right|> \sum_{j\neq i}^{}\left| \alpha _{i,j}\right| \text { }\forall i \text { }1\leqslant i\leqslant n \)

definition 1
A matrix A = (a i,j ) 1≤i,j≤n is said to be ' diagonally dominant ' if it verifies:

\( \left| \alpha_{i,i} \right|> \sum_{j\neq i}^{}\left| \alpha _{i,j}\right| \text { }\forall i \text { }1\leqslant i\leqslant n \)

Cette condition peut sembler très forte, mais en fait dans les gros systèmes, les matrices sont souvent des . Lorsque ces matrices sont issues d'équations aux dérivées partielles le coefficient diagonal est souvent dominant.
Nous allons maintenant supposer que A est à diagonale dominante, et chercher la forme de D-1M. Remarquons tout de suite que dans notre cas (système de Cramer) il est impossible qu'un seul élément diagonal soit nul sinon nous aurions une ligne et une colonne nulle contrairement à l'hypothèse qu'on a affaire à un système de Cramer.
La matrice D est donc sans aucun 0. Son inverse est donc la matrice diagonale dont les éléments sont les inverses des élements de D. Calculons maintenant les coefficients de D-1M soient les ci,j on a :
ci,i=0 ∀ i 1≤i≤n
ci,j= ai,j/ai,i i ≠ j
d'où |ci,j|<1 ∀(i,j) 1≤i,j≤n i ≠ j

On a évidemment ||D-1M|| < 1
