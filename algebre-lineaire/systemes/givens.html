Givens algorithm

We start by doing the :
G(2,1,A)A=A(2,1) which will have a 0 in position 2,1
then G(3,1,A(2,1))A(2,1) which will have zeros in positions 2,1 and 3,1
and so on until:
G(n,1,A(n-1,1))A(n-1,1)=A(n,1) whose first column will be null except the first element. Then we continue with
G(3,2,A(n,1))=A(2,2)
...........
G(n,2,A(n-1,2))A(n-1,2)=A(n,2) which will have zeros below the diagonal for the first two columns.
and so on, the form of the matrices G(p,q,X) and the order of the multiplications ensuring that acquired zeros are not lost.
We come to the end at
G(n,n-1,A(n,n-2))=A(n,n-1) which must be .

\( \prod_{p=n-1}^{1}\prod_{q=n}^{p-1}G\left ( q,p,A(q,p) \right ) \)
