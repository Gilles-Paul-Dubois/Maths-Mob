  1. L'application f(x,y,z)=xz de ℝ3 dans ℝ est-elle linéaire?
  2. L'application x → x2 de ℤ/2ℤ dans lui-même est-elle linéaire?

aide

Revoir la définition d'une application linéaire

solution

  1. f((x,y,z)+(x',y',z'))= f(x+x',y+y',z+z')= (x+x')(z+z')=xx'+xz+x'z+zz'.
    f(x,y,z)+f(x',y',z')=xz+x'z'
    De sorte que f((x,y,z)+(x',y',z')) = f(x,y,z)+f(x',y',z')
    équivaut à:
    xx'+xz+x'z+zz'=xz+x'z'
    soit encore à:
    xx'+zz'=0
    Ce qui est faux en général.
    L'application n'est donc pas linéaire.
  2. Calculons (x+y)2 dans ℤ/2ℤ
    On a (x+y)2=x2+y2+2xy
    Or 2xy=0 puisqu'on est dans un corps de caractéristique 2.
    L'application est donc un isomorphisme de groupes.
    Calculons maintenant (λx)2 = λ2x2.
    Le problème de la linéarité de cette application revient donc à savoir si λ2 est toujours égal à λ dans ℤ/2ℤ
    De fait cette égalité est vraie pour λ =0 et λ = 1 or il n'y a que ces deux éléments.
    L'application est donc bien linéaire.
  1. Is the mapping f(x,y,z)=xz from ℝ3 to ℝ linear ?
  2. Is the mapping x → x2 from ℤ/2ℤ to itself, linear ?

hint

Review the definition of linear maps.

solution

  1. f((x,y,z)+(x',y',z'))= f(x+x',y+y',z+z')= (x+x')(z+z')=xx'+xz+x'z+zz'.
    f(x,y,z)+f(x',y',z')=xz+x'z'
    So that f((x,y,z)+(x',y',z')) = f(x,y,z)+f(x',y',z')
    is equivalent to :
    xx'+xz+x'z+zz'=xz+x'z'
    or again to :
    xx'+zz'=0
    Which is false in general.
    As a conclusion, the map is not linear.
  2. Let us compute (x+y)2 in ℤ/2ℤ
    We have (x+y)2=x2+y2+2xy
    But 2xy=0 because we are in a field with characteristic 2.
    So the map is a group homomorphism.
    Let us compute now (λx)2 = λ2x2.
    The problem of linearity just boils up to decide if λ2 is always equal to λ in ℤ/2ℤ
    Actually this equality is true for λ =0 and λ = 1 but there are no other element.
    So we conclude that the mapping is linear.
