Montrer que si K =ℤ/pℤ, et si E et F sont deux K-espaces vectoriels, pour que f soit linéaire il suffit que f soit un homomorphisme de groupes de (E,+) dans (F,+).
Montrer que c'est encore vrai si K=ℚ.

aide

Si n est entier f(nx)= f(x+ ... +x)

solution

Soit donc n entier on a :
f(nx)=f(x+ ... +x)=f(x)+ ... +f(x)=nf(x)
Supposons maintenant que K = ℚ
f(x)=f(q(x/q))=qf(x/q)
D'où f(x/q)=(1/q)f(x)
Et finalement f((p/q)x)=p(f(x/q)=(p/q)f(x)

Show that if K =ℤ/pℤ, and if E and F are two K-vector spaces, for f to be linear it suffices that f is a homomorphism of groups from (E,+) into (F,+).
Show that it is still true if K=ℚ.

hint

If n is an integer f(nx)= f(x+ ... +x)

solution

Let n be an integer, we have:
f(nx)=f(x+ ... +x)=f(x)+ ... +f(x)=nf(x)
Now suppose that K = ℚ
f(x)=f(q(x/q))=qf(x/q)
Hence f(x/q)=(1/q)f(x)
And finally f((p/q)x)=p(f(x/q)=(p/q)f(x)
