Soient f1, f2, ...,fn n formes linéaires sur Kn.
Montrer que:
∃ x ≠ 0 | f1(x)=f2(x)=...=fn(x)=0 ⇒ (f1,...,fn) liée.

aide

On se souviendra que E s'identifie à E** et que tout x ∈ E peut être identifié à un élément du bidual.

solution

Si (f1,...,fn) n'est pas liée alors c'est une base de (Kn)*.
Mais dans ce cas il existerait une forme linéaire x non nulle sur (Kn)*, nulle sur tous les éléments d'une base, c'est impossible.

Let f1, f2, ...,fn n linear forms on Kn .
Show that:
∃x ≠ 0 | f1(x)=f2(x)=...=fn(x)=0 ⇒ (f1,...,fn) linearly dependent.

hint

Recall that E identifies with E** and that every x ∈ E can be identified with an element of the double dual.

solution

If (f1,...,fn) is not linearly dependent then it is a basis of (Kn)*.
But in this case there would exist a non-null linear form x on (Kn)*, null on all the elements of a basis, it is impossible.
