Transposition

theorem
Let E and F be two vector spaces, E*and F* dual spaces. Let f : E → F be a linear map. Then for any linear form φ of F* the application φof is an element of E*. The map φ → φof is denotedtf and is called the map 'transposed' of f, it is a linear map. Moreover if the spaces are of finite dimension the map f →tf from L(E,F) to L(F*,E*) is an isomorphism of vector spaces.

proof

Both spaces are of dimension m×n so that to show that f →tf is an isomorphism it suffices to show that it is injective, so that its kernel is zero. It all boils down to showing that if tf=0 then f=0. So if φof=0 ∀ φ ∈ F* then f is zero. To see this, it suffices to take for φ all the coordinate forms with respect to a basis of F.
