Montrer que dans le cas euclidien réel, comme dans le cas hilbertien complexe, si < , > désigne le produit scalaire, on a:
|<x,y>| ≤ ||x||×||y||

aide

Supposer d'abord que <x,y> soit réel.
Etudier le polynôme P(t)=< x+ty,x+ty>

solution

Commençons par le cas complexe.
Supposons d'abord que <x,y> soit réel.
On a ∀ t ∈ ℝ
P(t)=< x+ty,x+ty>=<x,x> +2t<x,y>+t2 <y,y> ≥ 0
C'est donc que le discriminant réduit de ce binôme du second degré est négatif.
<x,y>2-<y,y><x,x> ≤ 0
Ce qui nous donne:
<x,y>2 ≤ <y,y><x,x> et finalement, par passage aux racines carrées:
|<x,y>| ≤ ||x||×||y||
Supposons maintenant que <x,y> ne soit pas réel.
Soit θ l'argument de ce nombre complexe. Posons x'=xe-iθ alors l'inégalité est vraie pour x' et y.
Mais comme |<x,y>|=|<x',y>|
et
||x||=||x'||
L'inégalité est également vraie pour x et y.
La preuve est identique dans le cas réel.

Show that in the real Euclidean case and in the complex Hilbertian as well, if < , > denotes the dot (scalar) product, we have:
|<x,y>| ≤ ||x||×||y||

hint

Suppose first that <x,y> is real.
Then study the polynomial P(t)=< x+ty,x+ty>

solution

Let's begin with the complex case.
Suppose first that <x,y> is real.
We have ∀ t ∈ ℝ
P(t)=< x+ty,x+ty>=<x,x> +2t<x,y>+t2 <y,y> ≥ 0
It means that the discriminant of this polynomial is negative.
<x,y>2-<y,y><x,x> ≤ 0
Which gives us :
<x,y>2 ≤ <y,y><x,x> and finally passing to the square roots :
|<x,y>| ≤ ||x||×||y||
Suppose now that <x,y> is not real.
Let θ the angle of this complex number. Let x'=xe-iθ Then the inequality is true for x' and y.
But since |<x,y>|=|<x',y>|
and
||x||=||x'||
The inequality is also true for x and y.
The proof is similar in the real case..
