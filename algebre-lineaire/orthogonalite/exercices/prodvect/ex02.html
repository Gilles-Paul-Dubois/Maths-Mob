E est un espace euclidien orienté de dimension 3.
Le produit scalaire est noté x.y, le produit mixte (x|y|z), le produit vectoriel x ∧ y.
Résoudre l'équation a∧x=b

aide

Commencer par étudier le cas a=0.
Si a≠0 trouver d'abord une solution particulière, puis résoudre a∧x=0

solution

Supposons d'abord a=0.
alors si b≠0 pas de solution.
si b=0 tout x est solution.
Considérons maintenant le cas où a ≠0.
remarquons que par la formule du double produit vectoriel:
a∧(b∧a)=(a.a)b-(a.b)b = (a.a)b puisque a.b=0
Donc une solution particulière est x0 =(1/||a||2)b∧a
D'une façon générale la solution est x=x0+λa.

E is an oriented Euclidean space of dimension 3.
The dot product is denoted x.y, the mixed product (x|y|z), the cross product x ∧ y.
Solve the equation a∧x=b

hint

Start by studying the case a=0.
If a≠0 first find a particular solution, then solve a∧x=0

solution

First suppose a=0.
then if b≠0 no solution.
if b=0 every x is a solution.
Now consider the case where a≠0.
note that by the formula of the double vector product:
a∧(b∧a)=(a.a)b-(a.b)b = (a.a)b since a.b=0
So a particular solution is x0 =(1/||a||2)b∧a
Generally the solution is x=x0+λa.
