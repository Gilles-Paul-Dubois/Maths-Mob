Définition

définition
Une matrice carrée est dite ' diagonale ' si tous ses coefficients sont nuls, à l'exception éventuelle des coefficients de la diagonale. Soit encore:
A=(αi,j) diagonale si et seulement si αi,j=0 si i ≠ j.
La matrice nulle et la matrice unité sont des exemples de matrices 'diagonales'.

Definition

definition
A square matrix is ​​said to be ' diagonal ' if all its coefficients are zero, with the possible exception of the coefficients of the diagonal. Or again:
A=(αi,j ) diagonal if and only if αi,j =0 if i ≠ j.
The null matrix and the unit matrix are examples of 'diagonal' matrices.
