On considère dans l'espace Kn les vecteurs:
u1=(1,0, ...,0)
u2=(1,1,0, ..., 0)
u3=(1,1,1,0, ...,0)
.................
un=(1,1, .... 1,1)
Montrer que le système S=(u1,u2,...,un) est un système générateur de Kn.

aide

Si par combinaison linéaire des vecteurs de A on peut obtenir tous les vecteurs d'un système B générateur alors A est lui-même générateur.

solution

Soient:
e1=(1,0, ..,0)
e2=(0,1, ..., 0)
e3=(0,0,1,0, ... ,0)
.........
en=(0,0, ... ,0,1)
Alors (e1,e2,...,en) est générateur.
mais on a :
e1=u1
e2=u2-u1
e3=u3-u2
.............
en=un-un-1
Donc (u1,....,un) est générateur.

We consider in the space Kn the vectors:
u1= (1,0, ..., 0)
u2=(1,1,0, ..., 0)
u3= (1,1,1,0, ..., 0)
.................
un=(1,1, .... 1,1)
Show that the system S=(u1,u2,...,un) is a spanning system of Kn.

hint

If by linear combination of the vectors of A we can obtain all the vectors of a spanning system B then A is itself spanning.

solution

Let:
e1= (1,0, .., 0)
e2=(0,1, ..., 0)
e3= (0,0,1,0, ..., 0)
.........
in)=(0,0, ... ,0,1
Then (e1,e2,...,en) is spanning.
But we have :
e1= u1
e2= u2-u1
e3= u3-u2
.............
en=un-un-1
So (u1,....,un) is spanning.
