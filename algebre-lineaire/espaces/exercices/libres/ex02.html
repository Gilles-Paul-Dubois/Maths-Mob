Soit E l'espace de toutes les fonctions de ℝ dans ℝ
Pour tout entier n ≥ 0, on désigne par fn la fonction x → enx.
Montrer que pour tout entier p, le système (f0,f1,...,fp) est libre.

aide

S=(a1 , .... , ap ) libre équivaut à:
Aucun des ai n'est combinaison linéaire des aj pour j < i .

solution

Supposons que fn soit combinaison linéaire des fm pour m < n.
On aurait alors des scalaires k0,...,kn-1 tels que
fn= k0f0+...+kn-1fn-1
C'est à dire
enx= k0+ k1ex+...+kn-1e(n-1)x ∀ x ∈ ℝ
En divisant par enx
1= k0e-nx+k1e(1-n)x+....+kn-1e-x ∀ x ∈ ℝ
Mais le membre de droite est une fonction qui tends vers 0 quand x tends vers + ∞ .

Let E be the space of all the functions from ℝ into ℝ
For any integer n ≥ 0, we denote by fn the function x → enx.
Show that for any integer p, the system (f0,f1,...,fp) is linearly independent.

hint

S=(a1 , .... , ap ) linearly independent is equivalent to:
None of the ai n is a linear combination of aj for j < i.

solution

Suppose fn or linear combination of the fm for m < n.
We would then have scalars k0,...,kn-1 such as
fn= k0f0+...+kn-1fn-1
That's to say
enx= k0+ k1ex+...+kn-1e(n-1)x ∀x∈ ℝ
By dividing by enx
1= k0e-nx+k1e(1-n)x+....+kn-1e-x ∀ x ∈ ℝ
But the right-hand side is a function that tends to 0 as x tends to +∞.
