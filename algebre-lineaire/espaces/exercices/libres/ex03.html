On tout entier naturel p on désigne par Up la suite dont le terme général est up,n=np
Ainsi
U0=(1,1,1, ...,1, .......)
U1=(1,2,3, ...,n, .......)
U2=(1,4,9, ...,n², .......)
................
Up=(1,2p,3p, ...,np, .......)
Montrer que pour tout p (U1,...,Up) est un système libre.

aide

S=(a1 , .... , ap ) libre équivaut à:
Aucun des ai n'est combinaison linéaire des aj pour j < i .

solution

Supposons que Up soit combinaison linéaire de (U0,U1,...,Up-1)
Il existerait alors des scalaires k0, .., kp-1 tels que :
np=k0+k1n+....+kp-1np-1
D'où 1= k0n-p+k1n-p+1+...+kp-1n-1 ∀ n ∈ ℕ
Or le membre de droite tend vers 0 quand n → ∞ .

For any natural integer p we denote by Up the sequence whose general term is up,n=np
Thereby
U0=(1,1,1, ...,1, .......)
U1= (1,2,3, ..., n, .......)
U2=(1,4,9,...,n²,.......)
................
Up=(1,2p,3p, ...,np, .......)
Show that for all p (U1,...,Up) is a linearly independent system.

hint

S=(a1 , .... , ap ) linearly independent is equivalent to:
None of the ai n is a linear combination of aj for j < i.

solution

Suppose that Up or linear combination of (U0,U1,...,Up-1)
There would then exist scalars k0, .., kp-1 such as :
np=k0+k1n+....+kp-1np-1
Hence 1= k0n-p+k1n-p+1+...+kp-1n-1 ∀ n ∈ ℕ
But the right-hand side tends to 0 when n → ∞ .
