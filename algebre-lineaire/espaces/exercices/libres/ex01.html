On considère dans l'espace Kn les vecteurs:
u1=(1,0, ...,0)
u2=(1,1,0, ..., 0)
u3=(1,1,1,0, ...,0)
.................
Un=(1,1, .... 1,1)
Montrer que le système S=(u1,u2,...,un) est un système libre de Kn.

aide

S=(a1,....,ap) libre équivaut à:
Aucun des ai n'est combinaison linéaire des aj pour j < i

solution

u1=(1,0, ...,0)
u2=(1,1,0, ..., 0)
(u1,u2) est libre parce que u2 n'est pas multiple de u1.
u1=(1,0, ...,0)
u2=(1,1,0, ..., 0)
u3=(1,1,1,0 ..., 0)
(u1,u2,u3) est libre car u3 ne peut pas être combinaison linéaire de (u1,u2), etc...

We consider in the space Kn the vectors:
u1= (1,0, ..., 0)
u2=(1,1,0, ..., 0)
u3= (1,1,1,0, ..., 0)
.................
Un=(1,1, .... 1,1)
Show that the system S=(u1,u2,...,un) is a linearly independent system of Kn.

hint

S=(a1,....,ap) linearly independent is equivalent to:
None of the ai n is a linear combination of aj for j < i

solution

u1= (1,0, ..., 0)
u2=(1,1,0, ..., 0)
(u1,u2) is linearly independent because u2 is not a multiple of u1.
u1= (1,0, ..., 0)
u2=(1,1,0, ..., 0)
u3= (1,1,1,0 ..., 0)
(u1,u2,u3) is linearly independent because u3 cannot be a linear combination of (u1,u2), etc...
