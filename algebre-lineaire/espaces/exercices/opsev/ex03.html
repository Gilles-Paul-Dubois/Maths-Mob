Soient E un espace vectoriel et A,B,C,D quatre sous-espaces de E vérifiant:
E=A⊕B
E=C⊕D
A ⊆ C
B ⊆ D
Montrer que dans ces conditions:
A=C et B=D

aide

La donnée dit que A et B sont supplémentaires d'une part et que C et D le sont aussi.

solution

Montrons que C ⊆ A.
Soit x ∈ C, comme E= A⊕B on peut écrire c=a+b avec a ∈ A et b ∈ B.
On a donc b=c-a ∈ C puisque a ∈ C
Mais comme b ∈ B on a aussi b ∈ D
donc b ∈ C ∩ D = {0}
d'où b=0 et c=a
Ce qui prouve que c ∈ A.
On montre de la même façon que D ⊆ B

Let E be a vector space and A,B,C,D four subspaces of E verifying:
E=A⊕B
E=C⊕D
A ⊆ C
B ⊆ D
Show that under these conditions:
A=C and B=D

hint

The data says that A and B are supplementary on the one hand and that we can say the same about C and D.

solution

Let us show that C ⊆ A.
Let x ∈ C, since E= A⊕B we can write c=a+b with a ∈ A and b ∈ B.
We therefore have b=ca ∈ C since a ∈ C
But as b ∈ B we also have b ∈ D
so b ∈ C ∩ D = {0}
hence b=0 and c=a
Which proves that c ∈ A.
We show in the same way that D ⊆ B
