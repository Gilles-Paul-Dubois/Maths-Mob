On considère dans l'espace ℝ4 , le système formé par les 5 vecteurs:
u1=(-1,0,0,-1)
u2=(0,1,1,0)
u3=(1,1,1,1)
u4=(1,2,3,4)
u5=(0,1,2,3)
Calculer la dimension du sous-espace qu'ils engendrent.

aide

u5=u4-u3
u3=u2-u1
Le sous-espace engendré par (u1,u2,u3,u4,u5) est donc le même que celui engendré par (u1,u2,u4)

solution

u5=u4-u3
u3=u2-u1
Le sous-espace engendré par (u1,u2,u3,u4,u5) est donc le même que celui engendré par (u1,u2,u4)
Montrons que (u1,u2,u4) est un système libre.
Considérons une relation linéaire entre ces 3 vecteurs:
α(1,0,0,-1)+β(2,1,1,0)+γ(1,2,3,4)=(0,0,0,0)
Cela conduit au système:
α+2β+γ=0
β+2γ=0
β+3γ=0
qui possède comme unique solution α=β=γ=0
Donc (u1,u2,u4) est une base du sous-espace qu'il engendre et la réponse à la question posée est 3.

We consider in space ℝ4, the system formed by the 5 vectors:
in1= (- 1,0,0, -1)
u2=(0,1,1,0)
u3= (1,1,1,1)
u4= (1,2,3,4)
u5= (0,1,2,3)
Calculate the dimension of the subspace they span.

hint

u5= u4-u3
u3=u2-u1
The subspace spanned by (u1,u2,u3,u4,u5) is therefore the same as that spanned by (u1,u2,u4)

solution

u5= u4-u3
u3=u2-u1
The subspace spanned by (u1,u2,u3,u4,u5) is therefore the same as that spanned by (u1,u2,u4)
Let us show that (u1,u2,u4) is a linearly independent system.
Consider a linear relation between these 3 vectors:
α (1,0,0, -1) + β (2,1,1,0) + γ (1,2,3,4) = (0,0,0,0)
This leads to the system:
α + 2β + γ = 0
β + 2γ = 0
β + 3γ = 0
which has as unique solution α=β=γ=0
So (u1,u2,u4) is a basis of the subspace it spans and the answer to the question asked is 3.
