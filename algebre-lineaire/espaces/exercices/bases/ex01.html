  1. Montrer que dans l'espace ℝ3
    Le système ((1,1,1),(1,1,2)) n'est pas une base.
  2. Pour quelles valeurs du paramètre a le système
    ((1,-1,0),(2,-1,2),(1,0,a)) est-il une base de ℝ3 ?

aide

  1. Une base doit être un système générateur
  2. Une base doit être un système libre

solution

  1. Toute combinaison linéaire des deux vecteurs a ses deux premières coordonnées égales. Le vecteur (0,1,0) ne peut donc être combinaison linéaire de ces deux là.
  2. Cherchons pour quelle valeur de a le vecteur (1,0,a) est combinaison linéaire de (1,-1,0) et (2,-1,2).
    En posant et résolvant le système (1,0,a)=α(1,-1,0)+β(2,-1,2), on trouve qu'on doit avoir α=-1 et β = 1.
    3. Ce qui nous donne comme condition nécessaire a=2.
    Le système ne pourra donc être libre que si a ≠ 2.
    Pour montrer que c'est une base il faut montrer qu'il est alors générateur.
    Soit (x,y,z) un vecteur quelconque de ℝ3
    Dire que ce vecteur est combinaison linéaire de ((1,-1,0),(2,-1,2),(1,0,a)), équivaut à l'existence de scalaires α, β, δ, tels que:
    (x,y,z)=(α,-α,0)+(2β,-β,2β)+(δ,0,aδ).
    Cela s'écrit comme un système:
    α+2β+δ=x
    -α-β=y
    2β+aδ=z
    On voit que ce système possède la solution unique:
    δ=(z-2x-2y)/(a-2)
    β=x+y-δ
    α=x-2β-δ
    d'où notre conclusion:
    ((1,-1,0),(2,-1,2),(1,0,a)) est une base si et seulement si a ≠ 2.
  1. Show that in space ℝ3
    The system ((1,1,1),(1,1,2)) is not a basis.
  2. For which values ​​of the parameter has the system
    ((1,-1,0),(2,-1,2),(1,0,a)) is a basis of ℝ3 ?

hint

  1. A basis must be a spanning system
  2. A basis must be a linearly independent system

solution

  1. Any linear combination of the two vectors has its first two coordinates equal. The vector (0,1,0) cannot so be linear combination of these two .
  2. Find for what value of a the vector (1,0,a) is a linear combination of (1,-1,0) and (2,-1,2).
    By setting and solving the system (1,0,a)=α(1,-1,0)+β(2,-1,2), we find that we must have α=-1 and β = 1.
    3. Which gives us as necessary condition a=2.
    The system can therefore only be linearly independent if a ≠ 2.
    To show that it is a basis it is necessary to show that it is then spanning.
    Let (x,y,z) be any vector of ℝ3
    Say that this vector is a linear combination of ((1,-1,0),(2,-1,2),(1,0,a)), is equivalent to the existence of scalars α, β, δ, such that:
    (x, y, z) = (α, -α, 0) + (2β, -β, 2β) + (δ, 0, aδ).
    This is written as a system:
    α + 2β + δ = x
    -α-β = y
    2β + aδ = z
    We see that this system has the unique solution:
    δ=(z-2x-2y)/(a-2)
    β = x + y-δ
    α = x-2β-δ
    hence our conclusion:
    ((1,-1,0),(2,-1,2),(1,0,a)) is a basis if and only if a ≠ 2.
