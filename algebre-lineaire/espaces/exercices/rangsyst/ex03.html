On considère le système de 3 vecteurs:
S=(u,v,w) où u=(1,1,0) v=(0,2,1) w=(1,0,1)

  1. Quel est le rang de S lorsqu'on considère les 3 vecteurs comme éléments de ℝ3 ?
  2. Même question quand on les considère comme éléments de l'espace (ℤ/3ℤ)3 ?

aide

Dans ℤ/3ℤ on a 3=0.

solution

On se place tout d'abord dans l'espace ℝ3.
Les deux premiers vecteurs sont indépendants car tout multiple du premier a sa troisième coordonnée nulle.
L'équation w=αu+βv équivaut au système:
α=1
2β+α=0
β=1
Ce système n'a aucune solution dans ℝ, les 3 vecteurs sont indépendants, et le rang est 3.
Si on se place maintenant dans (ℤ/3ℤ)3.
Le système (u,v) est de rang 2 pour les mêmes raisons que précédemment.
Par contre, on a w=u+v, donc le système (u,v,w) est de rang 2.

We consider the system of 3 vectors:
S=(u,v,w) where u=(1,1,0) v=(0,2,1) w=(1,0,1)

  1. What is the rank of S when we consider the 3 vectors as elements of ℝ3 ?
  2. Same question when we consider them as elements of space (ℤ/3ℤ)3 ?

hint

In ℤ/3ℤ we have 3=0.

solution

We first place ourselves in space ℝ3.
The first two vectors are independent because any multiple of the first has its third coordinate zero.
The equation w=αu+βv is equivalent to the system:
α = 1
2β + α = 0
β = 1
This system has no solution in ℝ, the 3 vectors are independent, and the rank is 3.
If we now place ourselves in (ℤ/3ℤ)3.
The system (u,v) has rank 2 for the same reasons as before.
But this time, we have w=u+v, so the system (u,v,w) has rank 2.
