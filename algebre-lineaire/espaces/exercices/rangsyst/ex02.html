Calculer le rang du système suivant dans ℝ5 :
(u,v,w,z,t) où u=(1,0,0,1,4) v=(0,1,0,2,5) w=(0,0,1,3,6) z=(1,2,3,14,32) t=(4,5,6,32,77)
Calculer le rang du système suivant dans ℝ4 :
(x,y,z,t,u,v) où x=(2,1,1,1) y=(1,3,1,1) z=(1,1,4,1) t=(1,1,1,5) u=(1,2,3,4) v=(1,1,1,1)

aide

Pour le calcul du rang on peut éliminer les vecteurs qui sont combinaisons linéaires d'autres vecteurs du système.

solution

Première question:
les 3 premiers vecteurs sont clairement indépendants.
On constate que z=u+2v+3w et t=4u+5v+6w.
Le rang est donc 3.
Seconde question:
y ne peut être multiple de x car il aurait ses trois dernières coordonnées égales.
z ne peut être combinaison linéaire de x et y car il aurait ses deux dernières coordonnées égales.
(x,y,z) est donc libre et le rang est ≥ 3.
Mais le rang est au plus égal à 4.
Donc le rang est 3 ou 4.
Supposons que le vecteur t soit combinaison linéaire des vecteurs x,y,z avec des coefficient a,b,c.
Posons S=a+b+c
On a alors S=5 d'une part, mais aussi 2S+9=3 Ceci est impossible. C'est donc que t n'est pas combinaison linéaire de x,y et z.
Il en résulte que le rang est 4.

Calculate the rank of the next system in ℝ5 :
(u,v,w,z,t) where u=(1,0,0,1,4) v=(0,1,0,2,5) w=(0,0,1,3,6) z=(1,2,3,14,32) t=(4,5,6,32,77)
Calculate the rank of the next system in ℝ4 :
(x,y,z,t,u,v) where x=(2,1,1,1) y=(1,3,1,1) z=(1,1,4,1) t=( 1,1,1,5) u=(1,2,3,4) v=(1,1,1,1)

hint

For the calculation of the rank of a system one can eliminate the vectors which are linear combinations of other vectors of the system.

solution

First question:
the first 3 vectors are clearly independent.
We note that z=u+2v+3w and t=4u+5v+6w.
The rank is therefore 3.
Second question:
y cannot be a multiple of x because it would have its last three coordinates equal.
z cannot be a linear combination of x and y because it would have its last two coordinates equal.
(x,y,z) is therefore linearly independent and the rank is ≥ 3.
But the rank is at most equal to 4.
So the rank is 3 or 4.
Suppose vector t is a linear combination of vectors x,y,z with coefficients a,b,c.
Let S=a+b+c
We then have S=5 on the one hand, but also 2S+9=3 This is impossible. It is therefore that t is not a linear combination of x,y and z.
It follows that the rank is 4.
