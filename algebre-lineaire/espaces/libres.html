Relations linéaires

définition
Soit S=(u1,u2,....,un) un système de vecteurs de E, sur K. On appelle 'relation linéaire' entre les vecteurs ui un système de scalaires (λ1,λ2,...,λn) non tous nuls vérifiant:
λ1u1+λ2u2+...+λnun=0

Non tous nuls ne signifie pas tous non nuls !

On vérifie imméditemment le résultat suivant:
théorème
Il existe une relation linéaire pour le système S=(u1,u2,....,un) équivaut à:
Un au moins des vecteurs ui est combinaison linéaire des autres vecteurs du système.
Ainsi (-1,-2,-1,1) est une relation linéaire pour le système de polynômes (1,x,x2,(1+x)2)

Systèmes liés

définition
On dit qu'un système S=(u1,u2,....,un) de vecteurs de E est 'lié' s'il existe une relation linéaire pour ce système.
Voici des exemples de systèmes liés:
  1. (u,2u)
  2. (u,v,u+v)
  3. (u,v,w,w-2u+v)
  4. Tout système comportant le vecteur nul

Systèmes libres

définition
On dit qu'un système S=(u1,u2,....,un) de vecteurs de E est 'libre' s'il n'est pas lié, c'est à dire s'il n'existe aucune relation linéaire pour ce système, soit encore si aucun des vecteurs du système n'est combinaison linéaire d'autres vecteurs du système. On dit encore que les vecteurs ui sont 'linéairement indépendants'

Exemples

  1. Un système constitué d'un unique vecteur non nul est un système libre.
  2. Un système constitué d'un vecteur non nul et d'un second vecteur non multiple du premier est libre.
  3. Les trois vecteurs (1,0,0,0) (1,1,0,0) (1,1,1,0) forment un système libre dans l'espace ℝ4.
  4. Pour toute valeur de n le système de n+1 vecteurs (1,x,x2,....,xn) est libre dans l'espace vectoriel des polynômes à coefficients réels.

Remarque

théorème
Tout système extrait d'un système libre est lui-même libre.

Linear relations

definition
Let S=(u1,u2,....,un) be a vector system of E, on K. We call 'linear relation' between the vectors ui a system of scalars (λ1,λ2,...,λn) not all zero verifying:
λ1u1+ λ2u2+ ... + λnun= 0

Not all zero does not mean all non zero!

We immediately verify the following result:
theorem
There is a linear relation for the system S=(u1,u2,....,un) is equivalent to:
at least one vector ui is a linear combination of other vectors of the system.
Thus (-1,-2,-1,1) is a linear relation for the system of polynomials (1,x,x 2,(1+x)2)

Linearly dependent systems

definition
We say that a system S=(u1,u2,....,un) of vectors of E is 'linearly dependent' if there is a linear relationship for this system.
Here are examples of linearly dependent systems:
  1. (u, 2u)
  2. (u,v,u+v)
  3. (u, v, w, w-2u + v)
  4. Any system with the zero vector

Linearly independent systems

definition
We say that a system S=(u1,u2,....,un) of vectors of E is 'linearly independent' if it is not linearly dependent, i.e. if there is no linear relation for this system, or even if none of the vectors of the system is linear combination of other vectors of the system. It is still said that vectors ui are 'linearly independent'

Examples

  1. A system consisting of a single non-zero vector is a linearly independent system.
  2. A system made up of a nonzero vector and a second vector that is not a multiple of the first is linearly independent.
  3. The three vectors (1,0,0,0)(1,1,0,0)(1,1,1,0) form a linearly independent system in the space ℝ 4.
  4. For any value of n the system of n+1 vectors (1,x,x 2,....,x n) is linearly independent in the vector space of polynomials with real coefficients.

Remark

theorem
Any system extracted from a linearly independent system is itself linearly independent.
