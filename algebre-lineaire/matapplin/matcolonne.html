définition
Une 'matrice colonne' est une ne comportant qu'une seule colonne. elle représente donc une d'un de 1 dans un espace de dimension m, m étant le nombre de lignes de la matrice.
Or une application f de K dans F s'identifie à un vecteur de F, plus précisément au vecteur f(1).
L'application se confondant alors avec λ → λf(1).
L'identification est courante. Donc, une base étant fixée dans F le vecteur x de coordonnées (x1,...,xm) de F s'identifie avec la matrice colonne X:
\(\begin{pmatrix} x_{1}\\ x_{2}\\ .\\ .\\ x_{m}\end{pmatrix}\)
Avec ces notations:
théorème
Si f est une application linéaire de E dans F, si B est une base de E et B' une base de F, x un vecteur de E identifié à une matrice colonne X. Si A=M(f,B,B') alors le vecteur y = f(x) s'identifie à un vecteur colonne Y.
La relation entre X et Y est la suivante: Y=AX, formule permettant le calcul de l'image de x en utilisant uniquement les opérations matricielles.
La vérification de cette affirmation est immédiate.

definition
A 'column matrix' is a having only one column. it therefore represents a of a of 1 in an m-dimensional space, where m is the number of rows in the matrix.
Now a map f from K to F is identified with a vector of F, more precisely with the vector f(1).
The map then coincides with λ → λf(1).
Identification is common. Therefore, a base being fixed in F the vector x of coordinates (x1,...,xm) of F is identified with the column matrix X:
\(\begin{pmatrix} x_{1}\\ x_{2}\\ .\\ .\\ x_{m}\end{pmatrix}\)
With these notations:
theorem
If f is a linear map from E to F, if B is a basis of E and B' a basis of F, x a vector of E identified with a column matrix X. If A=M(f,B,B') then the vector y = f(x) identifies with a column vector Y.
The relationship between X and Y is as follows: Y=AX, formula allowing the calculating the image of x using only the operations matrix.
The verification of this assertion is immediate.
