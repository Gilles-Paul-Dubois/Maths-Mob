Since complexes are exactly pairs of reals, storing them in memory generates exactly the same problems as storing reals (see ). You just need to provide an array or a list of two floats. Insofar as complexes are very precise mathematical 'objects', with their rules of formation, their methods and their operations, it is quite appropriate to use an 'object-oriented' language (C++, Python, Java, etc. ...) in which we will define a class (see ) with operator overloading.
