Soient θ1 et θ2 deux réels vérifiant   0 < θl < θ2 < π
Soient r1 et r2 deux réels vérifiant : 0 < r1 < r2
Soit V le sous-ensemble du plan complexe défini par:
V={ z=[ρ, θ] |   ( r1 < ρ < r2 ) ∧ (θ1 < θ < θ2 )}
Montrer que V est ouvert.
Dessiner V quand r1 =1  r2 =2 θ1 = π /4  θ2 = π /2

aide

L'intersection d'un nombre fini d'ouverts est un ouvert.

solution

Soient
P1 ={ z=[ρ , θ] | θ > θ1 }
P2 ={ z=[ρ , θ] | θ < θ2 }
B={ z=[ρ , θ] | ρ < r2 }
U={ z=[ρ , θ] | ρ > r1 }

Alors P1 et P2 sont des demi-plans ouverts, B est un disque ouvert, U est ouvert car complémentaire d'un disque fermé.
V=P1 ∩ P2 ∩ B ∩ U est ouvert comme intersection de 4 ouverts.

Let θ1 and θ2 two reals verifying 0 <θl < θ2 < π
Let r1 and r2 two reals verifying: 0 < r1 < r2
Let V be the subset of the complex plane defined by:
V = {z = [ρ, θ] | (r1 <ρ < r2 ) ∧ (θ1 <θ <θ2 )}
Show that V is open.
Sketch V when r1 =1 r2 =2 θ1 =π/4 θ2 = π/2

hint

The intersection of a finite number of open sets is an open set.

solution

be
P1 = {z = [ρ, θ] | θ> θ1 }
P2 = {z = [ρ, θ] | θ <θ2 }
B = {z = [ρ, θ] | ρ < r2 }
U = {z = [ρ, θ] | ρ> r1 }

SoP1 andP2 are open half-planes, B is an open disk, U is open because complementary to a closed disk.
V=P1 ∩ P2 ∩ B ∩ U is open as the intersection of 4 open sets.
