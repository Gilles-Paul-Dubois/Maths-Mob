Topologie sur ℂ

Topology on ℂ

Définitions et propriétés

Nous allons refaire exactement la même chose qui a été faite pour l'ensemble ℝ, le rôle des intervalles étant joué par celui des 'disques'.
définition
Dans ℂ, le 'disque ouvert' de centre z0 et de rayon r (r réel >0) est ainsi défini :
D(z0,r)={ z ∈ ℂ | | z-z0 |<r}.
Graphiquement on peut voir que cet ensemble correspond exactement à un disque.
définition
De la même façon, le 'disque fermé' de centre z0 et de rayon r (r réel >0) est ainsi défini :
D(z0,r)={ z ∈ ℂ | | z-z0 | ≤ r}.
définition
Un ensemble est dit 'ouvert' dans ℂ, si chaque fois qu'il contient un point z0 il contient un disque de centre z0 et de rayon r >0.
U ouvert ⇔ ∀ z0 ∈ U ∃ r >0 | D(z0,r) ⊆ U.
définition
Un ensemble est dit 'fermé' si son complémentaire est ouvert.
F fermé ⇔ ℂ-F ouvert.
Comme dans ℝ, les ouverts possèdent les propriétés suivantes:
  • ∅ et ℂ sont ouverts.
  • Toute réunion d'ouverts est un ouvert.
  • Toute intersection finie d'ouverts est un ouvert.
  • Un disque ouvert est un ouvert.
Comme dans ℝ, les fermés possèdent les propriétés suivantes:
  • ∅ et ℂ sont fermés.
  • Toute réunion finie de fermés est un fermé.
  • Toute intersection de fermés est un fermé.
  • Un disque fermé est fermé.
Quelques autres exemples :
  • Un demi-plan ouvert ( Re(z)>0 , Im(z) <2, Im(z)<Re(z), etc ...) est un ensemble ouvert.
  • Un demi plan fermé ( Re(z) ≤ 1, Im(z) ≤ 2, etc...) est un ensemble fermé.
  • Un point isolé est un ensemble fermé.
  • Une bande ouverte ( a < Re(z) < b ) est un ensemble ouvert.
  • Une bande fermée ( a ≤ Re(z) ≤ b } est un ensemble fermé.
  • Un rectangle ouvert (a < Re(z) < b et c < Im(z) <d) est un ensemble ouvert.
On peut également définir l'intérieur, l'adhérence et la frontière de tout ensemble:
définition
"L'intérieur" de X est la réunion de tous les ouverts U inclus dans X, c'est un ouvert inclus dans X et c'est le plus grand ouvert contenu dans X.
définition
"L'adhérence" de X est l'intersection de tous les fermés contenant X, c'est un fermé contenant X et c'est le plus petit d'entre eux.
définition
'La frontière' de X est le complémentaire de l'intérieur de X dans l'adhérence de X.
Exemples: Pour un disque ouvert D(z0 ,r). L'intérieur de D c'est lui-même, l'adhérence de D c'est le disque fermé, la frontière de D c'est le cercle |x-z0|=r.
La notion de 'voisinage' d'un point existe aussi.
définition
V est appelé un 'voisinage' de z si V contient un ouvert contenant z.
Un ouvert est un voisinage de chacun de ses points.
Une intersection finie de voisinages de x est encore un voisinage de x.

Definitions and properties

We're going to do exactly the same thing that was done for the set ℝ, the role of intervals being played by that of 'discs' .
definition
In ℂ, the 'open disk' of center z0 and of radius r (r real >0) is thus defined:
D(z0,r)={ z ∈ ℂ | | z-z0 |<r}.
Graphically we can see that this set corresponds exactly to a disk.
definition
In the same way, the 'closed disk' of center z0 and of radius r (real r >0) is defined as follows:
D(z0,r)={ z ∈ ℂ | | z-z0 | ≤ r}.
definition
A set is said 'open' in ℂ, if each time it contains a point z0 it contains a disk with center z0 and of radius r >0.
U open ⇔ ∀ z0 ∈ U ∃ r >0 | D(z0,r) ⊆ U.
definition
A set is said 'closed' if its complement is open.
F closed ⇔ ℂ-F open.
As in ℝ, the open sets have the following properties :
  • ∅ and ℂ are open.
  • Any union of open sets is an open.
  • Any finite intersection of open sets is an open set.
  • An open disk is open.
As in ℝ, closed sets have the properties following:
  • ∅ and ℂ are closed.
  • Any finite union of closed is a closed.
  • Any intersection of closed sets is a closed set.
  • A closed disk is closed.
Some other examples:
  • An open half-plane ( Re(z)>0 , Im(z) <2, Im(z)<Re(z), etc ...) is a open set.
  • A closed half-plane ( Re(z) ≤ 1, Im(z) ≤ 2, etc...) is a closed set.
  • An isolated point is a closed set.
  • An open strip ( a < Re(z) < b ) is a open set.
  • A closed strip ( a ≤ Re(z) ≤ b } is a closed set.
  • An open rectangle (a < Re(z) < b and c < Im(z) < d) is an open set.
One can also define the interior, closure and boundary of any set:
definition
'The interior' of X is the union of all open sets U included in X, it is a open set included in X and it is the largest open set contained in X.
definition
'The closure' of X is the intersection of all the closed sets containing X, it is a closed set containing X and it is the smallest of them.
definition
'The boundary' of X is the complement of the interior of X in the closure of X.
Examples: For an open disk D(z0 ,r). The interior of D is itself, the closure of D is the closed disk, the boundary of D is the circle |xz0|=r.
The notion of 'neighborhood' of a point also exists.
definition
V is called a 'neighborhood' of z if V contains an open set containing z.
An open set is a neighborhood of each of its points.
A finite intersection of neighborhoods of x is still a neighborhood of x.
