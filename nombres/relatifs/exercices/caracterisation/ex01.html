Trouver tous les idéaux de ℤ contenant 48ℤ. Ecrire toutes les relations d'inclusion entre eux.

aide

pℤ⊆qℤ ⇔ q divise p.

solution

Les diviseurs de 48 sont: {1, 2, 3, 4, 6, 8, 12, 16, 24, 48}
Donc les idéaux cherchés sont les pℤ où p appartient à l'ensemble ci-dessus.
24ℤ est contenu dans tous les pℤ pour p dans {1, 2, 3, 4, 6, 8, 12, 24}
16ℤ est contenu dans tous les pℤ pour p dans {1, 2, 4, 8, 16}
12ℤ est contenu dans tous les pℤ pour p dans {1, 2, 3, 4, 6, 12}
8ℤ est contenu dans tous les pℤ pour p dans {1, 2, 4, 8}
6ℤ est contenu dans tous les pℤ pour p dans {1, 2, 3, 6}
4ℤ est contenu dans tous les pℤ pour p dans {1, 2, 4}
3ℤ est contenu dans tous les pℤ pour p dans {1, 3}
2ℤ est contenu dans tous les pℤ pour p dans {1, 2}
