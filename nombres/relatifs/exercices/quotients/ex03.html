Démontrer le théorème dit de 'Wilson'.
Si p est un entier premier
(p-1)!+1 ≡ 0 [p]

aide

Utiliser le fait que (p-1)! est le produit de tous les éléments inversibles du corps ℤ/pℤ.

solution

Dans le facteur (p-1)!=1×2×....×(p-2)×(p-1), on trouve tous les éléments non nuls du corps ℤ/pℤ lesquels sont tous inversibles.
On trouve donc aussi tous leurs inverses. Ainsi on pourra  grouper chaque élément avec son inverse pour obtenir un facteur 1 (donc simplifier) pour peu que le nombre et son inverse soient distincts.
Cherchons donc d'abord les nombres égaux à leur propre inverse.
Ce sont les solutions de l'équation x2=1.
L'équation peut s'écrire x2-1=0 donc (x-1)(x+1)=0 et comme on est dans un corps elle équivaut à x=1 ou x=-1 (ici -1=p-1 ...)
finalement on obtient dans ℤ/pℤ :
(p-1)!=1×(p-1) =-1 CQFD.

Prove the so-called 'Wilson' theorem.
If p is a prime integer
(p-1)!+1 ≡ 0 [p]

hint

Use the fact that (p-1)! is the product of all the invertible elements of the body ℤ/pℤ.

solution

In the factor (p-1)!=1×2×....×(p-2)×(p-1), we finds all non-zero elements of the body ℤ/pℤ which are all invertible.
We also find all their inverses. So we can group each element with its inverse to obtain a factor 1 (therefore simplify) provided that the number and its inverse are distinct.
So let's first look for numbers equal to their own inverse.
These are the solutions of the equation x2 =1.
The equation can be written x2 -1=0 so (x-1)(x+1)=0 and as we are in a body it is equivalent to x=1 or x=-1 (here -1=p-1 ...)
finally we get in ℤ/pℤ :
(p-1)! = 1 × (p-1) = -1 QED.
