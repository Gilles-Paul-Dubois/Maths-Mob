\(C_{p}^{k}=\frac{p!}{(p-k)!k!}=\frac{p(p-1)...(p-k+1)}{k!}\) Tous les facteurs de k! sont premiers avec p puisque k<p.

Write a Python and/or Julia program that verifies identity:
(x+y)p=xp+yp in the field ℤ/pℤ where p is prime.
Demonstrate that \(C_{p}^{k}\) is divisible by p if p is prime and 0 <k <p
Deduce a proof of this identity.
Deduce that if p is prime ap-a=0 ∀a∈ℤ/pℤ and therefore if a ≠ 0 ap-1=1

hint

Use a to compute \(C_{p}^{k}\) together with Gauss's theorem.
For the last question (ap-a=0) use induction.

Solution with Julia

\(C_{p}^{k}=\frac{p!}{(p-k)!k!}=\frac{p(p-1)...(p-k+1)}{k!}\) all factors of k! are coprime with p since k<p.
So in the expansion of (x+y)p by the binomial formula all coefficients are zero except the extremes.
So let's suppose p prime. Then ap-a=0 is true for a= 0, suppose it is true for any a with  0 < a <p-2
Set b=a+1
then bp-b=(a+1)p-(a+1)= ap+1p-a-1=ap-a=0.
