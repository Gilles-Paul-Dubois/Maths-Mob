Définition

définition
Un entier relatif non nul p est dit 'premier' s'il vérifie les conditions suivantes:
  • p est non inversible (p ≠ 1 ∧ p ≠ -1)
  • l'ensemble des diviseurs de p est {-p,-1,1,p}
Il est clair que:
p premier ⇔ -p premier.
Les relatifs premiers sont donc les augmentés de leurs opposés.
Compte tenu de ce qui précède:
(p premier dans ℤ) ⇔ ( |p| premier dans ℕ)
Il n'est donc pas besoin de créer de nouveaux algorithmes pour décider si un relatif p est premier.
Le existe encore pour les entiers relatifs, simplement il demeure vrai 'au signe près'.

Definition

definition
A nonzero integer p is said to be 'prime' if it verifies the following conditions :
  • p is not a unit (p ≠ 1 ∧ p ≠ -1)
  • the set of divisors of p is {-p,-1,1,p}
It's clear that:
p prime ⇔ -p prime.

Considering what is just above :
(p prime in ℤ) ⇔ ( |p| prime in ℕ)
