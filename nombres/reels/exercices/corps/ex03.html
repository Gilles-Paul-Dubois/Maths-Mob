  2. Calculer f(1) et montrer que ∀ x ∈ ℝ* on a f(x-1)=(f(x))-1
  3. Calculer f(x) pour x ∈ ℕ.
  4. Calculer f(x) pour x ∈ ℚ.
  5. Montrer que pour tout x  ≥ 0 on a f(x) ≥ 0.
  6. Montrer que f est une application .

aide

Le seul point non évident est la question 5.
Utiliser le fait que tout réel x ∈ ℝ+ possède une racine carrée.

solution

f(x+0)=f(x)+f(0)=f(x) donc f(0)=0
f(x+(-x))=f(x)+f(-x)=f(0)=0 donc f(-x)=-f(x)
f(a)= f(a×1) = f(a)×f(1) donc f(1)=1 puisque f(a) ≠ 0.
f(x-1x)=f(x)f(x-1)=f(1)=1 donc f(x-1)=f(x)-1
si n ∈ ℕ on a n=1+...+1 (n termes), donc f(n)=f(1)+.....+f(1)= 1+ ...+1 = n
Si r est rationnel  r=p/q on a f(r)=f(p×1/q)= f(p)×f(1/q) = f(p)×1/f(q) = p×1/q = p/q
Si x ≥ 0, soit α tel que α²= x, alors f(x)=f(α)² ≥ 0.
Si y ≥ x alors f(y)=f(x+(y-x))= f(x)+f(y-x) ≥ f(x) puisque y-x ≥ 0
Question 7: f(A)=A puisque tous les éléments de A sont leur propre image. f(λ ) est donc un majorant de f(A)=A puisque f est croissante, comme λ est la borne supérieure de A il en résulte que f(λ) ≥ λ .
On montre de la même façon que f(λ) ≤ λ .
D'où f(λ)= λ et f est l'application identique. Il n'existe donc aucun autre homomorphisme non nul de ℝ dans ℝ que l'application identique.

hint

The only non-obvious point is question 5.
Use the fact that any real x ∈ ℝ+ has a square root.

solution

f(x+0)=f(x)+f(0)=f(x) so f(0)=0
f(x+(-x))=f(x)+f(-x)=f(0)=0 so f(-x)=-f(x)
f(a)= f(a×1) = f(a)×f(1) so f(1)=1 since f(a) ≠ 0.
f(x-1x)=f(x)f(x-1)=f(1)=1 donc f(x-1)=f(x)-1
if n ∈ ℕ we have an=1+...+1 (n terms), so f(n)=f(1)+.....+f(1)= 1+ ...+1 = n
If r is rational  r=p/q we have f(r)=f(p×1/q)= f(p)×f(1/q) = f(p)×1/f(q) = p×1/q = p/q
If x ≥ 0, let α such that α²= x, then f(x)=f(α)² ≥ 0.
If y ≥ x then f(y)=f(x+(yx))= f(x)+f(yx) ≥ f(x) since yx ≥ 0
Question 7: f(A)=A since all elements of A are their own image. f(λ ) is therefore an upper bound of f(A)=A since f is increasing, as λ is the upper bound of A it follows that f(λ) ≥ λ .
We show in the same way that f(λ) ≤ λ .
Hence f(λ)= λ and f is the identical map. There is no hence no other nonzero homomorphism of ℝ in ℝ than the identical application.
