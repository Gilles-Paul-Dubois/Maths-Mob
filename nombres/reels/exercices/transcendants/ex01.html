Soit x un réel. Pour tout polynôme formel P(X)=a0+a1X+a2X2+...+anXn, on désigne par P(x) la valeur de P en x c'est à dire le nombre P(x)=a0+a1x+a2x2+...+anxn.
Démontrer que l'ensemble des P(x) forme un avec l'addition et la multiplication.
Démontrer également que c'est le plus petit sous-anneau de ℝ contenant ℚ et x. On note cet anneau ℚ[x].
A quoi est égal ℚ[x] quand x est rationnel ?
Montrer que si x est , racine d'une équation de degré n, toute puissance de x peut s'exprimer sous la forme: a0+a1x+...+an-1xn-1
Si x ≠ 0, montrer l'équivalence: x dans ℚ[x] ⇔ x algébrique
x et y désignant deux réels, on désigne par ℚ[x,y], le plus petit sous-anneau de ℝ contenant x et y. A quoi est égal cet anneau ?
Montrer que K[x+y] ⊆ K[x,y]

aide

Revoir la définition d'un anneau.
Utiliser un raisonnement par récurrence pour la question 4.

solution

L'ensemble des polynomes formels forme lui-même un anneau avec les opérations usuelles (somme et produit de deux polynomes). On remarque tout de suite que: P(x)+Q(x)=(P+Q)(x) par définition de la somme de deux polynômes.
P(x)Q(x)=PQ(x) par définition du produit de deux polynômes
L'ensemble des valeurs P(x) est donc bien un anneau.
Tout anneau contenant ℚ et x doit nécessairement contenir tout polynôme P(x), c'est donc le plus petit à posséder cette propriété.
Quand x est rationnel, ℚ[x]=ℚ
Supposons x algébrique, alors on a une équation du type: anxn+an-1xn-1+ ... + a1x+a0=0 avec an ≠ 0 où tous les ai ∈ ℚ. Quitte à diviser tous les coefficients par an on peut supposer que an =1. On a alors xn=-an-1xn-1-...-a1x-a0. Donc notre propriété est vraie pour les ordres 0,1,2, ..,n.
Supposons qu'elle soit vraie à l'ordre m.
xm=b0+b1x+...+bn-1xn-1.
Multiplions par x: xm+1=b0x+b1x2+...+bn-2xn-1+bn-1xn.
Il suffit donc maintenant de remplacer xn par -an-1xn-1-...- a1x-a0 pour obtenir notre proposition à l'ordre m+1.
Si x est inversible dans ℚ[x], on a une relation du type: xP(x)=1 soit xP(x)-1=0 donc x est algébrique.
Si x ≠ 0 est algébrique alors on a une équation: anxn+an-1xn-1+...+a1x+a0=0 avec a0 ≠ 0.
On peut réecrire cela: anxn+an-1xn-1+...+a1x=-a0
En divisant les deux membres par -a0 on voit que:
x(anxn-1+an-1xn-2+...+a2x+a1)/-a0=1
Ce qui prouve que x est inversible.
ℚ[x,y] est bien évidemment l'ensemble des P(x,y) où P parcourt l'ensemble des polynômes à deux variables X et Y. Comme X+Y est un de ces polynômes, il en résulte que x+y ∈ ℚ [x,y], donc tout nombre de la forme P(x+y) est aussi dans ℚ [x,y], ce qui achève notre démonstration.
Remarque: Cet exercice constitue le début de la démonstration du fait que les nombres algébriques forment un sous-corps de ℝ, pour conclure il nous manque seulement un résultat d'algébre linéaire (théorie de la dimension).

Let x be a real number. For any formal polynomial P(X)=a0+a1X+a2X2+...+anXn, we denote by P(x) the value of P at x, i.e. the number P(x)=a0+a1x+a2x2+...+anxn.
Prove that the set of P(x) forms a with the addition and multiplication.
Also demonstrate that it is the smallest subring of ℝ containing ℚ and x. Let's denote this ring ℚ[x].
What is ℚ[x] when x is rational?
Show that if x is , root of a degree n equation, any power of x can be expressed as: a0+a1x+...+an-1xn-1
If x ≠ 0, show equivalence: x in ℚ[x] ⇔ x algebraic
x and y denoting two real numbers, we denote by ℚ[x,y], the smallest subring of ℝ containing x and y. What is this ring ?
Show that K[x+y] ⊆ K[x,y]

hint

Review the definition of a ring.
Use induction for question 4.

solution

The set of formal polynomials itself forms a ring with the usual operations (sum and product of two polynomials). We immediately notice that: P(x)+Q(x)=(P+Q)(x) by definition of the sum of two polynomials.
P(x)Q(x)=PQ(x) by definition of the product of two polynomials
The set of values ​​P(x) is therefore indeed a ring.
Any ring containing ℚ and x must necessarily contain any polynomial P(x), so it is the smallest to possess this property.
When x is rational, ℚ[x]=ℚ
Suppose x is algebraic, then we have an equation of the type: anxn+an-1xn-1+ ... + a1x+a0=0 with an ≠ 0 where all ai ∈ ℚ. Even if it means dividing all the coefficients by an we can assume that an =1. We then have xn=-an-1xn-1-...-a1x-a0. So our property is true for the orders 0,1,2, ...,n.
Suppose it is true at order m.
xm=b0+b1x+...+bn-1 xn-1.
Multiply by x: xm+1=b0x+b1x2+...+bn-2xn-1+bn-1xn.
So now just replace xn by -an-1xn-1-...- a1x-a0to obtain our proposition of order m+1.
If x is invertible in ℚ[x], we have a relation of the type: xP(x)=1 or xP(x)-1=0 so x is algebraic.
If x ≠ 0 is algebraic then we have an equation: anxn+an-1xn-1+...+a1x+a0=0 with a0≠ 0.
We can rewrite this: anxn+an-1xn-1+...+a1x=-a0
Dividing both sides by -a0 we see that:
x(anxn-1+an-1xn-2+...+a2x+a1)/-a0=1
This proves that x is invertible.
ℚ[x,y] is obviously the set of P(x,y) where P runs through the set of polynomials in two variables X and Y. As X+Y is one of these polynomials, it follows that x+y ∈ ℚ [x,y], so any number of the form P(x+y) is also in ℚ [x,y], which completes our proof.
Note: This exercise constitutes the beginning of the proof that the algebraic numbers form a subfield of ℝ, to conclude we only lack a result of linear algebra (theory of dimension).
