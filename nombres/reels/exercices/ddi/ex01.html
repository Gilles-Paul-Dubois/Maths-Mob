Write a program that converts a decimal number written in base 10, into the same number written in base 2 (with comma).
Make sure that we can fix the precision (number of digits after the decimal point).

hint

Use a decimal → binary conversion function for the integers.
Multiply the number to be converted by an appropriate power of 2.

solution

NB: English speak readers may change the comma ',' into dot in accordance with the usage of their own country. This original program was made for France specially.
