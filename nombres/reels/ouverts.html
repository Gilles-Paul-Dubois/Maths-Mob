Les intervalles de ℝ sont définis exactement comme les de ℚ.
En particulier, les définitions d'intervalles ouverts et fermés sont les mêmes.
définition
Un sous-ensemble de ℝ est dit 'ouvert', si chaque fois qu'il contient un point x il contient un intervalle ouvert non vide contenant x.
Formellement :
U ouvert ⇔ ( ∀ x ∈ U ∃ (a,b) ∈ ℝ × ℝ | a<x<b ∧ ]a,b[ ⊆ U)
Cette définition entraîne immédiatement les propriétés suivantes:
  • ∅ et ℝ sont ouverts.
  • Tout intervalle ouvert est un ensemble ouvert.
  • Toute réunion d'ouverts est encore un ouvert.
  • de deux, ou d'un nombre fini d'ouverts est encore un ouvert.
  • Un n'est pas ouvert
définition
On dit maintenant qu'un sous-ensemble F de ℝ est 'fermé' si son complémentaire ℝ-F est ouvert.
Cette nouvelle définition entraîne les propriétés suivantes :
  • ∅ et ℝ sont fermés.
  • Tout intervalle fermé est un ensemble fermé.
  • Toute finie de fermés est encore est encore un fermé.
  • L'intersection d'une famille quelconque (même infinie) de fermés est encore un fermé.
  • Un singleton est fermé.
  • Tout ensemble fini est fermé.
Nous allons montrer maintenant que l'intersection d'une famille infinie d'ouverts n'est pas forcément un ouvert.
Désignons, pour n ∈ ℕ* par I n l'intervalle ]-1/n, +1/n[ . L'intersection de tous les I n est {0} et ce n'est pas un ouvert. Nous montrons aussi que la réunion d'une famille infinie de fermés n'est pas forcément un fermé. Désignons, pour n ∈ ℕ* par F n le singleton {1/n}. La réunion X de tous les F n est donc constituée de tous les points de la suite u n =1/n pour n ≠ 0. Il est clair que 0 ∉ X. X fermé ⇔ ℝ-X ouvert. Or si ℝ-X était ouvert comme il contient 0 il devrait contenir un intervalle ]a,b[ avec a<0 et b>0, or ceci est faux car tout tel intervalle contient un point de X ( ∀ b >0 ∃ n ∈ ℕ* | 1/n <b).
Donnons encore quelques définitions qui peuvent être utiles:
définition
Soit X ⊆ ℝ la réunion de tous les ouverts contenus dans X est encore un ouvert contenu dans X et c'est évidemment le plus grand d'entre eux. On l'appelle "l'intérieur" de X.
définition
L'intersection de tous les fermés contenant X est encore un fermé contenant X et c'est le plus petit d'entre eux . On l'appelle "l'adhérence" de X (notation X).
Quelques exemples:
  • Si U est ouvert son intérieur est égal à U.
  • Si F est fermé son adhérence est égale à F.
  • Si I=[a,b] ou I=[a,b[ ou I= ]a,b] l'intérieur de I est ]a,b[.
  • Si I=]a,b] ou I= [a,b[ ou I=]a,b[ l'adhérence de I est [a,b].
  • L'intérieur d'un ensemble fini est toujours vide.
définition
Soit maintenant X un sous-ensemble quelconque de ℝ. On appelle 'frontière' de X l'intersection de X et de ℝ-X.
Exemples:
  • Les frontières de ∅ et de ℝ sont vides.
  • La frontière de [a,b] est {a,b}.
  • La frontière de ]a,b] est {a,b}.
  • La frontière de ]a,b[ est {a,b}.
  • La frontière de [a,b[ est {a,b}.
  • La frontière d'un ensemble fini de points est égale à cet ensemble.
The intervals of ℝ are defined exactly like the of ℚ.
In particular, the definitions of open and closed intervals are the same.
definition
A subset of ℝ is said 'open' , if every time it contains a point x it contains a non-empty open interval containing x.
Formally:
U open ⇔ ( ∀ x ∈ U ∃ (a,b) ∈ ℝ × ℝ | a<x<b ∧ ]a,b[ ⊆ U)
This definition immediately results in the following properties:
  • ∅ and ℝ are open.
  • Any open interval is an open set.
  • Every union of open sets is still open.
  • of two, or a finite number of open sets is again open.
  • A is not open
definition
We now say that a subset F of ℝ is 'closed' if its complement ℝ-F is open.
This new definition results in the following properties:
  • ∅ and ℝ are closed.
  • Any closed interval is a closed set.
  • All finite >of closed sets is still closed.
  • The intersection of any (even infinite) family of closed sets is still a closed set.
  • A singleton is closed.
  • Every finite set is closed.
We will now show that the intersection of an infinite family of open sets is not necessarily an open set.
Denote, for n ∈ ℕ* by I n the interval ]-1/n, +1/n[ . The intersection of all I n is {0} and it is not open. We also show that the union of an infinite family of closed sets is not necessarily a closed set. Denote, for n ∈ ℕ* by F n the {1/n} singleton. The union X of all F n is therefore made up of all the points of the sequence u n =1/n for n ≠ 0. It is clear that 0 ∉ X. X closed ⇔ ℝ-X open. But if ℝ-X were open as it contains 0 it should contain an interval ]a,b[ with a<0 and b>0, or this is false because any such interval contains a point of X ( ∀ b >0 ∃ n ∈ ℕ* | 1/n<b).
Let's give a few more definitions that may be useful:
definition
Let X ⊆ ℝ be the union of all open contents in X is still an open contained in X and this is obviously the most big of them. It is called "the interior" of X.
definition
The intersection of all closed sets containing X is still a closed set containing X and it is the smallest of them. It is called the "closure" of X (notation X ).
Some examples:
  • If U is open its interior is equal to U.
  • If F is closed its closure is equal to F.
  • If I=[a,b] or I=[a,b[ or I= ]a,b] the interior of I is ]a,b[.
  • If I=]a,b] or I= [a,b[ or I=]a,b[ the closure of I is [a,b].
  • The interior of a finite set is always empty.
definition
Let X be any subset of ℝ. We call 'boundary' of X the intersection of X and of ℝ-X .
Examples:
  • The boundaries of ∅ and ℝ are empty.
  • The boundary of [a,b] is {a,b}.
  • The boundary of ]a,b] is {a,b}.
  • The boundary of ]a,b[ is {a,b}.
  • The boundary of [a,b[ is {a,b}.
  • The boundary of a finite set of points is equal to this set.
