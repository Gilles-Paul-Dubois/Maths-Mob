Métrique sur ℝ

Metric on ℝ

Définition

définition
Soit E un ensemble quelconque, on appelle 'distance' sur E, toute application d: E×E → ℝ vérifiant les propriétés suivantes:
  • d(x,y) ≥ 0 ∀ (x,y) ∈ E×E
  • d(x,y)=d(y,x) ∀ (x,y) ∈ E×E (symétrie)
  • d(x,y)=0 ⇔ x=y ∀ (x,y) ∈ E×E
  • d(x,z) ≤ d(x,y)+d(y,z) ∀ (x,y,z) ∈ E×E×E (inégalité dite 'triangulaire')
Il existe de nombreux exemples de telles fonctions comme la distance 'euclidienne' de deux points dans le plan ou dans l'espace.
définition
La donnée d'un ensemble E et d'une distance d sur E, soit un couple (E,d) est appelée un 'espace métrique' .
définition
La 'valeur absolue' d'un nombre réel est définie comme dans le cas de ℤ et de ℚ.
|x|=Sup(x,-x)
ou encore
|x| = x si x>0 ou -x si x ≤ 0
Nous affirmons que sur ℝ×ℝ la fonction d(x,y)=|x-y| est une distance.
Les trois premières propriétés sont évidentes. La quatrième s'énonce:
|x-z| ≤ |x-y|+|y-z|
posant a=x-y et b=y-z, elle équivaut à:
|a+b| ≤ |a|+|b|
Elle est évidente quand les 2 nombres a et b sont de même signe. Vu la symétrie de la relation en a et b on peut supposer en toute généralité pour achever la démonstration que a ≥ 0 et b ≤ 0. Si |a| >|b| alors |a+b| ≤ |a| dans le cas contraire |a+b| ≤ |b|, donc dans tous les cas |a+b| ≤ |a|+|b|.

Definition

definition
Let E be any set, we call 'distance' on E, any map d: E×E → ℝ verifying the following properties :
  • d(x,y) ≥ 0 ∀ (x,y) ∈ E×E
  • d(x,y)=d(y,x) ∀ (x,y) ∈ E×E (symmetry)
  • d(x,y)=0 ⇔ x=y ∀ (x,y) ∈ E×E
  • d(x,z) ≤ d(x,y)+d(y,z) ∀ (x,y,z) ∈ E×E×E (so-called 'triangular' inequality)
There are many examples of such functions like 'Euclidean' distance of two points in the plane or in space.
definition
The data of a set E and a distance d on E, i.e. a pair (E,d) is called a 'metric space' .

The 'absolute value or modulus' of a real number is defined as in the case of ℤ and ℚ.
|x|=Sup(x,-x)
or
|x| = x if x>0 or -x if x ≤ 0
We assert that on ℝ×ℝ the function d(x,y)=|x-y| is a distance.
The first three properties are obvious. The fourth is:
|x-z| ≤ |x-y|+|y-z|
setting a=x-y and b=y-z, it is equivalent to:
|a+b| ≤ |a|+|b|
It is obvious when the 2 numbers a and b have the same sign. Saw the symmetry of the relation in a and b one can suppose in all generality to complete the proof that a ≥ 0 and b ≤ 0. If |a| >|b| then |a+b| ≤ |a| otherwise |a+b| ≤ |b|, so in all cases |a+b| ≤ |a|+|b|.
