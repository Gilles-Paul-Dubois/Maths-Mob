Comme nous l'avons déjà vu, il est possible de passer d'une construction à l'autre, associer un DDI à une suite de Cauchy, associer un DDI à une coupure, etc. Nous avons vu aussi que dans tous les cas ℚ s'identifie à une partie du nouvel ensemble construit.
  • A un rationnel donné peut être associé un et un seul DDI propre (périodique).
  • A un rationnel donné peut être associée une coupure unique.
  • A un rationnel donné peut être associée une unique classe d'équivalence de suites de Cauchy.
Il s'agit ici de 'prolonger' la relation d'ordre ≤ sur ℚ, c'est à dire de définir une relation d'ordre ≤ sur ℝ, compatible avec la précédente en ce sens que si x et y sont des réels associés à des rationnels r et s (par l'une quelconque des méthodes exposées). Les deux relations x ≤ y et r ≤ s soient équivalentes.
Cela fait, nous définirons les relations associées ≥, <, > de la façon usuelle ≥ est la réciproque de ≤, x<y signifie x ≤ y et x ≠ y, etc.
Commençons avec les DDI. Nous allons d'abord comparer deux DDI propres positifs x et y qui s'écriront donc:
x=e,a1a2....an ...... e partie entière, ai décimales
y=f,b1b2...bn ...... f partie entière, bi décimales
Nous commençons par comparer les parties entières. Si e < f, nous écrivons que x ≤ y.
Si f >e nous affirmons que x ≤ y est fausse.
Si e=f, nous comparons les décimales ai et bi une à une dans l'ordre.
Si a0 < b0 nous affirmons que x ≤ y.
Si a0 >b0 nous affirmons que x ≤ y est fausse.
si a0 =b0 , nous passons à la comparaison de a1 et b1, et ainsi de suite.
Si toutes les décimales sont égales nous écrivons encore x ≤ y (cas de l'égalité).
L'introduction des négatifs ne pose pas de problèmes particulier si x et y sont négatifs nous conviendrons que x ≤ y ⇔ -y ≤ -x, que tout nombre négatif est ≤ à tout positif, et que tout nombre strictement positif n'est jamais ≤ à un nombre négatif.
Il reste évidemment à vérifier que l'on obtient bien ainsi une relation transitive, réflexive et antisymétrique. Nous omettrons la démonstration.
Pour les suites de Cauchy on peut utiliser la définition résultant de la remarque suivante.
La relation entre deux suites de Cauchy r=(rn)n∈ℕ et s=(sn)n∈ℕ , définie par r ≤ s ⇔ ∃ m | n ≥ m ⇒ rn ≤ sn ∀ n ∈ ℕ est une relation d'ordre compatible avec la relation d'équivalence définissant les réels.
La démonstration de ce résultat est longue, purement technique et de peu d'intérêt, il s'agit de vérifier les axiomes des et d'appliquer la définition de la relation d'équivalence des suites de Cauchy.

As we have already seen, it is possible to go from one construction to another, associate an UDE with a Cauchy sequence, associate a DDI to a cut, etc. We have also seen that in all cases ℚ identifies with a part of the new set built.
  • A given rational can be associated with one and only (periodic) proper UDE.
  • A given rational can be associated with a single cut.
  • A given rational can be associated with a single class of equivalence of Cauchy sequences.
This is about 'to extend' the order relation ≤ on ℚ, i.e. to define a relation of order ≤ on ℝ, compatible with the previous in the sense that if x and y are real numbers associated with rationals r and s (by any of the methods discussed). The two relations x ≤ y and r ≤ s are equivalent.
Having done this, we will define the associated relations ≥, <, > in the usual way ≥ is the reciprocal of ≤, x<y means x ≤ y and x ≠ y, etc.
Let's start with the UDEs. We will first compare two proper positive UDEs x and y which will therefore be written:
x=e.a1a2....an ...... e integer part, ai decimals digits
y=f.b1b2...bn ...... f integer part, bi decimals digits
We start by comparing the integer parts. If e < f, we write that x ≤ y.
If f >e we assert that x ≤ y is false.
If e=f, ​​we compare the digits ai and bi one by one in order.
If a0 <b0 we assert that x ≤ y.
If a0 >b0 we assert that x ≤ y is false.
if a0 =b0 , we go to the comparison of a1 and b1, And so on.
If all the decimals are equal we still write x ≤ y (case of equality).
The introduction of negatives does not pose any particular problems if x and y are negative we will agree that x ≤ y ⇔ -y ≤ -x, that every negative number is ≤ to every positive, and that every number strictly positive is never ≤ a negative number.
It obviously remains to verify that we thus obtain a relation transitive, reflexive and antisymmetric. We will omit the demonstration.
For Cauchy sequences we can use the definition resulting from the following remark.
The relation between two Cauchy sequences r=(rn)n∈ℕ and s=(sn)n∈ℕ , defined by r ≤ s ⇔ ∃ m | n ≥ m ⇒ rn ≤sn ∀n ∈ ℕ is a partial order compatible with the equivalence relation defining the reals.
The proof of this result is long, purely technical and of little interest, it is a question of verifying the axioms of and to apply the definition of the equivalence relation of sequences of Cauchy.
