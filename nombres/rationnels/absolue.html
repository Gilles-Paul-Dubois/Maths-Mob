Définitions

Si (x,y) ∈ℚ×ℚ on pose :
définition
\[sup(x,y)=\begin{cases} & x \text{ si } x\geqslant y\\ & y \text{ sinon } \end{cases}\]
sup(x,y) est donc le plus grand élément de l'ensemble {x,y}
On pose également.
définition
\[x_{+}=sup(x,0)\] \[x_{-}=sup(-x,0)\] \[\left| x\right|=\begin{cases} & x \text{ si } x\geqslant 0 \\ & -x \text{ sinon } \end{cases}\]
On définit enfin la 'valeur absolue' de x :
définition
\[ abs(x)=\left|x \right|=sup(x,-x)\]

Inégalités usuelles

Nous avons donc les relations:
\[x=x_{+}-x_{-}\] \[\left| x\right|=x_{+}+x_{-}\] \[\left| x\right|=\begin{cases} & x \text{ si } x\geqslant 0 \\ & -x \text{ sinon } \end{cases}\]
Voici une inégalité très importante dont la preuve ne présente aucune difficulté:
\(\left|x+y \right|\leqslant \left|x \right|+\left| y\right|\) ∀ (x,y) ∈ℚ×ℚ

Qui peut aussi s'écrire:

\(\left|x-z \right|\leqslant \left|x -y\right|+\left| y-z\right|\) ∀ (x,y,z) ∈ℚ×ℚ×ℚ
Cette inégalité s'étend immédiatement par récurrence à n éléments x 1, x 2, ..., x n :
\( \left|x_{n}-x_{1} \right|\leqslant \left| x_{2}-x_{1}\right|+\left| x_{3}-x_{2}\right|+\text{ ... } + \left| x_{n}-x_{n-1}\right|\)
Pour finir traduisons l'inégalité
\( \left| x\right|< \epsilon \Leftrightarrow -\epsilon < x< \epsilon \)
Et enfin :
\( \left| x-a\right|< \epsilon \Leftrightarrow a-\epsilon < x< a+\epsilon \)

Toutes ces équivalences sont d'une utilisation permanente.

Definitions

If (x,y) ∈ℚ×ℚ we define :
definition
\[sup(x,y)=\begin{cases} & x \text{ if } x\geqslant y\\ & y \text{ else } \end{cases}\]
Hence sup(x,y) is the largest element of the set {x,y}
We define as well :
definition
\[x_{+}=sup(x,0)\] \[x_{-}=sup(-x,0)\] \[\left| x\right|=\begin{cases} & x \text{ if } x\geqslant 0 \\ & -x \text{ else } \end{cases}\]
Finally we define the 'absolute value or modulus' of x :
definition
\[ abs(x)=\left|x \right|=sup(x,-x)\]

Usual inequalities

So we have the relations :
\[x=x_{+}-x_{-}\] \[\left| x\right|=x_{+}+x_{-}\] \[\left| x\right|=\begin{cases} & x \text{ if } x\geqslant 0 \\ & -x \text{ else } \end{cases}\]
Here is a very important inequality the proof of which is not difficult :
\(\left|x+y \right|\leqslant \left|x \right|+\left| y\right|\) ∀ (x,y) ∈ℚ×ℚ

That we can write also :

\(\left|x-z \right|\leqslant \left|x -y\right|+\left| y-z\right|\) ∀ (x,y,z) ∈ℚ×ℚ×ℚ
This inequality can be extended by induction to a finite number n of elements x 1, x 2, ..., x n :
\( \left|x_{n}-x_{1} \right|\leqslant \left| x_{2}-x_{1}\right|+\left| x_{3}-x_{2}\right|+\text{ ... } + \left| x_{n}-x_{n-1}\right|\)
To finish let's translate some inequalities :
\( \left| x\right|< \epsilon \Leftrightarrow -\epsilon < x< \epsilon \)
And finally :
\( \left| x-a\right|< \epsilon \Leftrightarrow a-\epsilon < x< a+\epsilon \)

All of these equivalences are of permanent use, mainly in calculus.
