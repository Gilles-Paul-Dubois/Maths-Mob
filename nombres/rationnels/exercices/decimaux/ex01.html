Montrer que l'ensemble des nombres 'dyadiques' { r ∈ ℚ |∃ (a,k) ∈ ℤ×ℕ | r=a/2k } est dans ℚ.

aide

Les dyadiques sont l'équivalent des décimaux en base 2.

solution

Il s'agit de montrer que tout rationnel r peut être approché par un dyadique d'aussi près qu'on veut.
Montrons d'abord cela si r ∈ [0,1].
Tout d'abord on peut supposer que r n'est pas dyadique sans cela il n'y a rien à démontrer.
Nous allons utiliser un procédé qu'on appelle 'dichotomie' pour fabriquer une suite rn de rationnels telle que:
|rn-r|<1/2n
On divise [0,1] en 2 intervalles de longueur 1/2 [1,1/2] [1/2,1]
r se trouve forcément dans l'un des deux sans être une borne.
si r ∈ [0,1/2] on pose r1=0 sinon r1=1/2
Par la suite si r1=0 on partage à son tour [0,1/2] en [0,1/4], [1/4,1/2] Supposons que r ∈ [1/4, 1/2] alors r2=1/4 sinon r2=0
Ainsi on construit par récurrence une suite rn telle que rn=k/2n où k est un entier <2n.
rn+1 se déduit de rn de la façon suivante:
rn se trouve dans un intervalle [i/2n,(i+1)/2n] que nous coupons en deux [i/2n,i/2n+1/2n+1]  [i/2n+1/2n+1,(i+1)/2n].
r appartient à l'un de ces deux sous-intervalles s'il appartient au premier on pose :
rn+1=rn
sinon
rn+1 =rn+1/2n+1.
Supposons maintenant que r soit >0 et non compris dans l'intervalle [0,1] alors r ∈ [k,k+1] où k ∈ ℕ.
Alors r-k est dans [0,1] et peut être approché par une suite rn de dyadiques d'aussi près qu'on veut.
Donc les dyadiques k+rn approchent r d'aussi près qu'on veut.
Si maintenant r <0 on a (-r) >0 donc -r peut être approché par une suite de dyadiques rn et r est approché par la suite (-rn)

Show that the set of 'dyadic' numbers { r ∈ ℚ |∃ (a,k) ∈ ℤ×ℕ | r=a/2k } is in ℚ.

hint

Dyadics are equivalents of decimals in base 2

solution

We must show that everery rational r can be approached by a dyadic as close as we want.
Let's begin to show it wheni r ∈ [0,1].
We can suppose first that r is not dyadic, otherwise nothing to prove
We will use aprocess which we call 'dichotomy' to build a sequence rn of rationals such that :
|rn-r|<1/2n
We divide [0,1] in 2 sub-intervals with length 1/2 [1,1/2] [1/2,1]
r is by necessity in one of them without being a bound.
If r ∈ [0,1/2] we set r1=0 else r1=1/2
Then if r1=0 we share in own turn [0,1/2] in [0,1/4], [1/4,1/2] Suppose that r ∈ [1/4, 1/2] then r2=1/4 else r2=0
So we build by induction a sequence rn such that rn=k/2n where k is a natural <2n.
rn+1 is deduced from rn the following way :
rn is located in an interval [i/2n,(i+1)/2n] which we cut in two [i/2n,i/2n+1/2n+1]  [i/2n+1/2n+1,(i+1)/2n].
r belongs to one of these sub-intervals if it belongs to the first, we set :
rn+1=rn
else
rn+1 =rn+1/2n+1.
Suppose now that >0 and not included in [0,1] then r ∈ [k,k+1] where k ∈ ℕ.
then r-k is in [0,1] and can be approached by a sequence rn of dyadics as close as we want.
So the dyadics k+rn approach r as close as we want.
If now r <0 we have (-r) >0 so -r can be approached by a sequence of dyadics rn and ris approached by the sequence (-rn) .
