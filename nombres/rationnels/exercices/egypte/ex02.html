La conjecture de Sierpinski peut s'ennoncer ainsi:
Pour tout entier n >5 la fraction 5/n peut être décomposée en au plus 3 fractions égyptiennes:

\(\frac{5}{n}=\frac{1}{a}+\frac{1}{b}+\frac{1}{c}\)

On ne sait pas démontrer ce résultat à ce jour.
Pouvez vous écrire un programme qui vérifie cette conjecture pour n ≤ 5000.

aide

Commencer par établir qu'on peut supposer n premier.
Utiliser ensuite une génération rapide des nombres premiers entre 2 et 5000
Utiliser ensuite et en cas d'échec un algorithme type 'force brute'.

Sierpinski's conjecture can be stated as follows:
For any integer n >5 the fraction 5/n can be decomposed into at most 3 Egyptian fractions:

\(\frac{5}{n}=\frac{1}{a}+\frac{1}{b}+\frac{1}{c}\)

We do not yet know how to prove this result.
Can you write a program that verifies this conjecture for n ≤ 5000.

hint

Start by establishing that we can assume n to be prime.
Then use a fast generation of prime numbers between 2 and 5000
Use next and in case of failure a 'brute force' type algorithm.
