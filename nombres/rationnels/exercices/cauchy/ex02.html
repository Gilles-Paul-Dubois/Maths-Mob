Montrer que toute suite de rationnels vérifiant:
|un+1-un| ≤ 1/2n
est une suite de Cauchy.

aide

Trouver une majoration de :
|un-um| en utilisant cette .

solution

Supposons m ≤ n, on a donc
|un-um| ≤ |um+1-um|+| um+2-um+1|+....+|un-un-1| ≤ 1/2m+1/2m+1+....+1/2n = 1/2m+1/2m+1+....+1/2m+n-m = 1/2m(1+1/2+...+1/2n-m).
Mais (1+1/2 +...+1/2n-m) = (1-(1/2)n-m+1)/(1-1/2)=2(1-(1/2)n-m+1) ≤ 2
Donc
|un-um| ≤ (1/2m)×2 = 1/2m-1
Si maintenant ε est un nombre >0 quelconque
choisissons N tel que 1/2N-1<ε
Alors si m,n ≥ N on a bien
|un-um| ≤ ε
