Montrer que la suite (un)n∈ℕ, définie par:
un=(3n+1)/(5n+2)
est une suite de Cauchy de rationnels.

aide

Montrer que pour m ≥ n on a:
|um-un| ≤ |1-m/n|/15m

solution

Un calcul direct donne:
um-un = (m-n)/(15mn+10m+6n+4)
Comme 15mn+10m+6n+4 ≥ 15 mn
Il en résulte:
|um-un| ≤ |m-n|/15mn et en divisant par n si n > m ou par m dans le cas contraire
|um-un| ≤ |1-m/n|/15m ≤ 1/15m
ou
|um-un| ≤ |1-n/m|/15n ≤ 1/15n
Soit maintenant ε >0 donné
Choisissons N suffisamment grand pour que 1/15N ≤ ε
Alors pour m,n ≥ N on a bien
|um-un| ≤ ε

Show that the sequence (un)n∈ℕ, defined by:
un=(3n+1)/(5n+2)
is a Cauchy sequence of rationals.

hint

Show that for m ≥ n we have :
|um-un| ≤ |1-m/n|/15m

solution

Direct computation gives :
um-un = (m-n)/(15mn+10m+6n+4)
since 15mn+10m+6n+4 ≥ 15 mn
We have :
|um-un| ≤ |m-n|/15mn and dividing by n if n > m or by m else
|um-un| ≤ |1-m/n|/15m ≤ 1/15m
or
|um-un| ≤ |1-n/m|/15n ≤ 1/15n
Let now ε >0 a given number
Choose N big enough so that 1/15N ≤ ε
Then if m,n ≥ N we have
|um-un| ≤ ε
