Sachant que |x-1| ≤ 2 et -5 ≤ y ≤ -4, encadrer les quantités:
  1. x+y
  2. x-y
  3. xy
  4. x/y
  5. |x|-|y|

aide

L'inégalité |x-1| ≤ 2 peut se traduire par   -1 ≤ x ≤ 3

solution


Par simple addition on a:
-6 ≤ x+y ≤ -1
On a en inversant les signes:
4 ≤ -y ≤ 5, et par addition:
3 ≤ x-y ≤ 8
Partant de  -1 ≤ x ≤ 3 et -5 ≤ y ≤ -4 on obtient  -12 ≤ xy ≤ 5
Puis  -3/4 ≤ x/y ≤ 1/5
Enfin |x| ≤ 3 et |y| ≤ 5 donne:
-5 ≤ |x| - |y| ≤ 3

Knowing that |x-1| ≤ 2 and -5 ≤ y ≤ -4, find upper and lower bounds for the quantities :
  1. x+y
  2. x-y
  3. xy
  4. x/y
  5. |x|-|y|

hint

Inequality; |x-1| ≤ 2 can be translated by   -1 ≤ x ≤ 3

solution


By simple addition we get :
-6 ≤ x+y ≤ -1
By reversing signs :
4 ≤ -y ≤ 5, et par addition:
3 ≤ x-y ≤ 8
Starting from  -1 ≤ x ≤ 3 and -5 ≤ y ≤ -4 we get  -12 ≤ xy ≤ 5
Then  -3/4 ≤ x/y ≤ 1/5
At last |x| ≤ 3 and |y| ≤ 5 give :
-5 ≤ |x| - |y| ≤ 3
