#décomposition de la fraction p/q en fraction continue 
 
 
def fracontinue (p,q): 
    sol=[] 
    p0=p 
    p1=q 
    while p1: 
        a=p0//p1 
        sol.append(a) 
        p=p1 
        p1=p0-a*p1 
        p0=p 
    return sol 
 
def main(): 
    print(fracontinue(15625,6842)) 
 
if __name__ == '__main__': 
    main()
