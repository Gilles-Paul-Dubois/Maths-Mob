Algorithme dit du crible d'Atkin

Voici la description de cet algorithme qui est un raffinement du crible d'Eratostène :

  • Tous les restes sont des restes modulo 60
  • Tous les nombres, y compris x et y , sont des entiers naturels.
  • Switcher une case voudra dire la marquer ou la démarquer (premier ou non).
  1. Créer une liste de résultats initialisée avec 2, 3, et 5.
  2. Créer une liste à cribler les nombres étant tous au départ marqués non premiers.
  3. Pour chaque nombre n dans le crible :
    • Si le reste dans la division par 60  appartient à {1, 13, 17, 29, 37, 41, 49,  53} switcher le pour toute solution de 4x2+y2=n
    • Si le reste dans la division par 60  appartient à {7, 19, 31,  43}, switcher le pour toute solution de 3x2+y2= n.
    • Si le reste dans la division par 60  appartient à { 11, 23, 47,  59},  switcher le pour toute solution de 3x2-y2=n où x> y.
    • Si n possède un autre reste ignorez le complètement.
  4. Démarrez avec le nombre le plus petit dans le crible.
  5. Prenez le nombre suivant dans le crible marqué premier
  6. Inclure ce nombre dans la liste.
  7. Marquez comme non premiers tous les multiples du carré de ce nombre.
  8. Bouclez sur l'instruction 5 jusqu'à épuisement du crible.

Atkin sieve algorithm

Here is the description of this algorithm which is a refinement of the Sieve of Eratostene:

  • All remainders are modulo 60 remainders
  • All numbers including x and y , are natural numbers.
  • Switching a box will mean marking it or unmarking it (prime or not).
  1. Create a result list initialized with 2, 3, and 5.
  2. Create a list to be sieved with the numbers all initially marked as non-prime.
  3. For each number n in the sieve:
    • If the remainder in division by 60 belongs to {1, 13, 17, 29, 37, 41, 49, 53} switch it for any solution of 4x2+y2=n.
    • If the remainder in division by 60 belongs to {7, 19, 31, 43}, switch it for any solution of 3x2+y2= n.
    • If the remainder in division by 60 belongs to { 11, 23, 47, 59}, switch it for any solution of 3x2-y2= n where x> y.
    • If n has another remainder ignore it completely.
  4. Start with the smallest number in the sieve.
  5. Take the next number in the sieve marked first
  6. Include this number in the list.
  7. Mark as non-prime all multiples of the square of this number.
  8. Loop on instruction 5 until the sieve is exhausted.
