Montrer que toute partie infinie de ℕ est dénombrable.

aide

Utiliser le fait que toute partie non vide de ℕ possède un plus petit élément.

solution

Soit P une partie infinie de ℕ.
Désignons par x0 le plus petit élément de P.
Par x1, le plus petit élément de P-{x0}
Par x2, le plus petit élément de P-{x0,x1}
et ainsi de suite.
La suite (xn) est strictement croissante. Il en résulte que x → xn est injective, mais elle est aussi surjective car si y est un élément quelconque de P, il existe un indice n0 tel que xn0 >y, si y n'avait pas d'antécédent cela contredirait la définition de xn0

Show that any infinite part of ℕ is countable.

hint

Use the fact that any non-empty part of ℕ has a smallest element.

solution

Let P be an infinite part of ℕ.
Denote by x0 most small element of P.
By x1, the smallest element of P-{x0}
By x2, the smallest element of P-{x0,x1}
And so on.
The sequence (xn) is strictly increasing. It follows that x → xn is injective, but it is also surjective because if y is an element any of P, there exists an index n0 such that xn0 >y, if y had no antecedent it would contradict the definition of xn0
