Définition

définition
On appelle 'suite' une de ℕ ou d'une de ℕ dans un ensemble quelconque E.
définition
On distingue les suites 'finies' dans le cas où l'ensemble source est une partie finie de ℕ, des suites 'infinies' où l'ensemble source est une partie de ℕ.
En pratique, dans la plupart des cas l'ensemble source sera:
  • Soit {0,1,....,n} (cas des suites finies).
  • Soit l'ensemble ℕ lui-même (cas des suites infinies).
Très exceptionnellement il pourra arriver que l'ensemble de départ soit {1,2,..n} ou bien ℕ* (pas d'image de zéro).

Notations

Elles sont un peu particulières. En général on utilise pour noter les fonctions des lettres comme f,g,h etc.. Pour les suites on utilisera plus volontiers u, v, w. Encore une fois il s'agit là uniquement d'une règle d'usage et pas du tout d'une prescription absolue !
Dans le cas général l'image de l'entier n par une fonction u se note u(n). Pour les suites on utilise souvent la notation indicée, c'est à dire que l'image de n par u sera notée un.
Les images des entiers par n s'appellent plus souvent les 'termes' de la suite.
La suite elle-même dans son intégralité pourra être notée (un)n∈ℕ au lieu de la notation classique u: ℕ → E, ou bien (uk)k∈{0,..,n}

Exemples

Nous prendrons ici E=ℕ pour fixer les idées. Les suites peuvent être définies de différentes manières:
  • Soit par une formule directe un=n2 par exemple la suite est donc {0,1,4,9,16,25,.....}
  • Soit par une simple: u0=1 et un=n×un-1 (définition possible de la suite factorielle) {1,1,2,6,24,120,720,.....}
  • Soit par une récurrence multiple comme la suite de Fibonacci :
    • un+1=un-1+un
    • u0=0
    • u1=1
    {0,1,1,2,3,5,8,13,21,.....}

Suites extraites

définition
Si (un ) désigne une suite d'éléments de E, on dit que (vn ) est une 'sous-suite' ou une suite 'extraite' ou encore une 'suite partielle ' de (un ) s'il existe une application p croissante et : ℕ → ℕ telle que:
vn=up(n) ∀n.
Cela pourrait être dit autrement v est la de u et p, v=u op où p est strictement croissante.

Exemple

si un=n2 vn=u2n =4n2 est une suite extraite de un {0,4,16,36, 64 ....}

Definition

definition
We call 'sequence' a of ℕ or a of ℕ into any set E.
definition
We distinguish between the 'finite' sequences in the case where the source set is a finite subset of ℕ, and the 'infinite' sequences where the source set is an subset of ℕ.
In practice, in most cases the source set will be:
  • Either {0,1,....,n} (case of finite sequences).
  • Or the set ℕ itself (case of infinite sequences).
Very exceptionally, it may happen that the departure set is {1,2,..n} or else ℕ* (no image of zero).

Notations

They are a bit special. In general we use to denote functions letters such as f, g, h etc. For the sequences we more likely use u, v, w. Again this is only a rule of use and not at all an absolute prescription !
In general the image of the integer n by a function u is denoted u(n). For the sequences we often use the indexed notation, that is to say that the image of n by u will be denoted by un.
The images of the integers by n are more often called the 'terms' of the sequence.
The sequence itself as a whole can be noted (un)n∈ℕ instead of the classical notation u: ℕ → E, or else (uk)k∈{0,..,n}

Note to English speaking readers
Very often you will find in textbooks notations such as {u}n=0∞, where parentheses are replaced by curly braces. Actually, because of notations in use for finite sets (roster form) it may let think that a sequence is something like an infinite set which is definitely not the case. An element can be repeated in a sequence and order is important. So a sequence is a map and the notation suggested here is an extension of the notation for finite n-tuples which are actually mappings.

Examples

We will take here E=ℕ to fix the ideas. The sequences can be defined in different ways:
  • Either by a direct formula un=n2 for example the sequence is therefore {0,1,4,9,16,25,.....}
  • Either by a simple : u0=1 and un=n×un-1 (possible definition of the factorial sequence) {1,1,2,6,24,120,720,.....}
  • Either by a multiple induction like the sequence of Fibonacci:
    • un+1=un-1+un
    • u0=0
    • u1= 1
    {0,1,1,2,3,5,8,13,21,.....}

Subsequences

definition
If (un) denotes a sequence of elements of E, we say that (vn ) is a 'subsequence' or a sequence 'extracted' or even a 'partial sequence' of (un) if there is a strictly increasing so map p: ℕ → ℕ such that:
vn=up(n) ∀n.
It could be said another way v is the of u and p, v=uop where p is strictly increasing.

Example

if un=n2 vn=u2n=4n2 is a sequence extracted from un {0,4,16,36, 64....}
