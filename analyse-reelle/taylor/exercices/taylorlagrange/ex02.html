Jusqu'à quel ordre faut-il prendre l'approximation de Taylor de sin(x) pour obtenir une erreur inférieure à 0.001 sur [0,+π] ?
Même question avec 10-5 ?

aide

Toutes les dérivées supérieures de la fonction sinus sont majorées par 1.

solution

A l'ordre n on obtient donc une majoration du reste par πn+1/(n+1)!
Il faut donc trouver les valeurs de n pours lequelles cette quantité est inférieure à 10-3 et 10-5.
Un petit programme Python donne rapidement 12 et 15. 

import math

def fact(n):
    r=1.0
    for i in range(1,n+1):
        r*=i
    return r

def f(n):
    return math.pi**n/fact(n)

for n in range(1,20):
    print n,f(n)

Up to which order should the Taylor approximation of sin(x) be taken to obtain an error less than 0.001 on [0,+π]?
Same question with 10-5?

hint

All derivatives of any order of the sine function are bounded by 1 in absolute value.

solution

At order n we therefore obtain an upper bound of the remainder by πn+1/(n+1)!
We must therefore find the values of n for which this quantity is less than 10-3 and 10-5.
A small Python program quickly gives 12 and 15.

def fact(n):
    r=1.0
    for i in range(1,n+1):
        r*=i
    return r

def f(n):
    return math.pi**n/fact(n)

for n in range(1,20):
    print n,f(n)
