Donner une majoration de l'erreur commise en calculant ln(3/2) par l'expression (1/2) -(1/2)2/2 + (1/2)3/3.....-(1/2)2n/2n.

aide

Utiliser une majoration du reste de Cauchy d'ordre 2n, pour la fonction f(x)=ln(1+x) entre les points 0 et 1/2.

solution

f étant la fonction x → ln(1+x) on a :

\( \displaystyle R(\frac{1}{2})=f^{(2n+1)}(c)\frac{1/2(1/2-c)^{2n}}{(2n)!} \)

Sachant que

\( \displaystyle f^{(2n+1)}(x)=\frac{(2n)!}{(1+x)^{2n+1}} \)

où c est un nombre dans l'intervalle ]0,1/2[
On obtient une majoration du type :

\( \displaystyle \left | R(1/2) \right |\leqslant \frac{(1/2)^{2n+1}}{(1+c)^{2n+1}}\leqslant \frac{1}{2^{2n+1}} \)

Give an upper bound for the error made in calculating ln(3/2) by the expression (1/2) -(1/2)2/2 + (1/2)3/3.....-(1/2)2n/2n.

hint

Use an upper bound of the Cauchy remainder of order 2n, for the function f(x)=ln(1+x) between points 0 and 1/2.

solution

We get an upper bound of the type :
