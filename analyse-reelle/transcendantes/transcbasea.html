Remarquons d'abord que la propriété essentielle du logarithme népérien, à savoir la relation fonctionnelle f(xy)=f(x)+f(y) pour x et y dans ]0,+∞[ n'est pas satisfaite par cette seule fonction.
En effet toute fonction du type x → k×ln(x) où k est une constante réelle satisfait la même relation.
Inversement supposons que f est une fonction dérivable sur ]0,+∞[ vérifiant f(xy)=f(x)+f(y).
Faisons tout d'abord quelques remarques sur une telle fonction:
On a nécessairement f(1)=0.
Si f est constante alors f est la fonction nulle.
On a par dérivation par rapport à x yf'(xy)=f'(x), donc f'(y)=f'(1)/y.
Ce qui donne deux possibilités:
  • soit f'(1)=0 auquel cas la fonction est constante et donc nulle
  • soit f'(1)≠0 dans ce cas f'(y)=k×ln(y) où k=f'(1)
On voit donc que les seules fonctions dérivables non nulles qui vérifient f(xy)=f(x)+f(y) sont les fonctions x → k×ln(x).

Logarithme de base a

Note first that the essential property of the natural logarithm, namely the functional relation f(xy)=f(x)+f(y) for x and y in ]0,+∞[ is not satisfied by this single function.
Indeed any function of type x → k×ln(x) where k is a real constant satisfies the same relation.
Conversely suppose that f is a differentiable function on ]0,+∞[ satisfying f(xy)=f(x)+f(y).
Let's first make some remarks about such a function:
We necessarily have f(1)=0.
If f is constant then f is the null function.
We have by derivation with respect to x yf'(xy)=f'(x), so f'(y)=f'(1)/y.
This gives two possibilities:
  • let f'(1)=0 in which case the function is constant and therefore zero
  • let f'(1)≠0 in this case f'(y)=k×ln(y) where k=f'(1)
We can therefore see that the only nonzero differentiable functions which verify f(xy)=f(x)+f(y) are the functions x → k×ln(x).

Logarithm base a

définition 1
Nous posons pour tout nombre réel a > 0, a ≠ 1 et tout x > 0
loga(x)=ln(x)/ln(a).
Nous définissons ainsi une fonction loga de ℝ+ dans ℝ, appelée ' logarithme de base a '.
definition 1
We set for any real number a > 0, a ≠ 1 and all x > 0
loga(x)=ln(x)/ln(a).
We thus define a function loga from ℝ+ onto ℝ, called ' logarithm base a '.
propriété 1
Ainsi ces fonctions sont continues et dérivables sur ]0,+∞[ et de dérivée 1/(x.ln(a)). Elles sont donc croissantes pour a > 1 et décroissantes pour a ∈ ]0,1[.
property 1
Thus these functions are continuous and differentiable on ]0,+∞[ and with derivative 1/(x.ln(a)). They are therefore increasing for a > 1 and decreasing for a ∈ ]0.1[.
Remarquons que pour a entier la partie entière de loga(x) est égale au nombre de chiffres de la partie entière de x écrite en base a moins une unité.
log10(10)=1 log10(100)=2, etc...
Pour des raisons historiques (et pratiques) la base 10 était autrefois largement utilisée dans les tables numériques. Depuis l'apparition des calculettes et des ordinateurs, cette utilisation tombe en désuétude.
Par contre les logarithmes de base 2 restent largement utilisés dans la théorie de l'information (une branche du calcul des probabilités).

Note that when a is an integer the integer part of loga(x) is equal to the number of digits of the integer part of x written in base a minus one unit.
log10(10)=1 log10(100)=2, etc...
For historical (and practical) reasons base 10 was once widely used in number tables. Since the appearance of calculators and computers, this use has fallen into disuse.
On the other hand, base 2 logarithms remain widely used in information theory (a branch of probability calculus).

Exponentielle de base a

Exponential base a

définition 2
La fonction 'exponentielle de base a' est la réciproque du logarithme de base a. On la noté x → ax.
definition 2
The function 'exponential to base a' is the inverse of the logarithm base a. It is denoted by x → ax.
propriété 2
Nous avons donc ax=exln(a).
property 2
So we have ax=exln(a).
propriété 3
Les fonctions x → ax sont donc continues et dérivables sur ℝ. Leur sens de variation est le même que celui de loga.
property 3
The functions x → ax are thus continuous and differentiable on ℝ. Their direction of variation is the same as that of loga.
propriété 4
La dérivée de x → ax est x → ln(a)ax.
property 4
The derivative of x → ax is x → ln(a)ax.

A l'attention des débutants en programmation

On peut donner un sens à toute expression du genre yx où y est un nombre strictement positif.
Ceci doit être compris comme exln(y).
Le calcul de cette quantité nécessite donc l'évaluation d'un logarithme et d'une exponentielle.
La plupart des langages informatiques, y compris le langage Python, proposent des fonctions de la biliothèque de math pour le calcul de yx.
Ces fonctions sont généralement notées 'pow' (en anglais power=puissance) et on peut les obtenir par surcharge d'opérateurs comme ^ (caret) ou bien **.
Souvent les débutants, pour élever une variable X au carré, utilisent l'expression pow(X,2) ou X**2 ou X^2, au lieu de X*X. Ce faisant, ils remplacent une simple multiplication par un appel de fonction logarithme et un appel de fonction exponentielle.

For beginners in programming

Meaning can be given to any expression like yx where y is a strictly positive number.
This should be understood as exln(y).
Calculating this quantity therefore requires the evaluation of a logarithm and an exponential.
Most computer languages, including Python, provide math library functions for calculating yx.
These functions are generally denoted 'pow' and they can be obtained by overloading operators such as ^ (caret) or **.
Often beginners, to square a variable X, use the expression pow(X,2) or X**2 or X^2, instead of X*X. In doing so, they replace a simple multiplication with a logarithm function call and an exponential function call.
