Démontrer que pour tout x on a :

\( \displaystyle e^{x}=\lim_{n \to \infty}\left ( 1+\frac{x}{n} \right )^{n} \)

aide

Pour n suffisament grand (disons 1+x/n > 0) on a :

\( \displaystyle \left ( 1+\frac{x}{n} \right )^{n}=e^{n\times ln\left ( 1+\frac{x}{n} \right )} \)

solution

L'approximation du premier ordre de la fonction x → ln(1+x/n) nous donne :

\( \displaystyle ln\left ( 1+\frac{x}{n} \right )=\frac{x}{n}+o\left ( \frac{1}{n} \right ) \)

Donc, compte tenu de l'indication fournie dans l'aide :

\( \displaystyle \left ( 1+\frac{x}{n} \right )^{n}=e^{n\left ( \frac{x}{n}+o\left ( \frac{1}{n} \right ) \right )}=e^{x}e^{o(1)} \)

De sorte que le résultat découle de la continuité de l'exponentielle à l'origine.

\( \displaystyle \left ( 1+\frac{x}{n} \right )^{n}=e^{n\times ln\left ( 1+\frac{x}{n} \right )} \)

The first order approximation of function x → ln(1+x/n) gives us:

\( \displaystyle ln\left ( 1+\frac{x}{n} \right )=\frac{x}{n}+o\left ( \frac{1}{n} \right ) \)

Hence, using the indication given in the help:

\( \displaystyle \left ( 1+\frac{x}{n} \right )^{n}=e^{n\left ( \frac{x}{n}+o\left ( \frac{1}{n} \right ) \right )}=e^{x}e^{o(1)} \)
