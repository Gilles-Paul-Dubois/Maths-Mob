Solve the following system in ℝ×ℝ :

\( \displaystyle \left\{\begin{matrix} 8^{x}=10y \\ 2^{x}=5y \end{matrix}\right. \)

hint

Set X=2x

solution

With the suggested substitution the system can be written :

\( \displaystyle \left\{\begin{matrix} X^{3}=10y \\ X=5y \end{matrix}\right. \)

Whence the necessary condition X2=2
Knowing that X is positive, the only possible solution is l x=1/2 and y=√2/5
