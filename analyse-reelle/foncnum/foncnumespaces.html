Furthermore ℝD is finite dimensional only if D is itself a finite set.
En effet si D={x1, ..., xn} ℝD s'identifie à ℝn. Mais si D contient un ensemble infini {xi}i∈I, pour toute partie finie J de I les fonctions fi avec i∈J sont linéairement indépendantes. Ce qui implique que ℝD contient des systèmes libres ayant un nombre fini n d'éléments quel que soit l'entier n.
Indeed if D={x1, ..., xn} ℝD identifies with ℝn . But if D contains an infinite set {xi}i∈I, for any finite subset J of I the functions fi with i∈J are linearly independent. This implies that ℝD contains free systems having a finite number n of elements whatever the integer n.
