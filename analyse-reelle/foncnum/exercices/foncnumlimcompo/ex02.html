Montrer que si g vérifie:
limy→0+ y≠0g(y)=a (a réel fini)
alors
limx→+∞g(1/x)=a

aide

Posant f(x)=1/x et en appliquant le théorème sur les limites des quotients, on voit que:
limx→∞f(x)=0+
Ce qui fait que la proposition est bien un cas particulier du théorème général sur la limite des composées.
On peut donc soit appliquer ce théorème (en le supposant démontré dans tous les cas), soit démontrer ce théorème dans le cas particulier où:
f ne s'annule pas.
limx→+∞f(x)=0+
et
limy→0+,x≠0g(y)=a

solution

Démontrons donc le théorème dans le cas particulier où:
limx→+∞f(x)=0+
et
limy→0+ y≠0g(y)=a
Soit ε > 0.
Il existe un nombre η > 0 tel que:
y ∈]0,η[ ⇒ |g(y)-a| < ε
Il existe donc un réel X tel que x > X ⇒ f(x)∈]0,η[
Mais alors si x > X on a g(f(x)) ∈ ]a-ε,a+ε[

Show that if g verifies:
limy→0+ y≠0g(y)=a (a finite real)
then
limx→+∞g(1/x)=a

hint

Setting f(x)=1/x and applying the theorem on the limits of quotients, we see that:
limx→∞f(x)=0+
This makes the proposition a special case of the general theorem on the limit of composites.
We can therefore either apply this theorem (assuming it is proven in all cases), or prove this theorem in the particular case where:
f does not cancel.
limx→+∞f(x)=0+
and
limy→0+,x≠0g(y)=a

solution

So let's prove the theorem in the particular case where:
limx→+∞f(x)=0+
and
limy→0+ y≠0g(y)=a
Let ε > 0.
There is an η > 0 such as:
y ∈]0,η[ ⇒ |g(y)-a| < ε
So there exists a real X such that x > X ⇒ f(x)∈]0,η[
But then if x > X we have g(f(x)) ∈ ]a-ε,a+ε[
