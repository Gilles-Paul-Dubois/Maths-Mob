Montrer que si f est injective et que gof est définie:
Δgof(x0,x1)=Δg(f(x0),f(x(1))×Δf(x0,x1).
En déduire les résultats suivants:
Si f est injective et strictement croissante et g est strictement croissante alors gof est strictement croissante.
Si f est injective et strictement décroissante et g est strictement décroissante alors gof est strictement croissante.
Si f est injective et strictement croissante et g est strictement décroissante alors gof est strictement décroissante.
Si f est injective et strictement décroissante et g est strictement croissante alors gof est strictement décroissante.

aide

Utiliser la définition du taux d'accroissement ainsi que les résultats du cours.

solution

\( \displaystyle \frac{g\left ( f\left ( x_{1} \right ) \right )-g\left ( f\left ( x_{0} \right ) \right )}{x_{1}-x_{0}} = \frac{g\left ( f\left ( x_{1} \right ) \right )-g\left ( f\left ( x_{0} \right ) \right )}{f\left ( x_{1} \right )-f\left ( x_{0} \right )} \times \frac{f\left ( x_{1} \right )-f\left ( x_{0} \right )}{x_{1}-x_{0}} \)

d'où le résultat. Il suffit ensuite d'appliquer la règle des signes.

Show that if f is injective and gof is defined:
Δgof(x0,x1)=Δg(f(x 0),f(x(1))×Δf(x0,x1).
Deduce the following results:
If f is injective and strictly increasing and g is strictly increasing then gof is strictly increasing.
If f is injective and strictly decreasing and g is strictly decreasing then gof is strictly increasing.
If f is injective and strictly increasing and g is strictly decreasing then gof is strictly decreasing.
If f is injective and strictly decreasing and g is strictly increasing then gof is strictly decreasing.

hint

Use the definition of the average rate of change as well as the results of the course.

solution

\( \displaystyle \frac{g\left ( f\left ( x_{1} \right ) \right )-g\left ( f\left ( x_{0} \right ) \right )}{x_{1}-x_{0}} = \frac{g\left ( f\left ( x_{1} \right ) \right )-g\left ( f\left ( x_{0} \right ) \right )}{f\left ( x_{1} \right )-f\left ( x_{0} \right )} \times \frac{f\left ( x_{1} \right )-f\left ( x_{0} \right )}{x_{1}-x_{0}} \)

Whence the result. After this just apply rule of signs.
