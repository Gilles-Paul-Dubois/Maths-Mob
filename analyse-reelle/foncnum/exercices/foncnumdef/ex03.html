Ecrire un programme Python et/ou Julia en s'inspirant du modèle de représentation graphique avec le module Tkinter (pour Python). Cependant le programme devra être interactif. Prévoir une barre d'outils en dessous de la zone de traçage avec 4 boutons:

  1. Un bouton 'Quitter' pour mettre fin à l'application
  2. Un bouton 'Effacer' pour effacer tous les graphes
  3. Un bouton 'Graphe 1'pour afficher le graphe de la première fonction
  4. Un bouton 'Graphe 2' pour afficher le graphe de la seconde fonction

aide

Consulter les nombreux sites dédiés à la documentation sur les widgets de Tkinter, la fonction pack, etc.. etc..

Pour Julia on pourra consulter la documentation des packages 'Gtk' et 'Cairo'.

Write a Python and/or Julia program inspired by the graphical representation model with the Tkinter module (for Python). However, the program should be interactive. Provide a toolbar below the plotting area with 4 buttons:

  1. A 'Quit' button to terminate the application
  2. A 'Clear' button to clear all graphs
  3. A 'Graph 1' button to display the graph of the first function
  4. A 'Graph 2' button to display the graph of the second function

hint

For Julia you can consult the documentation of the 'Gtk' and 'Cairo' packages.
