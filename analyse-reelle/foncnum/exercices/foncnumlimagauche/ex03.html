Soit f une fonction numérique de domaine D et x0 un point adhérent à D.

  1. Montrer que limx→x0+ f(x)=a ⇒ limn→∞un=a avec un=x0+1/n pour n entier n ≥ 1.
  2. Enoncer une propriété analogue si limx→x0- f(x)=a.
  3. La réciproque est-elle vraie ?

aide

Revoir la définition des limites des suites et des limites à droite des fonctions.
Remarquer que pour n ≥ 1 x0+1/n > x0.

solution

  1. Donnons nous ε > 0 et soit η tel que x ∈[x0,x0+η[ ⇒ |f(x)-a| < ε
    Soit N un entier tel que 1/N < η
    alors si n > N on a x0+1/n ∈ [x0,x0+η[ donc |un-a| < ε
  2. limx→x0- f(x)=a ⇒ limn→∞un=a avec un=x0-1/n pour n entier n ≥ 1.
  3. Soit f la fonction définie par f(x)=x si x∈ℚ et f(x)=1 si x∉ ℚ.
    Si on pose un=f(1/n)=1/n on a bien limn→∞ un=0.
    Cependant il est clair qu'il n'existe aucune limite à droite pour f en 0 compte tenu de sa définition.

Let f be a numerical function of domain D and x0 a cluster point for f.

  1. Show that limx→x0+ f(x)=a ⇒ limn→∞un=a with un=x0+1/n for n integer n ≥ 1.
  2. State an analogous property if limx→x0- f(x)=a.
  3. Is the converse true?

hint

Review the definition of limits of sequences and right limits of functions.
Note that for n ≥ 1 x0+1/n > x0.

solution

  1. Let ε > 0 be given and let η such that x ∈[x0,x0+η[ ⇒ |f(x)-a|<ε
    Let N be an integer such that 1/N < η
    then if n > N we have x0+1/n ∈ [x0,x0+η[ so |un-a| < ε
  2. limx→x0- f(x)=a ⇒ limn→∞un=a with un=x0-1/n for n integer n ≥ 1.
  3. Let f be the function defined by f(x)=x if x∈ℚ and f(x)=1 if x ∉ ℚ.
    If we set un=f(1/n)=1/n we have limn→∞ un=0.
    However it is clear that there is no right limit for f at 0 given its definition.
