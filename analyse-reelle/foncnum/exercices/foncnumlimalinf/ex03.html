Soit f une fonction croissante.
Montrer qu'il y a équivalence entre:
limx→+∞f(x)=+∞
et limn→+∞un=+∞
où la suite (un)n∈ℕ est définie par un=f(n) ∀n∈ℕ.

aide

Utiliser f(n) ≤ f(x) ≤ f(n+1) ∀x∈[n,n+1]

solution

Supposons d'abord que la suite tend vers +∞.
Soit M un nombre quelconque. Il existe un entier N tel que:
n > N ⇒ un > M.
Mais alors si x > N on a f(x) ≥ un, donc f(x) > M
Il suffit donc de prendre X(M)=N(M).
Supposons maintenant que limx→+∞f(x)=+∞
Soit M un nombre quelconque.
Il existe X(M) tel que x > X(M) ⇒ f(x) > M.
Soit N(M) un entier quelconque vérifiant N(M) > X(M)
Alors si n > N(M) on a forcément:
un > M.

Let f be an increasing function.
Show that there is equivalence between:
limx→+∞f(x)=+∞
and limn→+∞un=+∞
where the sequence (un)n∈ℕ is defined by un=f(n) ∀n∈ℕ.

hint

Use f(n) ≤ f(x) ≤ f(n+1) ∀x∈[n,n+1]

solution

Let's first assume that the sequence tends to +∞.
Let M be any number. There is an integer N such that:
n > N ⇒ un > M.
But then if x > N we have f(x) ≥ un, then f(x) > M
So just take X(M)=N(M).
Now suppose that limx→+∞f(x)=+∞
Let M be any number.
There exists X(M) such that x > X(M) ⇒ f(x) > M.
Let N(M) be any integer satisfying N(M) > X(M)
So if n > N(M) we inevitably have:
un > M.
