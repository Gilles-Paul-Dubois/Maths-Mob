Soit Ep l'ensemble des des applications numériques de ℝ dans ℝ admettant pour période.

  1. Montrer que Ep est un de ℝℝ.
  2. Que peut-on dire de la somme d'un élément de Ep avec un élément de Eq du point de vue de la périodicité ?

aide

Calculer (λf+μg)(x+p).

solution

  1. Ep n'est pas vide puisqu'il contient toutes les constantes.
    (λf+μg)(x+p)=(λf)(x+p)+ (μg)(x+p)=λf(x+p)+ μg(x+p)=λf(x)+ μg(x)=(λf+μg)(x).
    Ep est donc bien un sous-espace vectoriel de ℝℝ.
  2. En général on ne peut rien dire, sauf si q est un multiple entier de p: q=kp, dans ce cas Ep⊆Eq.

Let Ep be the set of numerical maps of ℝ in ℝ admitting for period.

  1. Show that Ep is a from ℝℝ.
  2. What can be said about the sum of an element of Ep with an element of Eq from the point of view of periodicity?

hint

Compute (λf+μg)(x+p).

solution

  1. Ep is not empty since it contains all the constants.
    (λf+μg)(x+p)=(λf)(x+p)+ (μg)(x+p)=λf(x+p)+ μg( x+p)=λf(x)+ μg(x)=(λf+μg)(x).
    Ep is therefore a vector subspace of ℝℝ.
  2. In general nothing can be said, except if q is an integer multiple of p: q=kp, in this case Ep⊆Eq.
