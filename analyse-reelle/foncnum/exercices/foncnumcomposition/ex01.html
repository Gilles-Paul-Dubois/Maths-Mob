On dit qu'une fonction numérique est une 'involution' si elle est égale à sa propre réciproque.
Quelle propriété doit posséder la représentation graphique de f pour que f soit une involution ?
Montrer que la composée de deux involutions est une involution si ces deux fonctions commutent.
Donner des exemples de telles fonctions.

aide

Voir les exemples de fonctions réciproques donnés en cours.

solution

Le graphe doit être symétrique par rapport à la première diagonale (droite y=x).
Si f=f-1 et g=g-1 alors (gof)-1=f-1og-1=fog=gof si elles commutent.
Voici quelques exemples:

  • x → x (fonction identique)
  • x → -x
  • x → 1/x
  • x → -1/x

A numerical function is said to be an 'involution' if it is equal to its own inverse.
What property must the graph of f have for f to be an involution?
Show that the composite of two involutions is an involution if these two functions commute.
Give examples of such functions.

hint

See examples of inverse functions given in the course.

solution

The graph must be symmetrical (reflection) with respect to the first diagonal ( mirror :line y=x).
If f=f-1 and g=g-1 then (gof)-1 =f-1og-1=fog=gof if they switch.
Here are some examples:

  • x → x (identity function)
  • x → -x
  • x → 1/x
  • x → -1/x
