Soient (f,D) et (g,D') deux fonctions numériques, telles que f(D)⊆D' de sorte que gof est définie sur D.

  1. Si g est majorée (resp. minorée, bornée), montrer qu'alors gof est également majorée (resp. minorée, bornée).
  2. Montrer que gof peut être majorée sans que g le soit.
  3. Montrer que gof peut être majorée sans que f le soit.
  4. Montrer que gof peut être non majorée alors que f l'est.

aide

Pour les questions autres que la première trouver des contre-exemples.

solution

  1. Comme f(D)⊆D' si g(x) ≤M pour x∈D' on a à plus forte raison g(f(x))≤M pour x∈D
  2. g(x)=x, f(x)=sin(x)
  3. g(x)=sin(x), f(x)=x
  4. g(x)=1/x D'=]0,+∞[ f(x)=x D=]0,1[

Let (f,D) and (g,D') be two numerical functions, such that f(D)⊆D' such that gof is defined on D.

  1. If g is upper-bounded (resp. lower-bounded, bounded), show that then gof is also upper-bounded (resp. lower-bounded, bounded).
  2. Show that gof can be upper-bounded without g being upper-bounded .
  3. Show that gof can be upper-bounded up without f being itself upper-bounded.
  4. Show that gof can be not upper-bounded while f is.

hint

For questions other than the first find counterexamples.

solution

  1. Since f(D)⊆D' if g(x) ≤M for x∈D' we have a fortiori g(f(x))≤M for x∈D
  2. g(x)=x, f(x)=sin(x)
  3. g(x)=sin(x), f(x)=x
  4. g(x)=1/x D'=]0,+∞[ f(x)=x D=]0,1[
