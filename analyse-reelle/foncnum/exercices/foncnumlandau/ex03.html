Montrer que pour x→0+
1-sin(x)/x est O(x2).

aide

Utiliser conjointement avec :
cos(2x)=cos²(x)-sin²(x) appliquée à x/2

solution

Partons de :
cos(x) ≤ sin(x)/x ≤ 1/cos(x)
Nous obtenons: 1-cos(x) ≥ 1-sin(x)/x ≥ (cos(x)-1)/cos(x).
Comme limx→0cos(x)=1, il suffit de montrer que 1-cos(x) est O(x²) pour x→0.
Mais 1-cos(x)=1-(cos²(x/2)-sin²(x/2))=1-(1-2sin²(x/2))=2sin²(x/2).
il suffit maintenant d'utiliser le fait que limx→0sin(x/2)/(x/2)=1.
