  • x→+∞
  • x→-∞
  • x→x0 ( x0 real number)
  • x→x0, x≠x0 (same)
  • x→x0+ (x tends to x0 and x≥x0)
  • x→x0-(x tends to x0 and x≤x0)
  • x→x0+, x≠x0 (x tends to x0 and x>x0 )
  • x→x0-, x≠x0 (x tends to x0 and x 0 )
We will see that the results are essentially the same with the same indeterminacy cases , so to avoid the multiplication of equivalent statements we will use the notation x→ ... knowing that ... can cover any of the following cases. However when we study a sum, a product, a quotient, it will always be understood that the x→... will designate the same situation in both cases , and that if x tends to a finite value it will be the same in both cases.
We fully keep the study plan retained for the sequences.

L'indétermination majeure correspond au cas du produit d'une fonction qui tend vers zéro par une autre qui tend vers un infini. On peut la lever en utilisant des techniques déjà vues pour les suites ou bien d'autres empruntées au calcul différentiel.

The two functions tend to a finite limit

Cases of Indeterminacy

The major indeterminacy corresponds to the case of the product of a function which tends towards zero by another which tends towards an infinity. It can be solved by using techniques already seen for sequences or others borrowed from differential calculus.
