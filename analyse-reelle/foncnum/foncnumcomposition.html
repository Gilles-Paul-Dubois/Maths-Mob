Composition

In the previous page we considered operations (sum, product) on sets of numerical functions all having the same domain.
We now consider a couple of functions ((f,D),(g,D')) satisfying the following property:
f(D)⊆D'
so that for all x of D f(x)∈D' and g(f(x)) exists.
This amounts to saying that the destination of f is included in the departure of g. ( ).
définition 1
On peut dans ce cas définir la de f et g, soit gof.
gof est donc définie sur D par:
gof(x)=g(f(x)) ∀x∈D.
definition 1
In this case, we can define the of f and g, i.e. gof.
gof is therefore defined on D by:
gof(x)=g(f(x)) ∀x∈D.
Ainsi si D=D'=ℝ et si:
f:x → x2+1
g:x → sin(x+3)
On a:
gof(x)=sin(x2+4)
fog(x)=sin(x+3)2+1
On voit donc qu'en général: gof≠fog.
D'ailleurs, très souvent gof est défini alors que fog ne l'est pas ou réciproquement.

Réciproque

On considère maintenant la situation où f:D → ℝ est .
Dans ce cas si on pose D'=f(D) f induit une de D sur D'.
Il existe alors une fonction numérique g:D' → ℝ symétrique de f pour la composition, c'est à dire la bijection réciproque de f qu'on note généralement f-1.
Par définition f-1 vérifie:
f-1(f(x))=x ∀ x ∈ D
f(f-1(x))=x ∀ x ∈ D'

Inverse functions

We now consider the situation where f:D → ℝ is .
In this case if we set D'=f(D) f induces a from D to D'.
Then, there exists a numerical function g:D' → ℝ symmetric to f for the composition, ie the inverse bijection of f which is generally denoted f-1.
By definition f-1verifies:
f-1(f(x))=x ∀ x ∈ D
f(f-1(x))=x ∀ x ∈ D'
