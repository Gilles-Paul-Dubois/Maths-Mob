Definition

définition 1
Soit f une fonction numérique définie sur un domaine D. On dit que F est une 'primitive' de f sur D si:
  • F est dérivable sur D
  • F'(x)=f(x) ∀ x ∈ D
definition 1
Let f be a numerical function defined on a domain D. We say that F is a 'anti-derivative or inverse derivative or primitive function or primitive integral or indefinite integral' of f on D if:
  • F is differentiable on D
  • F'(x)=f(x) ∀ x ∈ D
NB: Au lieu de 'primitive de f' on dira aussi souvent 'intégrale' ou 'intégrale indéfinie' de f sur D.

Cherchons par exemple une primitive de x2.
Nous savons que x3 se dérive en 3x2, compte tenu de la il suffit donc de faire précéder le terme x3 d'un facteur 1/3. Nous avons donc une primitive de x2 soit (1/3)x3.
Cherchons une primitive de f(x)=sin(x)+cos(x).
Nous savons qu'une primitive de cos(x) est sin(x). En outre cos(x) se dérive en -sin(x) donc -cos(x) se dérive en sin(x).
D'où notre proposition : sin(x)-cos(x) comme primitive de f(x).

Case where the domain is an interval

In general, if F is a primitive function of f on D, and if K is a constant, F+K is another. So any function admitting a primitive function admits an infinity of them.
Now consider the zero function f defined on ]0,1[ ∪ ]2,3[. this function has, among others, as primitive functions on D:
  • Itself
  • The function constant and equal to 1 on ]0.1[ ∪ ]2.3[
  • The function equal to 1 over ]0.1[ and equal to 2 over ]2.3[
We therefore see that we have primitive functions of the same function which do not differ from a constant, but this is linked to the particular form of the domain which is presented as the union of two disjoint intervals. In fact:
théorème 1
Si D est un intervalle alors deux primitives de f sur D (à supposer qu'elles existent) diffèrent d'une constante.
theorem 1
If D is an interval then two primitive functions of f on D (assuming they exist) differ by a constant.

Approximation locale

Donnons nous une fonction pour laquelle nous ne savons pas a priori déterminer une primitive.
Disons par exemple f(x)= √(1+sin2(x)).
Faute de trouver une formule 'exacte' pour une primitive, nous cherchons une fonction F dont la dérivée soit aussi voisine que possible de f(x) et satisfaisant à une 'condition initiale' comme par exemple F(0)=0.
Si une telle fonction existe on sait qu'au voisinage de 0.
F(x)=F(0)+f(0)x +o(x)
Soit ε un réel >0 donné on peut donc prendre comme approximation de F(ε) la valeur F(0)+f(0)ε
Puis comme approximation de F(2ε) la valeur F(ε)+f(ε)ε
Et on a par récurrence:

Local approximation

Give us a function for which we do not know how to determine a primitive function.
Say for example f(x)= √(1+sin2(x)).
Failing to find an 'exact' formula for a primitive function, we are looking for a function F whose derivative is as close as possible to f(x) and satisfying an 'initial condition' such as for example F(0)=0.
If such a function exists, we know that in the neighborhood of 0.
F(x)=F(0)+f(0)x +o(x)
Let ε be a given real >0, so we can take as an approximation of F(ε) the value F(0)+f(0)ε
Then as an approximation of F(2ε) the value F(ε)+f(ε)ε
And we have by recurrence:
