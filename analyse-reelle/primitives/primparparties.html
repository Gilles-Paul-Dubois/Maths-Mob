The formula known as "integration by parts for indefinite integrals" can be written

La formule d'intégration par parties est efficace quand on l'utilise à répétition pour chercher une primitive d'une fonction de type p(x)g(x) où p(x) est une fonction polynomiale et g(x) une fonction dont les dérivées (ou les primitives sont périodiques), par exemple comme les fonctions trigonométriques. En effet, une application répétée de cette formule fera apparaître les dérivées successives du polynôme, or on sait qu'on parviendra tôt ou tard à une dérivée nulle.

The integration by parts formula is efficient when used repeatedly to find an antiderivative of a function of type p(x)g(x) where p(x) is a polynomial function and g(x) a function the derivatives (or primitive functions) of which are periodic, for example like trigonometric functions. Indeed, a repeated application of this formula will reveal the successive derivatives of the polynomial, but we know that sooner or later we will reach a null derivative.
For example, if we want to calculate the antiderivative of x2cos(x), we will have to use it twice:

\( \displaystyle \int x^{2}cos(x)dx=x^{2}sin(x)-\int 2xsin(x)dx \)

\( \displaystyle =x^{2}sin(x)-\left ( -2xcos(x) +\int 2cos(x)dx \right ) \)

\( \displaystyle =\left ( x^{2}-2 \right )sin(x)+2xcos(x) \)
