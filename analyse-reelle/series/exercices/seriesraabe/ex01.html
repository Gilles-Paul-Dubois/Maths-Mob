On considère la série de terme général:
un=nrexp(-k(1+1/2+ ...+1/n))
exp désigne ici la fonction exponentielle x→ex
Montrer que si r<k-1 la série converge absolument.

aide

On utilisera la formule: exp(k/n)=1+k/n+O(1/n²) pour n→∞

solution

\( \displaystyle \frac{u_{n+1}}{u_{n}}=\frac{(n+1)^{r}}{n^{r}}exp\left ( \frac{-k}{n+1} \right ) \)

\( \displaystyle \frac{u_{n+1}}{u_{n}}=\left ( 1+\frac{1}{n} \right )^{r}exp\left ( \frac{-k}{n+1} \right ) \)

Compte tenu de l'aide et du résultat préliminaire du cours:

\( \displaystyle \frac{u_{n+1}}{u_{n}}=\left ( 1+\frac{r}{n}+O\left ( \frac{1}{n^{2}} \right ) \right )\left ( 1-\frac{k}{n+1}+O\left ( \frac{1}{n^{2}} \right ) \right ) \)

\( \displaystyle \frac{u_{n+1}}{u_{n}}=1+\frac{r-k}{n}+\frac{k}{n(n+1)}+O\left ( \frac{1}{n^{2}} \right )+o\left ( \frac{1}{n^{2}} \right ) \)

Mais étant donné que:
O(1/n²)+o(1/n²)=O(1/n²)
et que:
k/(n(n+1))=O(1/n²)
Il vient:

\( \displaystyle \frac{u_{n+1}}{u_{n}}=1+\frac{r-k}{n}+O\left ( \frac{1}{n^{2}} \right ) \)

Et comme d'après l'hypothèse r-k<-1, il suffit d'appliquer le test de Raabe-Duhamel.

We consider the series of general term:
un=nrexp(-k(1+1/2+ ...+1/n))
exp denotes here the exponential function x→ex
Show that if r< k-1 the series converges absolutely.

hint

We will use the formula: exp(k/n)=1+k/n+O(1/n²) for n→∞

solution

\( \displaystyle \frac{u_{n+1}}{u_{n}}=\frac{(n+1)^{r}}{n^{r}}exp\left ( \frac{-k}{ n+1} \right ) \)

\( \displaystyle \frac{u_{n+1}}{u_{n}}=\left ( 1+\frac{1}{n} \right )^{r}exp\left ( \frac{-k}{ n+1} \right ) \)

Considering the help and the preliminary result of the course:

\( \displaystyle \frac{u_{n+1}}{u_{n}}=\left ( 1+\frac{r}{n}+O\left ( \frac{1}{n^{2}} \right ) \right )\left ( 1-\frac{k}{n+1}+O\left ( \frac{1}{n^{2}} \right ) \right ) \)

\( \displaystyle \frac{u_{n+1}}{u_{n}}=1+\frac{rk}{n}+\frac{k}{n(n+1)}+O\left ( \frac {1}{n^{2}} \right )+o\left ( \frac{1}{n^{2}} \right ) \)

But given that:
O(1/n²)+o(1/n²)=O(1/n²)
and that:
k/(n(n+1))=O(1/n²)
We get:

\( \displaystyle \frac{u_{n+1}}{u_{n}}=1+\frac{r-k}{n}+O\left ( \frac{1}{n^{2}} \right ) \)

And as according to the hypothesis r-k<-1, it suffices to apply the Raabe-Duhamel test.
