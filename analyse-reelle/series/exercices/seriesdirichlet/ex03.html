  1. Montrer que la série de terme général (-1)n/(2n+1) est convergente.
  2. Calculer la limite du rapport |un+1/un|
  3. Qu'en conclure pour la vitesse de convergence de la série?
On admettra que cette série converge vers π/4.
  1. Jusqu'à quelle ordre faut-il calculer les sommes partielles pour être sûr d'avoir la valeur π/4 avec 3 décimales exactes
  2. Cette série vous paraît-elle utilisable pour le calcul de π?

aide

Appliquer le théorème sur les séries alternées.

solution

  1. La série est une série alternée. Le terme général tend vers 0. Donc la série est convergente.
  2. Le rapport |un+1/un| tend vers 1
  3. La série est à convergence lente
  1. A l'ordre n l'erreur est majorée par 1/(2n+1).
    Pour avoir 3 décimales exactes il faut donc prendre n assez grand pour que (2n+1)>1000 soit n≥500.
    Valeur trouvée pour n=500: 0.785897164896
    π/4=0.785398163397
  2. La convergence est trop lente.

  1. Up to what order should the partial sums be calculated to be sure of having the value π/4 with 3 exact decimal places
  2. Do you think this series can be used to actually calculate π?

hint

Apply the theorem on alternating series.

solution

  1. The series is an alternate series. The general term tends to 0. So the series is convergent.
  2. The ratio |un+1/un| tends to 1
  3. The series is slowly convergent
  1. At order n the error is increased by 1/(2n+1).
    To have 3 exact decimals, we must therefore take n large enough so that (2n+1) >1000 is n≥500.
    Value found for n=500: 0.785897164896
    π/4=0.785398163397
  2. Convergence is too slow.
