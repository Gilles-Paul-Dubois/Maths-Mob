Calculer sn pour la série de terme général:
un=sin(nx) pour n≥1
C'est à dire:
sn=u1+u2+ ... +un

aide

Revoir la .
un=Im(eix)

solution

si x=2kπ avec k entier sn=0 sinon
sn=Im(eix+e2ix+ ... +enix)
sn=Im(eix(1+eix+ .. ei(n-1)x)
sn=Im(eix(1-einx)/(1-eix)
Posons maintenant α=eix/2, de sorte que α2=eix.

\( \displaystyle s_{n}=Im\left ( \alpha ^{2}\frac{\alpha^{n}\left ( \alpha ^{n}-\alpha ^{n-1} \right ) }{\alpha\left ( \alpha -\alpha ^{-1} \right ) } \right ) \)

D'où la formule finale :

\( \displaystyle s_{n}=\frac{sin\left ( \frac{(n+1)x}{2} \right )sin\frac{nx}{2}}{sin\frac{x}{2}} \)
