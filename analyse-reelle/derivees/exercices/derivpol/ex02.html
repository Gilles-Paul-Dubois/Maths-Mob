On dit qu'un polynôme à ceofficients réels est 'scindé' si toutes ses racines sont réelles, autrement dit s'il se factorise en un produit de polynômes du premier degré à coefficients réels.
Montrer que si p(x) est scindé, sa dérivée est également scindée.

aide

Utiliser le résultat de l'exercice 01 de cette série, ainsi que le .

solution

On suppose p de degré n+1.
Supposons d'abord que toutes les racines de p sont simples soient x0,x1, ..., xn ces racines.
D'après le théorème de Rolle p' possède une annulation dans l'intervalle ]x0,x1[, une annulation dans l'intervalle ]x1,x2[,...., une annulation dans l'intervalle ]xn-1,xn[.
Donc p' possède n racines distinctes et est scindé.
Supposons maintenant que x0 soit racine double de p, disons que x1=x0, toutes les autres étant simples. Alors x0 est racine simple de p' en outre p' possède une annulation dans l'intervalle ]x0,x2[, une annulation dans l'intervalle ]x2,x3[,...., une annulation dans l'intervalle ]xn-1,xn[. Donc p' est scindé dans ce cas encore.
Le même raisonnement vaut si x0 est racine d'ordre k de p, c'est à dire si x0=x1=x2=xk-1. Dans ce cas x0 sera racine d'ordre k-1 de p' et p' possédera une annulation dans ]x0,xk[, une dans ]xk,xk+1[, etc.
Enfin le même raisonnement vaut si c'est x1 ou toute autre racine qui est de multiplicité k, les autres étant simples.
Maintenant si x0 est de multiplité k et x1 de multiplicité h alors p' admet x0 pour racine de multiplicité k-1 et x1 comme racine de multiplicité h-1. En outre p' possède une racine dans ]x0,x1[ une racine dans ]x1,x2[, etc.
Donc dans tout les cas p' est scindé tout comme p.

We say that a polynomial with real coefficients is 'split' if all its roots are real, in other words if it factors into a product of polynomials of the first degree with real coefficients.
Show that if p(x) is split, its derivative is also split.

hint

Use the result of exercise 01 of this series, as well as the .

solution

We assume p of degree n+1.
Suppose first that all the roots of p are simple let x0,x1, ..., xn these roots.
According to Rolle's theorem p' has a zero in the interval ]x0,x1[, a zero in the interval ]x1 ,x2[,...., a zero in the interval ]xn-1,xn[ .
So p' has n distinct roots and is split.
Now suppose that x0 is a double root of p, say x1=x0, all others being simple. Then x0 is a simple root of p' furthermore p' has a zero in the interval ]x0,x2[, a zero in the range ]x2,x3[,...., a zero in the range ]xn-1 ,xn[. So p' is split in this case again.
The same reasoning applies if x0 is root of order k of p, i.e. if x0=x1=x2=xk-1. In this case x0 will be root of order k-1 of p' and p' will possess a zero in ]x0,xk [, one in ]xk,xk+1[, etc.
Finally, the same reasoning applies if it is x1 or any other root which has multiplicity k, the others being simple.
Now if x0 has multiplicity k and x1 has multiplicity h then p' admits x0 for root of multiplicity k-1 and x1 as root of multiplicity h-1. Furthermore p' has a root in ]x0,x1[ a root in ]x1,x2[, etc.
So in any case p' is split just like p.
