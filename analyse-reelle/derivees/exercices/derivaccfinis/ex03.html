Soit f:]a,b[ une fonction dérivable.
Montrer qu'on ne peut avoir simultanément:

  • limx→a+f(x)=+∞
  • |f'(x)| < M où M est un nombre fixe

aide

Poser c=(a+b)/2
Appliquer l'inégalité des accroissement finis à un intervalle [x,c] où x∈]a,b[

solution

Posons c=(a+b)/2
Si x ∈ ]a,c[ on a |f(c)-f(x)| ≤ M(c-x).
Mais c-x ≤ (b-a)/2 si a < x < c
Donc f(x) ≤ f(c)+M(b-a)/2.
Par conséquent f est bornée sur ]a,c[ elle ne peut donc tendre vers +∞ quand x → a+.

Let f:]a,b[ be a differentiable function.
Show that we cannot have simultaneously:

  • limx→a+f(x)=+∞
  • |f'(x)| < M where M is a fixed number

hint

Set c=(a+b)/2
Apply the Mean Value Inequality to an interval [x,c] where x∈]a,b[

solution

Let's set c=(a+b)/2
If x ∈ ]a,c[ we have |f(c)-f(x)| ≤ M(c-x).
But c-x ≤ (b-a)/2 if a < x < c
So f(x) ≤ f(c)+M(b-a)/2.
Consequently f is bounded on ]a,c[ so it cannot tend to +∞ when x → a+.
