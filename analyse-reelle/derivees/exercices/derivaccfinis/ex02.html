Démontrer que pour tout x > 0 on a la double inégalité:
x/(1+x2) < atan(x) < x

aide

Appliquer la formule des accroissements finis à y → g(y)=atan(y)-y sur [0,x] d'une part
et à y → h(y)=y/(1+y2)-atan(y) sur [0,x] d'autre part.

solution

Appliquons la formule des accroissements finis à y → g(y)=atan(y)-y sur [0,x]
On à g'(y)=1/(1+y2)-1=-y2/(1+y2) donc g'(y) < 0 sur ]0,x[
Comme g(0)=0, la formule des accroissements finis donne g(x)=xg'(y) avec y ∈ [0,x] donc g(x) < 0 puisque x > 0 et g'(y) < 0.
Il vient donc atan(x) < x.
Appliquons la formule des accroissements finis à y → h(y)=y/(1+y2)-atan(y) sur [0,x]
h'(y)=-2y×atan(y) donc h'(y) < 0 sur ]0,x[ si x > 0 et on a l'autre inégalité.

Prove that for all x > 0 we have the double inequality:
x/(1+x2) < atan(x) < x

hint

Apply M.V.T. formula to y → g(y)=atan(y)-y on [0,x] on one hand
and to y → h(y)=y/(1+y2)-atan(y) on [0,x] on the other hand.

solution

Let's apply M.V.T. formula to y → g(y)=atan(y)-y on [0,x]
On at g'(y)=1/(1+y2)-1=-y2/(1+y2) so g'(y) < 0 of ]0,x[
As g(0)=0, the M.V.T. formula gives g(x)=xg'(y) with y ∈ [0,x] so g(x) < 0 since x> 0 and g'(y) < 0.
So it comes atan(x) < x.
Let us apply M.V.T. formula to y → h(y)=y/(1+y2)-atan(y) on [0,x]
h'(y)=-2y×atan(y) so h'(y) < 0 on ]0,x[ if x> 0 and we have the other inequality.
