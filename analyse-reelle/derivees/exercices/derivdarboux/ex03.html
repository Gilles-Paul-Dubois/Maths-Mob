Montrer qu'aucune fonction ne peut avoir pour dérivée sur ℝ la fonction x→ E(x), E(x) désignant la partie entière de x.

aide

x → E(x) ne prend que des valeurs entières.

solution

Si x → E(x) était la dérivée d'une fonction f, elle satisferait à la propriété de la valeur intermédiaire.
Mais cette foncton n'y satisfait pas.

Show that no function can have as a derivative on ℝ the function x→ E(x), E(x) designating the integral part of x.

hint

x → E(x) takes only integral values.

solution

If x → E(x) were the derivative of a function f, it would satisfy the intermediate value property.
But this function does not.
