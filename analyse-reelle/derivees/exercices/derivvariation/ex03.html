Soit y un réel > 0 et n un entier pair.
Montrer que l'équation (x+y)n=xn+yn ne possède que la solution x=0.
Etudier ensuite le cas où n est impair.

aide

Etudier la variation de f(x)=(x+y)n-xn-yn

solution

Soit donc f(x)=(x+y)n-xn-yn
On a f'(x)=n((x+y)n-1-xn-1)
Comme n est pair n-1 est impair.
f'(x)=0 équivaut à (x+y)n-1=xn
Comme n-1 est impair l'application x →xn-1 est une bijection
Donc f'(x)=0 ⇔ x+y=x ce qui est impossible si y≠0.
Donc la dérivée de f ne s'annule pas, et il est facile de voir qu'elle est positive.
f est donc strictement croissante, elle s'annule en 0, et ne peut donc s'annuler en un autre point.
Etudions maintenant le cas impair.
f'(x)=0 ⇔ x+y=-x soit x=-y/2
On voit que f'(x) est négative pour x < -y/2 et positive si -y/2 < x < 0.
La fonction f présente donc un minimum négatif égal à yn(1/2n-1-1) pour x=-y/2.
En plus de x=0 il y a donc une autre annulation entre -∞ et -y/2 car limx→-∞f(x)=-∞

Let y be a real > 0 and n an even integer.
Show that the equation (x+y)n=xn+yn has only the solution x=0.
Then consider the case where n is odd.

hint

Study the variation of f(x)=(x+y)n-xn-yn

solution

Let f(x)=(x+y)n-xn-yn
We have f'(x)=n((x+y)n-1-xn-1)
As n is even n-1 is odd.
f'(x)=0 is equivalent to (x+y)n-1=xn
As n-1 is odd the map x →xn-1 is a bijection
So f'(x)=0 ⇔ x+y=x which is impossible if y≠0.
So the derivative of f does not vanish, and it is easy to see that it is positive.
f is therefore strictly increasing, it vanishes at 0, and therefore cannot vanish at another point.
Let us now study the odd case.
f'(x)=0 ⇔ x+y=-x i.e. x=-y/2
We see that f'(x) is negative for x < -y/2 and positive if -y/2 < x < 0.
The function f therefore has a negative minimum equal to yn(1/2n-1-1) for x=-y/2.
In addition to x=0 there is therefore another cancellation between -∞ and -y/2 because limx→-∞f(x)=-∞
