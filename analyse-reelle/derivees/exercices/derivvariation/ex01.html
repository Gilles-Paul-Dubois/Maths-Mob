On dispose d'une plaque de carton rectangulaire de 2m ×1m.
On coupe un carré de côté L dans chaque coin 0 ≤ L ≤ 0.5.
On obtient ainsi le patron d'une boîte parallélépipédique (sans couvercle).
Le problème est le suivant:
Calculer la valeur théorique de L pour laquelle le volume de la boîte est maximal.

aide

Etudier la variation de la fonction L →V(L) quand L varie de 0 à 0.5

solution

V(L)=L(1-2L)(2-2L)=4L3-6L2+2L
V'(L)=12L2-12L+2
V'(L)=0 admet pour seule racine dans notre intervalle L0=(3-√3)/6 ≈ 0.21132..
L'étude du signe de V'(L) nous apprend que V(L) est croissant pour L ∈[0,L0[ et décroissant pour L ∈ ]L0,0.5]
La valeur cherchée est donc bien L0.

We have a rectangular cardboard plate of 2m ×1m.
We cut a square of side L in each corner 0 ≤ L ≤ 0.5.
We thus obtain the pattern of a parallelepipedic box (without lid).
The problem is this:
Calculate the theoretical value of L for which the volume of the box is maximal.

hint

Study the variation of the function L →V(L) when L varies from 0 to 0.5

solution

V(L)=L(1-2L)(2-2L)=4L3-6L2+2L
V'(L)=12L2-12L+2
V'(L)=0 admits for only root in our interval L0=(3-√3)/6 ≈ 0.21132..
The study of the sign of V'(L) teaches us that V(L) is increasing for L ∈[0,L0[ and decreasing for L ∈ ]L0,0.5]
The value sought is therefore L0.
