f étant une application bijective d'un intervalle I sur son image.
Quelle relation lie le taux d'accroissement Δf(x0,x) au taux d'accroissement Δf-1(f(x0) ?

solution

\( \displaystyle \Delta f^{-1}\left ( f\left ( x_{0} \right ),f\left ( x \right ) \right )= \frac{f^{-1}\left ( f\left ( x \right ) \right )-f^{-1}\left ( f\left ( x_{0} \right ) \right )}{f\left ( x \right )-f\left ( x_{0} \right )} = \frac{x-x_{0}}{f\left ( x \right )-f\left ( x_{0} \right )} =\frac{1}{\Delta f\left ( x,x_{0} \right )} \)

Une bijection et sa réciproque ont donc toujours le même sens de variation.
