Soit f une fonction numérique définie sur ℝ et vérifiant:
limx→-∞f(x) < 0 et limx→+∞f(x) > 0
Montrer que f s'annule au moins une fois sur ℝ.

aide

Utiliser le théorème des valeurs intermédiaires sur un intervalle convenablement choisi.

solution

Puisque limx→-∞f(x) < 0 il existe un réel a tel que f(a) < 0.
Puisque limx→+∞f(x) > 0 il existe un réel b tel que f(b) > 0.
En outre on peut supposer b > a.
Il suffit donc d'appliquer le théorème des valeurs intermédiaires à [a,b].

Let f be a numeric function defined on ℝ and checking:
limx→-∞f(x) < 0 and limx→+∞f(x) > 0
Show that f vanishes at least once on ℝ.

hint

Use the intermediate value theorem on a suitably chosen interval.

solution

Since limx→-∞f(x) < 0 there exists a real a such that f(a) < 0.
Since limx→+∞f(x) > 0 there exists a real b such that f(b) > 0.
Further we can assume b > a.
So just apply the Intermediate Value Theorem to [a,b].
