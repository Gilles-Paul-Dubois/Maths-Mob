Montrer que sin(acos(x))=cos(asin(x))=√(1-x2)
En déduire: asin(x)=atan(x/√(1-x²))

aide

Utiliser sin²(x)+cos²(x)=1 et acos(x)≥0

solution

On a donc sin²(acos(x))+cos²(acos(x))=1
d'où sin(acos(x))2=1-x2
donc sin(acos(x))=1-x²
Mais 0 ≤ acos(x) ≤ π
donc sin(acos(x))≥0
d'où: sin(acos(x))=+√(1-x²)
Mais aussi:
sin²(asin(x))+cos²(asin(x))=1
donc cos(asin(x))=1-x²
Comme -π/2 ≤ asin(x) ≤ +π/2
On a cos(asin(x))≥0
et donc cos(asin(x))=+√(1-x²)

Show that sin(acos(x))=cos(asin(x))=√(1-x2)
To deduce: asin(x)=atan(x/√(1-x²))

hint

Use sin²(x)+cos²(x)=1 and acos(x)≥0

solution

So we have sin²(acos(x))+cos²(acos(x))=1
whence sin(acos(x))2=1-x2
and sin(acos(x))=1-x²
but 0 ≤ acos(x) ≤ π
so sin(acos(x))≥0
whence sin(acos(x))=+√(1-x²)
but further:
sin²(asin(x))+cos²(asin(x))=1
hence cos(asin(x))=1-x²
since -π/2 ≤ asin(x) ≤ +π/2
we have cos(asin(x))≥0
and finally cos(asin(x))=+√(1-x²)
