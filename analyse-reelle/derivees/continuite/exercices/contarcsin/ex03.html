Montrer que |asin(x)| ≤ |x|/√(1-x²)

aide

Utiliser cos(x) < sin(x)/x < 1/cos(x) pour x > 0

solution

cos(x) < sin(x)/x < 1/cos(x)
conjointement avec le résultat de l'exercice 02 donne:
√(1-x²)asin(x) ≤ x pour x ≥ 0
Donc asin(x) ≤ x/√(1-x²) pour x ≥ 0
Il suffit donc d'utiliser le fait que asin est une fonction impaire.
