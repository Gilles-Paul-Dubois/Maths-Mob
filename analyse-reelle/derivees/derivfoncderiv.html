Fonction dérivée

Derivative function

Définition

Definition

définition 1
Soit f une fonction numérique définie sur D, et soit D'⊆(D) on dit que f est 'dérivable sur D' si f est dérivable en tout point de D', c'est à dire si le nombre dérivé f'(x) existe ∀x∈D'.
Dans ce cas, la fonction f': x → f'(x) définie sur D' est appelée 'la fonction dérivée' de f sur D'.
definition 1
Let f be a numerical function defined on D, and let D'⊆(D) f be said to be 'differentiable on D' if f is differentiable at every point of D', ie if the derivative f'(x) exists ∀x∈D'.
In this case, the function f': x → f'(x) defined on D' is called 'the derivative function' of f on D'.

Exemples

  • D=ℝ f(x)=k ∀x∈D (fonction constante) f est dérivable sur D et f'=0. La dérivée d'une fonction constante est nulle.
  • D=ℝ f(x)=kx ∀x∈D (fonction linéaire) f est dérivable sur D et f'=k. La dérivée d'une fonction linéaire est une constante.
  • D=ℝ f(x)=x2 ∀x∈D , f est dérivable sur D et f'(x)=2x ∀x∈D.
