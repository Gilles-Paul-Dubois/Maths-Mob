definition 1
Just as in the case of sequences of real numbers, the 'infinite products of functions' are sequences of recurrent functions (pn,vn), all defined on the same domain D, satisfying a relation of the type:
pn+1(x)=pn(x).vn(x) ∀x∈D.
Le plus souvent nous écrirons la fonction vn(x) sous la forme 1+un(x), de sorte que, , si aucun des vn n'est la fonction nulle, une condition nécéssaire de convergence (simple) est limn→∞un(x)=0.
En outre une condition suffisante de convergence (toujours simple) est que la série de fonctions de terme général un(x) soit absolument convergente ( ).

démonstration

Soit (an) une suite de réels positifs tels que:
  • |un(x)|≤an ∀x∈D ∀n entier ≥0.
  • La série de terme général an converge.
Une telle série existe d'après notre hypothèse.
Nous pouvons donc choisir N tel que:

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+a_{n} \right )-\prod_{n=0}^{N}\left ( 1+a_{n} \right )\right|< \varepsilon \)

Et nous avons alors :

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+u_{n}(x) \right )-\prod_{n=0}^{N}\left ( 1+u_{n}(x) \right )\right|=\prod_{n=0}^{N} \left ( 1+u_{n}(x) \right ) \left ( \prod_{n=N+1}^{N+p}\left ( 1 +u_{n}(x)\right )-1 \right ) \)

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+u_{n}(x) \right )-\prod_{n=0}^{N}\left ( 1+u_{n}(x) \right )\right|\leqslant \prod_{n=0}^{N} \left ( 1+a_{n} \right ) \left ( \prod_{n=N+1}^{N+p}\left ( 1 +a_{n}\right )-1 \right ) \)

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+u_{n}(x) \right )-\prod_{n=0}^{N}\left ( 1+u_{n}(x) \right )\right|\leqslant \varepsilon \)

proof

Let (an) be a sequence of positive reals such that:
  • |un(x)|≤an ∀x∈D ∀n integer ≥0.
  • The general term series an converges.
Such a series exists according to our hypothesis.
We can therefore choose N such that:

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+a_{n} \right )-\prod_{n=0}^{N}\left ( 1+a_{n} \right )\right|< \varepsilon \)

And then we have :

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+u_{n}(x) \right )-\prod_{n=0}^{N}\left ( 1+u_{n}(x) \right )\right|=\prod_{n=0}^{N} \left ( 1+u_{n}(x) \right ) \left ( \prod_{n=N+1}^{N+p}\left ( 1 +u_{n}(x)\right )-1 \right ) \)

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+u_{n}(x) \right )-\prod_{n=0}^{N}\left ( 1+u_{n}(x) \right )\right|\leqslant \prod_{n=0}^{N} \left ( 1+a_{n} \right ) \left ( \prod_{n=N+1}^{N+p}\left ( 1 +a_{n}\right )-1 \right ) \)

\( \displaystyle \left| \prod_{n=0}^{N+p}\left ( 1+u_{n}(x) \right )-\prod_{n=0}^{N}\left ( 1+u_{n}(x) \right )\right|\leqslant \varepsilon \)
