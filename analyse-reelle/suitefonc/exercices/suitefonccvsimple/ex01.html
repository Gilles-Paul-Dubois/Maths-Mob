Soit D=[0,1].
On considère sur D la suite de fonctions numériques (fn) définies par fn(x)=n2x(1-x)n.
Montrer que (fn) converge simplement vers la fonction nulle sur D.

aide

Distinguer 3 cas:

  1. x=0
  2. x=1
  3. x∈]0,1[

solution

Si x=0 ou x=1 fn(x)=0 ∀n donc limn→∞fn(x)=0.
Si x∈]0,1[ alors fn(x)=n2xk(x)n avec 0 < k(x) < 1.
Donc fn+1(x)/fn(x)=(1+1/n)2k(x)
Comme limn→∞(n+1/n)=1 on a également limn→∞(n+1/n)2=1
Donc limn→∞fn+1(x)/fn(x)=k(x) < 1
Cela veut dire que pour n suffisamment grand disons n > N on a fn+1(x)/fn(x) ≤ h(x) avec h(x) < 1 (on peut par exemple prendre h(x)=(1+k(x))/2).
De là nous tirons par récurrence fn(x) ≤ h(x)n-NfN(x) si n > N.
D'où limn→∞fn(x)=0.

Let D=[0,1].
We consider on D the sequence of numerical functions (fn) defined by fn(x)=n2x(1-x)n.
Show that (fn) pointwise converges to the null function on D.

hint

Distinguish 3 cases:

  1. x=0
  2. x=1
  3. x∈]0,1[

solution

If x=0 or x=1 fn(x)=0 ∀n then limn→∞fn (x)=0.
If x∈]0,1[ then fn(x)=n2xk(x)n with 0 < k(x) < 1.
So fn+1(x)/fn(x)=(1+1/n)2k(x)
As limn→∞(n+1/n)=1 we also have limn→∞(n+1/n)2=1
So limn→∞fn+1(x)/fn(x)=k(x) <1
This means that for n large enough say n > N we have fn+1(x)/fn(x) ≤ h(x) with h(x) < 1 (we can for example take h(x)=(1+k(x))/2).
From there we derive by induction fn(x) ≤ h(x)nNfN(x) if n > N.
Hence limn→∞fn(x)=0.
