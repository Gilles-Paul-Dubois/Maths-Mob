Montrer qu'il existe des fonctions f satisfaisant à:

  • f:ℝ+ → ℝ
  • f continue
  • f(0)=0
  • f non identiquement nulle
  • limx→+∞f(x)=0
Soit f une telle fonction.
Montrer que les deux suites (fn) et (gn) définies pour n≥1 par:
fn(x)=f(nx)
gn(x)=f(x/n)
Convergent simplement vers 0 sur ℝ+

aide

Distinguer deux cas:
x=0
x≠0

solution

Prendre par exemple f(x)=x/(1+x²).
On a fn(0)=gn(0)=f(0)=0 ∀n∈ℕ.
Donc limn→∞fn(0)=limn→∞gn(0)=0.
Soit maintenant x > 0 et soit ε > 0.
D'après les hypothèses faites sur f il existe x0 tel que x > x0 ⇒ |f(x)| < ε.
ℝ étant archimédien, il existe un entier N tel que n > N ⇒ nx > x0.
Donc si n > N on a bien |f(nx)|=|fn(x)| < ε, d'où la convergence simple de la suite (fn) vers 0.
De la même façon, puisque f est continue et nulle en 0, il existe η tel que 0 < t < η ⇒ |f(t)| < ε.
Soit N suffisamment grand pour que x/N < η alors si n > N on a |gn(x)|=|f(x/n)| < ε d'où la convergence simple de la suite (gn) vers 0.

Show that there exist functions f satisfying:

  • f:ℝ+ → ℝ
  • f continuous
  • f(0)=0
  • f not identically null
  • limx→+∞f(x)=0
Let f be such a function.
Show that the two sequences (fn) and (gn) defined for n≥1 by:
fn(x)=f(nx)
gn(x)=f(x/n)
Pointwise converge to 0 on ℝ+

hint

Consider separately two cases :
x=0
x≠0

solution

Take for example f(x)=x/(1+x²).
We have fn(0)=gn(0)=f(0)=0 ∀n∈ℕ.
So limn→∞fn(0)=limn→∞gn(0) =0.
Let now be x > 0 and let ε > 0.
According to the assumptions made on f there exists x0 such that x > x0 ⇒ |f(x)| < ε.
ℝ being Archimedean, there exists an integer N such that n > N ⇒ nx > x0.
So if n>N we have |f(nx)|=|fn(x)| < ε, hence the pointwise convergence of the sequence (fn) to 0.
Similarly, since f is continuous and zero at 0, there exists η such as 0 < t < η ⇒ |f(t)| < ε.
Let N be large enough so that x/N < η then if n> N we have |gn(x)|=|f(x/n)| < ε hence the pointwise convergence of the sequence (gn) to 0.
