f désigne la fonction x → x2 définie sur [0,1].
Montrer, en utilisant seulement la définition, (donc sans utiliser un argument de continuité), que f est intégrable au sens de Riemann sur [0,1].
Calculer son intégrale sur ce segment (toujours avec la définition sans utiliser la notion de primitive).

aide

Utiliser les résultats de .
Utiliser la formule:

\( \displaystyle \sum_{k=1}^{n}k^{2}=\frac{n(n+1)(2n+1)}{6} \)

solution

On a, avec les notations de l'exercice donné en référence:

\( \displaystyle D\left ( f,s_{n} \right )-d\left ( f,s_{n} \right )=\frac{1}{n} \)

il s'en suit que

\( \displaystyle \lim_{n \to \infty }\left (D\left ( f,s_{n} \right )-d\left ( f,s_{n} \right ) \right )=0 \)

La fonction est donc intégrable au sens de Riemann sur [0,1] et

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }D\left ( f,s_{n} \right ) \)

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }\frac{1}{n}\sum_{k=1}^{n}\frac{k^{2}}{n^{2}} \)

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }\frac{1}{n^{3}}\sum_{k=1}^{n}k^{2} \)

Soit en utilisant la formule donnée dans l'aide et qui se démontre facilement par récurrence

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }\frac{n(n+1)(2n+1)}{6n^{3}}=\frac{1}{3} \)

f designates the function x → x2 restricted to [0,1].
Show, using only the definition, (thus without using a continuity argument), that f is Riemann integrable over [0,1].
Calculate its integral on this interval (always with the definition without using the notion of primitive function).

hint

Use the results of .
Use formula:

\( \displaystyle \sum_{k=1}^{n}k^{2}=\frac{n(n+1)(2n+1)}{6} \)

solution

We have, using notations of the exercise given in the help:

\( \displaystyle D\left ( f,s_{n} \right )-d\left ( f,s_{n} \right )=\frac{1}{n} \)

it follows that

\( \displaystyle \lim_{n \to \infty }\left (D\left ( f,s_{n} \right )-d\left ( f,s_{n} \right ) \right )=0 \)

So the function is Riemann integrable on [0,1] and

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }D\left ( f,s_{n} \right ) \)

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }\frac{1}{n}\sum_{k=1}^{n}\frac{k^{2}}{n^{2}} \)

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }\frac{1}{n^{3}}\sum_{k=1}^{n}k^{2} \)

Or using the formula given in the help and which can be easily proven by induction

\( \displaystyle \int_{0}^{1}f(x)dx=\lim_{n \to \infty }\frac{n(n+1)(2n+1)}{6n^{3}}=\frac{1}{3} \)
