Montrer que la fonction définie par

\( \displaystyle f(x)=\left\{\begin{matrix} sin(\frac{1}{x})~~si~~x\neq 0 \\ 0~~si~~x=0 \end{matrix}\right. \)

est intégrable au sens de Riemann sur l'intervalle [0,1].

aide

Pour tout a ∈ [0,1] on a

\( \displaystyle \int_{0}^{1}f(x)dx=\int_{0}^{a}f(x)dx+\int_{a}^{1}f(x)dx \)

solution

Soit ε un réel >0 donné.
f étant continue sur [ε,1] on peut trouver deux fonctions étagées f1 et f2 sur [ε,1] telles que:
f1 ≤ f ≤ f2

\( \displaystyle \int_{\varepsilon }^{1}f_{2}(x)dx-\int_{\varepsilon }^{1}f_{1}(x)dx\leqslant \varepsilon \)

Soit g2 la fonction étagée sur [0,1] définie par g2(x)=1 si x ∈ [0,ε[ et g2(x)=f2(x) si x ∈ [ε, 1].
Soit g1 la fonction étagée sur [0,1] définie par g1(x)=-1 si x ∈ [0,ε[ et g1(x)=f1(x) si x ∈ [ε, 1].
On a toujours g1 ≤ f ≤ g2 sur [0,1]
et on a

\( \displaystyle \int_{0 }^{1}g_{2}(x)dx-\int_{0 }^{1}g_{1}(x)dx\leqslant 3\varepsilon \)

Show that the function defined by

\( \displaystyle f(x)=\left\{\begin{matrix} sin(\frac{1}{x})~~if~~x\neq 0 \\ 0~~if~~x=0 \end{matrix}\right. \)

is Riemann integrable over the interval [0,1].

hint

For every a ∈ [0,1] we have

\( \displaystyle \int_{0}^{1}f(x)dx=\int_{0}^{a}f(x)dx+\int_{a}^{1}f(x)dx \)

solution

Let ε a given real >0.
f being continuous on [ε,1] we can find two step functions f1 and f2 on [ε,1] such that:
f1 ≤ f ≤ f2

\( \displaystyle \int_{\varepsilon }^{1}f_{2}(x)dx-\int_{\varepsilon }^{1}f_{1}(x)dx\leqslant \varepsilon \)

Let g2 be the step function on [0,1] defined by g2(x)=1 if x ∈ [0,ε[ and g2(x)=f2(x) if x ∈ [ε, 1].
Let g1 be the step function on [0,1] defined by g1(x)=-1 if x ∈ [0,ε[ and g1(x)=f1(x) if x ∈ [ε, 1].
We still have g1 ≤ f ≤ g2 on [0,1]
and we have

\( \displaystyle \int_{0 }^{1}g_{2}(x)dx-\int_{0 }^{1}g_{1}(x)dx\leqslant 3\varepsilon \)
