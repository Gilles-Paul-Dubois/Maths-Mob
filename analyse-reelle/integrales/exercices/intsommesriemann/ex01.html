Review on partitions.
For Python add to the 'SubdivisionPointee' class a 'somme_riemann' method taking as argument a function defined on an interval having the same bounds as the partition.

For Julia, define a sum_riemman function, taking as argument a tagged partition and a function.

Application:
Use this object and this method to calculate the integral of the function x → x2 on the interval [0,1] with a regular arithmetic subdivision dividing the interval [0,1] into 1000 sub-intervals of length 0.001.
