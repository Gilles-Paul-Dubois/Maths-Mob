Gaston Darboux (1842/1917-FR)

Définitions

On suppose, dans tout ce qui suit que f est une sur [a,b].

Definitions

We assume, in what follows, that f is a on [a,b].
définition 1
Soit s=(x0,x1, ..., xn) une de [a,b]. On pose :

\( \displaystyle d(f,s)=\sum_{i=0}^{n-1}m_{i}\left ( x_{i+1}-x_{i} \right ) \)

\( \displaystyle D(f,s)=\sum_{i=0}^{n-1}M_{i}\left ( x_{i+1}-x_{i} \right ) \)

où :

\( \displaystyle m_{i}=Inf_{x\in \left [ x_{i},x_{i+1} \right ]}f(x) \)

\( \displaystyle M_{i}=Sup_{x\in \left [ x_{i},x_{i+1} \right ]}f(x) \)

d(f,s) s'appelle la 'somme de Darboux inférieure' relativement à s.
D(f,s) s'appelle la 'somme de Darboux supérieure' relativement à s.
Enfin on pose Δ(f,s)=D(f,s)-d(f,s).
definition 1
Let s=(x0,x1, ..., xn) be a of [a,b]. We set :

\( \displaystyle d(f,s)=\sum_{i=0}^{n-1}m_{i}\left ( x_{i+1}-x_{i} \right ) \)

\( \displaystyle D(f,s)=\sum_{i=0}^{n-1}M_{i}\left ( x_{i+1}-x_{i} \right ) \)

where:

\( \displaystyle m_{i}=Inf_{x\in \left [ x_{i},x_{i+1} \right ]}f(x) \)

\( \displaystyle M_{i}=Sup_{x\in \left [ x_{i},x_{i+1} \right ]}f(x) \)

d(f,s) is called the 'lower Darboux sum' with respect to s.
D(f,s) is called the 'upper Darboux sum' with respect to s.
Finally we set Δ(f,s)=D(f,s)-d(f,s).
Remarque: Dans le cas où f est positive et où nous pouvons attribuer une mesure à de f, l'intervalle [d(f,s), D(f,s)] donne clairement un encadrement de la mesure de cet hypographe.

Propriétés

Les résultats suivants serviront de lemmes techniques pour des résultats ultérieurs.
Note: In case f is positive and we can assign a measure to of f, the interval [d(f,s), D(f,s)] clearly gives a bounding of the measure of this hypograph.

Properties

The following results will serve as technical lemmas for later results.
propriété 1
Si s' est une subdivision que s alors D(f,s') ≤ D(f,s) et d(f,s') ≥ d(f,s).
property 1
If s' is a of s then D(f,s') ≤ D(f,s) and d(f,s') ≥ d(f,s).

démonstration

Voyons ce qui se passe quand on intercale un point supplémentaire y disons entre x0=a et x1. Dans ce cas s=(x0,x1,x2, ... ,xn) s'=(x0,y,x1, ... ,xn).
Si on compare les sommes de Darboux supérieures pour s et s', on a:
D(f,s)=Supx∈[x0,x1] f(x)(x1-x0)+S
et
D(f,s')=Supx∈[x0,y] f(x)(y-x0)+Supx∈[y,x1] f(x)(x1-y)+S.
où S est une somme de Darboux supérieure pour f restreinte à l'intervalle [x1,xn]
De sorte que notre inégalité résulte de
Supx∈[x0,y] f(x) ≤ Supx∈[x0,x1] f(x)
et
Supx∈[y,x1] f(x) ≤ Supx∈[x0,x1] f(x).
Il suffit ensuite d'utiliser le fait que si s' est plus fine que s on passe de s à s' par intercalage d'un nombre fini de points à partir des points de s.
Il suffit donc d'utiliser le raisonnement qui précède à répétition, un nombre fini de fois.
On raisonne de même pour les sommes de Darboux inférieures.

proof

Let's see what happens when we insert an extra point y between say x0=a and x1. In this case s=(x0,x1,x2, ... ,xn) s'=(x0,y,x1, ... ,xn).
If we compare the upper Darboux sums for s and s', we have:
D(f,s)=Supx∈[x0,x1] f(x)(x1-x0)+S
and
D(f,s')=Supx∈[x0,y] f(x)(y-x0)+Supx∈[y,x1] f(x)(x1-y)+S.
where S is an upper Darboux sum for f restricted to the interval [x1,xn]
So our inequality follows from
Supx∈[x0,y] f(x) ≤ Supx∈[x0,x1] f(x)
and
Supx∈[y,x1] f(x) ≤ Supx∈[x0,x1] f(x).
It then suffices to use the fact that if s' is finer than s we pass from s to s' by inserting a finite number of points between the points of s.
So just use the above reasoning repeatedly, a finite number of times.
We reason in the same way for the lower Darboux sums.
Le résultat qui précède entraîne bien évidemment :
The above result obviously leads to:
propriété 2
Si s' est plus fine que s alors Δ(f,s') ≤ Δ(f,s).
property 2
If s' is finer than s then Δ(f,s') ≤ Δ(f,s).
propriété 3
Si s1 et s2 sont des subdivisions quelconques de [a,b], on a toujours: d(f,s1) ≤ Df(s2)
property 3
If s1 and s2 are arbitrary subdivisions of [a,b], we always have: d(f,s1) ≤ Df(s2)
En effet, si on pose s=s1∨s2 on a d'après ce qui précède:
d(f,s1) ≤d(f,s) ≤ D(f,s) ≤ D(f,s2)

Cas des fonctions monotones

Case of monotonic functions

propriété 4
Si f est croissante sur [a,b] et si S est une subdivision quelconque de [a,b] S=x0,x1, ... ,xn, on a bien évidemment:

\( \displaystyle d(f,s)=\sum_{k=0}^{n-1}f\left ( x_{k} \right )\left ( x_{k+1}-x_{k} \right ) \)

\( \displaystyle D(f,s)=\sum_{k=0}^{n-1}f\left ( x_{k+1} \right )\left ( x_{k+1}-x_{k} \right ) \)

property 4
If f is increasing on [a,b] and if S is any subdivision of [a,b] S=x0,x1, ... ,xn, we obviously have:

\( \displaystyle d(f,s)=\sum_{k=0}^{n-1}f\left ( x_{k} \right )\left ( x_{k+1}-x_{k} \right ) \)

\( \displaystyle D(f,s)=\sum_{k=0}^{n-1}f\left ( x_{k+1} \right )\left ( x_{k+1}-x_{k} \right ) \)
