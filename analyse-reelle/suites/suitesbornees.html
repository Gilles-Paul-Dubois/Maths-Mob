Important theorem

The following result is of central importance in analysis.
théorème 1
Tout sous-ensemble majoré (resp. minoré) de ℝ, donc en particulier toute suite majorée (resp. minorée) possède une borne supérieure, resp. borne inférieure).
theorem 1
Any upper bounded set (resp. lower bounded) of ℝ, therefore in particular any upper bounded sequence (resp. lower bounded sequence) has a supremum , (resp. an infimum).

démonstration

Supposons par exemple que X est un sous-ensemble de l'intervalle ]0,1[, de sorte que tous les éléments x de X s'écrivent sous la forme d'un illimité x=0,a0a1a2... an ...où les 0≤ai≤9 sont les décimales successives de x.
Posons b0=Sup(a0(x))x ∈ X
b1=Sup(a1(x))x ∈ X
.................
bi=Sup(ai(x))x ∈ X
et soit B le nombre dont le développement décimal (illimité) est 0,b0b1b2...bn..... Alors il est clair que B est un majorant de X.
De plus considérons la suite de rationnels:
x0=0,b0
x1=0,b0b1
......
xn=0,b0b1...bn
Pour tout i il existe un elément x de X vérifiant x ≥ xi. La suite (xn) est croissante et majorée par B.
De par la définition de cette suite, on peut trouver des xi aussi voisins de B qu'on veut, donc aussi des éléments de X aussi voisins de B qu'on veut.
B est donc bien une borne supérieure pour X.
Si maintenant X n'est pas inclus dans l'intervalle ]0,1[, disons X ⊆ ]a,b[, alors posant k=1/(b-a), l'application f: x →k(x-a) applique X sur un ensemble Y=f(X) inclus dans ]0,1[, cette application est croissante, si M est la borne supérieure de Y, f(M) est la borne supérieure de X.
On raisonne de même pour les bornes inférieures.

proof

Suppose for example that X is a subset of the interval ]0,1[, so that all elements x of X can be written as an unlimited x=0,a0a1a2... an...where the 0≤ai≤9 are the successive decimals of x.
Let b0= Sup(a0(x))x ∈ X
b1=Sup(a1(x))x ∈ X
.................
bi= Sup(ai(x))x ∈ X
and let B be the number whose (unlimited) decimal expansion is 0,b0b1b2...bn..... Then it is clear that B is an upper bound for X.
Moreover consider the sequence of rationals:
x0=0,b0
x1=0,b0b1
......
xn=0,b0b1...bn
For every i there exists an element x of X satisfying x ≥ xi. The sequence (xn) is increasing and bounded by B.
From the definition of this sequence, we can find xi as close to B as we want, therefore also elements of X as close to B as we want.
B is therefore a supremum for X.
If now X is not included in the interval ]0,1[, say X ⊆ ]a,b[, then setting k=1/(b-a), the map f: x →k(xa) maps X on a set Y=f(X) included in ]0,1[, this map is increasing, if M is the upper limit of Y, f(M) is the upper limit of X.
We reason in the same way for the infimum.
