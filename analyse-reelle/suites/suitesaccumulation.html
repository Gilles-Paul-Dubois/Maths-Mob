Le théorème de Bolzano-Weierstrass

Ce résultat très important peut s'énoncer de diverses manières. En voici deux, manifestement équivalentes:
The Bolzano-Weierstrass theorem This very important result can be stated in various ways. Here are two of them, obviously equivalent:
théorème 1
Toute suite bornée de nombres réels possède au moins un point d'accumulation.
theorem 1
Any bounded sequence of real numbers has at least one accumulation point.
théorème 2
De toute suite bornée on peut extraire au moins une suite partielle convergente.
theorem 2
From any bounded sequence we can extract at least one convergent partial sequence.

démonstration

Supposons, dans un premier temps, que la suite (un) se trouve dans l'intervalle ]0,1[, de sorte que tous les un sont représentables par des du type 0,a0a1a2...an... On considère d'abord l'ensemble des valeurs de la première décimale a0 des éléments de la suite. Ces valeurs sont en nombre fini il y en a 10 qui sont 0,1,2, ...,9. Nous avons donc forcément une valeur b0, telle que pour une infinité de valeurs de n la première décimale de un soit b0. On a donc une sous-suite (vn) de (un) pour laquelle tous les éléments ont une première décimale égale à b0. Considérons maintenant les secondes décimales des éléments de la suite (vn), le même raisonnement s'applique. Il existe une valeur b1 telle que pour une infinité de valeurs de n la seconde décimale de vn soit égale à b1. Et ainsi de suite, soit b le nombre 0,b1b2b3...bn..... Il est clair que b possède la propriété suivante:
Pour tout entier k>0 il existe une infinité de valeurs de n pour lesquelles |un-b|<1/10k. b est donc un point d'accumulation de la suite (un).
Si maintenant la suite (un) n'est pas dans ]0,1[ mais dans ]m,M[, on peut par une contraction et une translation se ramener à une suite possédant cette propriété.

proof

Suppose, first, that the sequence (un) lies in the interval ]0,1[, so that all un are representable by of the type 0,a0a1a2...an... We first consider the set of values ​​from the first decimal a0 of the elements of the sequence. These values ​​are in finite number there are 10 of them which are 0,1,2, ...,9. We therefore necessarily have a value b0, such that for an infinity of values ​​of n the first decimal of un is b0. We therefore have a sub-sequence (vn) of (un) for which all the elements have a first decimal equal to b0. Consider now the second decimals of the elements of the sequence (vn), the same reasoning applies. There exists a value b1such that for an infinity of values ​​of n the second decimal of vn is equal to b1. And so on, let b be the number 0,b1b2b3...bn..... It is clear that b has the following property:
For any integer k>0 there exists an infinity of values ​​of n for which |un-b| <1 /10k. b is therefore an accumulation point of the sequence (un).
If now the sequence (un) is not in ]0,1[ but in ]m,M[, we can by a contraction and a translation reduce to a sequence possessing this property.
