Démontrer le résultat du cours suivant:
Soit n→rn une des rationnels. Alors pour la suite (rn) tout réel est un point d'accumulation.

aide

Utiliser le fait que ℚ est dans ℝ

solution

Soit a un réel quelconque et ε>0 Il existe un rationnel r tel que r ∈ ]a-ε,a+ε[.
Comme n → rn est une bijection de ℕ sur ℚ il existe un entier n tel que r=rn.
On peut répéter l'argument pour ε/2, ε/3, ... etc. a est donc un point d'accumulation de (rn).

Prove the following result from the lesson:
Let n→rn be an of rationals. Then for the sequence (rn) every real is a point of accumulation.

hint

Use the fact that ℚ is in ℝ

solution

Let a be any real number and ε>0 There exists a rational number r such that r ∈ ]a-ε,a+ε[.
As n → rn is a bijection from ℕ on ℚ there exists an integer n such that r=rn.
We can repeat the argument for ε/2, ε/3, ... etc. a is therefore an accumulation point of (rn).
