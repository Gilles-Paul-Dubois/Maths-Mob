On considère la suite récurrente donnée par son terme initial x0 et la relation xn+1=a×sin(xn)+b où a est une constante a∈]0,1[ et b une constante quelconque.
Montrer qu'il s'agit d'une suite de Cauchy.
Combien d'itérations sont nécessaires pour obtenir une approximation de la limite à 10-10 près quand a=0.5 b=5 et x0=1 ?

aide

Démontrer la formule |xp+1-xp|≤ap|x1-x0|

solution

xp+1-xp=a(sin(xp)-sin(xp-1))
Utilisons maintenant la formule de trigonométrie:
sin(k)-sin(h)=2sin((k-h)/2)×cos((p+h)/2)
xp+1-xp=2a(sin((xp-xp-1)/2)×cos((xp+xp-1)/2)
D'où nous tirons |xp+1-xp|≤2a|sin((xp-xp-1)/2)|
puisque |cos((xp+xp-1)/2)|≤1
Appliquons maintenant |sin(h)|≤h, pour parvenir à:
|xp+1-xp|≤a|xp-xp-1|
Si maintenant on a |xp-xp-1|≤ap-1|x1-x0|
Il en résulte que |xp+1-xp|≤ap|x1-x0|
Il en résulte que la formule |xp+1-xp|≤ap|x1-x0| qui est trivialement vraie quand p=0 est démontrée par récurrence pour tout p≥0.
Cela étant vu on a si p≥q:
xp-xq=(xp-xp-1)+(xp-1-xp-2)+ ... +xq+1-xq)
D'où nous tirons la majoration:
|xp-xq|≤(ap+ap-1+ ... +aq)|x1-x0||x1-x0|
|xp-xq|≤aq(1+a+ ...+ap-q)|x1-x0|
|xp-xq|≤aq|x1-x0|×1/(1-a)
Si ε est un réel >0 donné, la relation:
|xp-xq|≤ε résultera de: aq|x1-x0|×1/(1-a)≤ε soit:
aq≤cε où c est la constante (1-a)/(x1-x0)
Comme la suite aq tend vers 0 quand q →∞ puisque 0<a<1, il existe une valeur N telle que q>N ⇒aq≤cε
Si p et q sont >N on a donc |xp-xq|≤ε et la suite (xn) est de Cauchy.
Pour avoir la limite avec une précision de 10-10, il faut résoudre:
aq<c10-10 avec a=1/2 c=0.5/0.5sin(1)+4
La constante c vaut 0.351930393
Il suffit donc de trouver un entier q tel que 2q>3519303930
On trouve q≥32.
32 est donc le nombre d'itérations nécessaires pour obtenir la précision voulue.

We consider the recurrent sequence given by its initial term x0 and the relation xn+1=a×sin(xn)+b where a is a constant a∈]0,1[ and b any constant.
Show that this is a Cauchy sequence.
How many iterations are necessary to obtain an approximation of the limit to within 10-10when a=0.5 b=5 and x0=1?

hint

Prove the formula |xp+1-xp|≤ap|x1-x0|

solution

xp+1-xp=a(sin(xp)-sin(xp-1))
Now let's use the trigonometry formula:
sin(k)-sin(h)=2sin((k-h)/2)×cos((p+h)/2)
xp+1-xp=2a(sin((xp-xp-1)/2)×cos((xp+xp-1)/2)
From which we get |xp+1-xp|≤2a|sin((xp-xp-1)/2)|
since |cos((xp+xp-1)/2)|≤1
Now let's apply |sin(h)|≤h, to get:
|xp+1-xp|≤a|xp-xp-1|
If now we have |xp-xp-1|≤ap-1|x1-x0|
It follows that |xp+1-xp|≤ap|x1-x0|
It follows that the formula |xp+1-xp|≤ap|x1-x0| which is trivially true when p=0 is proved by induction for all p≥0.
This being seen, we have if p≥q:
xp-xq=(xp-xp-1)+(xp-1-xp-2)+ ... +xq+1-xq)
From where we derive the inequality:
|xp-xq|≤(ap+ap-1+ ... +aq)|x1-x0||x1-x0|
|xp-xq|≤aq(1+a+ ...+ap-q)|x1-x0|
|xp-xq|≤aq|x1-x0|×1/(1-a)
If ε is a given real >0, the relation:
|xp-xq|≤ε will result from: aq|x1-x0|×1/(1-a)≤ε i.e.:
aq≤cε where c is the constant (1-a)/(x1-x0)
As the sequence aq tends to 0 when q →∞ since 0 N ⇒aq≤cε
If p and q are >N then we have |xp-xq|≤ε and the sequence (xn) is Cauchy.
To have the limit with a precision of 10-10, it is necessary to solve:
aq<c10-10 with a=1/2 c=0.5/0.5sin(1)+4
The constant c is 0.351930393
It is therefore sufficient to find an integer q such that 2q>3519303930
We find q≥32.
32 is therefore the number of iterations necessary to obtain the desired precision.
