Soit (un)n≥1 une suite convergente.
Montrer que la suite (mn) définie pour n ≥1 par:
mn=(u1+u2+ ...+un)/n
converge vers la même limite que (un).
Montrer par un contre-exemple que la réciproque est fausse, c'est à dire que (mn) peut tendre vers une limite sans que pour autant (un) soit convergente.

aide

Soit f: ℕ → ℕ
une fonction vérifiant limn→∞f(n)=+∞ et limn→∞f(n)/n =0
(par exemple f(n)=E(√n) partie entière de la racine de n).
Ecrire: mn= ((u1+ ...+uf(n))+(uf(n)+1 + ... + un))/n
Pour le contre exemple prendre la suite définie par:
un=0 si n est pair.
un=1 si n est impair.

solution

Soit f une fonction comme indiqué dans l'aide.
Soit g la fonction g(n)=n-f(n), de sorte que limn→∞ g(n)/n=1
Posons:
an=(u1+ ...+uf(n))/n
et
bn=(uf(n)+1 + ... + un)/n
De sorte que:
mn=an+bn
La suite (un) étant convergente elle est bornée. Il existe donc un nombre M>0 tel que un≤M ∀ n ≥1 On a donc
|an| ≤M×f(n)/n
On en déduit que limn→∞ an=0
Il suffit donc de montrer que si u converge vers a la suite (bn) converge vers a également.
Soit ε >0 donné. Il existe un entier N tel que n>N ⇒ a-ε < un < a+ε
Mais alors si f(n) > N on a:
(g(n)/n)(a-ε) < bn < (g(n)/n)(a+ε)
Comme limn→∞ g(n)/n=1, pour η > 0 donné on a: 1-η < g(n)/n < 1+η
Donc pour n suffisamment grand:
(1-η)(a-ε) < bn < (1+η)(a+ε)
η et ε pouvant être choisis arbitrairement petits, il en résulte que:
limn→∞bn=a.
Pour le contre exemple prendre la suite définie par:
un=0 si n est pair.
un=1 si n est impair.
Cette suite périodique n'est évidemment pas convergente mais:
limn→∞un=1/2

Let (un)n≥1 be a convergent sequence.
Show that the sequence (mn) defined for n ≥1 by:
mn=(u1+u2+ ...+un)/n
converges to the same limit as (un).
Show by a counterexample that the converse is false, i.e. that (mn) can tend to a limit without (un) being convergent.

hint

Let f: ℕ → ℕ
a function verifying limn→∞f(n)=+∞ and limn→∞f(n)/n =0
(for example f(n)=E(√n) integer part of the root of n).
Write: mn= ((u1+ ...+uf(n))+(uf(n )+1 + ... + un))/n
For the counterexample take the sequence defined by:
un=0 if n is even.
un=1 if n is odd.

solution

Let f be a function as shown in the help.
Let g be the function g(n)=nf(n), so that limn→∞ g(n)/n=1
Let's pose:
an=(u1+ ...+uf(n))/n
and
bn=(uf(n)+1 + ... + un)/n
So that:
mn=an+bn
The sequence (un) being convergent, it is bounded. So there exists a number M>0 such that un≤M ∀ n ≥1 So we have
|an| ≤M×f(n)/n
We deduce that limn→∞ an=0
So it suffices to show that if u converges to a the sequence (bn) also converges to a.
Let ε >0 given. There exists an integer N such that n>N ⇒ a-ε < un < a+ε
But then if f(n) > No we have:
(g(n)/n)(a-ε) < bn < (g(n)/n)(a+ε)
Like limn→∞ g(n)/n=1, for η > 0 given we have: 1-η < g(n)/n < 1+η
So for n large enough:
(1-η)(a-ε) < bn < (1+η)(a+ε)
η and ε can be chosen arbitrarily small, it follows that:
limn→∞bn=a.
For the counterexample take the sequence defined by:
un=0 if n is even.
un=1 if n is odd.
This periodic sequence is obviously not convergent but:
limn→∞un=1/2
