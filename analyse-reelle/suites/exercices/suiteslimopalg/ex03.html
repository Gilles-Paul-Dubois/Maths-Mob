Démonstration du résultat suivant:
Si un →+∞ et vn→0-, alors un/vn → -∞.

solution

Les hypothèses faites sur vn (vn≠0 à partir d'un certain rang et vn→0-), entraînent que:
Pour n suffisamment grand (disons n>N1) on a -1<vn<0
Donc si n>N1 on a:
|un/vn|>|un|
Mais comme un →+∞ pour n suffisamment grand (disons n>N2), on a:
un>0.
Donc si n>N1 et n>N2, on a:
un/vn < -un
Soit M un réel négatif quelconque.
Puisque un → +∞ il existe un entier N3 tel que:
n>N3 ⇒ un>-M
Si maintenant on pose N=Sup(N1,N2,N3)
On aura bien un/vn <M.

Proof of the following result:
If un→+∞ and vn→0-, then un/vn→ -∞.

solution

The hypotheses made on vn(vn≠0 from a certain rank and vn→0-), entail that:
For n sufficiently large (say n>N1) we have -1 n <0
So if n>N1we have:
|un/vn|>|un|
But since un→+∞ for n sufficiently large (say n>N2), we have:
un>0.
So if n>N1and n>N2, we have:
un/vn < -un
Let M be any negative real number.
Since un→ +∞ there exists an integer N3such that:
n>N3⇒ un>-M
If now we put N=Sup(N1,N2,N3)
We will indeed have un/vn <M.
