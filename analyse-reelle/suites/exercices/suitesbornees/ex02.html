Montrer que la suite n×sin(n) est non bornée.

aide

Utiliser le fait que l'intervalle entre π/6 et 5π/6 est de longueur >1.

solution

Pour x +2kπ ∈ [π/6,5π/6] (k entier relatif) on a sin(x)≥1/2.
Comme la longueur de l'intervalle [π/6,5π/6] est >1, il existe une infinité de valeurs de n pour lesquelles n+2kπ ∈ [π/6,5π/6]
Désignons ces valeurs par n0,n1,n2, ..., nk, .....
On a donc |u(nk)| ≥ nk/2 et la suite n'est pas bornée.

Show that the sequence n×sin(n) is not bounded.

hint

Use the fact that interval between π/6 and 5π/6 has length >1.

solution

For x +2kπ ∈ [π/6,5π/6] (k integer) we have sin(x)≥1/2.
Since the length of interval [π/6,5π/6] is >1, there are an infinity of values of n for which n+2kπ ∈ [π/6,5π/6]
Let's denote these values by n0,n1,n2, ..., nk, .....
So we have |u(nk)| ≥ nk/2 and the sequence is not bounded.
