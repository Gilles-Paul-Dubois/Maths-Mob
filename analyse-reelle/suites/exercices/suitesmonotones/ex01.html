x désigne un réel x > 0.
Montrer que la suite de terme général un=xn/n! est décroissante à partir d'un certain rang.
Ce résultat reste-t-il vrai si x est < 0 ?

aide

Calculer le rapport un+1/un.

solution

Le rapport un+1/un est égal à x/(n+1) il devient donc <1 dès que n> x-1.
Quand x est négatif la suite est alternée (elle change de signe sans arrêt), la suite ne peut donc pas être monotone.

x denotes a real x > 0.
Show that the sequence with general term un=xn/n! decreases from a certain rank.
Does this result remain true if x is < 0 ?

hint

Calculate the ratio un+1/un.

solution

The ratio un+1/unis equal to x/(n+1) so it becomes <1 as soon as n> x-1.
When x is negative the sequence is alternating (it constantly changes sign), so the sequence cannot be monotonic.
