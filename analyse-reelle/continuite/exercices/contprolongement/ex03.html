Soit f la fonction numérique définie sur ℝ* par f(x)=(1-cos(x))/x2 si x≠0.
Montrer que f est prolongeable en une fonction continue définie sur ℝ entier.

aide

Etudier limx→0 x≠0f(x).

solution

On a 1-cos(x)=2sin²(x/2).
Donc (1-cos(x))/x²=(1/2) (sin(x/2)/(x/2))(sin(x/2)/(x/2))
D'où limx→0 x≠0f(x)=1/2.
On peut donc prolonger f en une fonction g définie en 0 par g(0)=1/2.

Let f be the numerical function defined on ℝ* by f(x)=(1-cos(x))/x2 if x≠0.
Show that f is extendable into a continuous function defined on the whole of ℝ.

hint

Study limx→0 x≠0f(x).

solution

We have 1-cos(x)=2sin²(x/2).
So (1-cos(x))/x²=(1/2) (sin(x/2)/(x/2))(sin(x/2)/(x/2))
Hence limx→0 x≠0f(x)=1/2.
We can therefore extend f into a function g defined at 0 by g(0)=1/2.
