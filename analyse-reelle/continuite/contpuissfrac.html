Nous allons maintenant appliquer le à des fonctions polynômes extrêmement simples: les fonctions x → xn pour n entier >0.
Remarquons d'abord que ces fonctions sont toutes strictement croissantes sur ℝ+, comme on peut s'en rendre compte immédiatement avec le taux de variation.
Elles possèdent donc des fonctions réciproques continues définies également sur ℝ+.

Racines n-ièmes

La réciproque de x → xn se note x → x1/n et s'appelle racine-nième de x.

We will now apply the to extremely simple polynomial functions : the functions x → xn for n integer >0.
Note first that these functions are all strictly increasing on ℝ+, as can be seen immediately with the rate of change.
They therefore have continuous inverse functions also defined on ℝ+.

n-th roots

The inverse of x → xn is denoted by x → x1/n and is called the n-th root of x.
Here is an applet showing the functions xn and their inverses.
Click 'n+' to increase power.
Click 'n-' to decrease the power.

Puissances fractionnaires positives

Soient maintenant p et q deux entiers > 0.
Pour tout x≥0 on pose xp/q = (x1/q)p.
On obtient ainsi des fonctions croissantes continues (produit de fonctions croissantes continues). Voici une appliquette montrant les fonctions x → xp/q.
Cliquez sur 'p+' pour augmenter le nombre p.
Cliquez sur 'p-' pour diminuer le nombre p.
Cliquez sur 'q+' pour augmenter le nombre q.
Cliquez sur 'q-' pour diminuer le nombre q.

Positive fractional powers

Now let p and q be two integers > 0.
For all x≥0 we set xp/q = (x1/q) p.
We thus obtain continuous increasing functions (product of continuous increasing functions). Here is an applet showing the functions x → xp/q.
Click 'p+' to increase the number p.
Click on 'p-' to decrease the number p.
Click 'q+' to increase the number q.
Click on 'q-' to decrease the number q.

Puissances fractionnaires négatives

Si p/q est négative xp/q est défini pour x≠0 comme 1/x-p/q .
On obtient des fonctions décroissantes.
Voici une appliquette montrant les fonctions x → xp/q, avec p négatif.
Cliquez sur 'p+' pour augmenter le nombre p.
Cliquez sur 'p-' pour diminuer le nombre p.
Cliquez sur 'q+' pour augmenter le nombre q.
Cliquez sur 'q-' pour diminuer le nombre q.

Negative fractional powers

If p/q is negative xp/q is defined for x≠0 as 1/x-p/q .
We obtain decreasing functions.
Here is an applet showing the functions x → xp/q, with negative p.
Click 'p+' to increase the number p.
Click on 'p-' to decrease the number p.
Click 'q+' to increase the number q.
Click on 'q-' to decrease the number q.
